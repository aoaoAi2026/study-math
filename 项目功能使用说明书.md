# 小学奥数知识宝典 · 功能使用说明书

> 版本: v2.0 | 更新时间: 2025-06 | 适用平台: Web / PWA / 移动端

---

## 一、项目概览

"小学奥数知识宝典"是一套面向1~6年级学生的数学学习应用，采用 **"五步深度教学法" + "三阶思维引导" + "六类学习卡片"** 的体系，将知识点、练习题、游戏、工具与家长辅导内容整合到一个应用中。

### 1.1 整体架构

```
┌─────────────────────────────────────────────┐
│              主应用外壳 (AppLayout)           │
│  顶部Logo / 快捷入口 / 个人中心 / 主题切换     │
├─────────────────────────────────────────────┤
│              功能模块 (22个页面)              │
│  ─ 首页 / 知识地图 / 专题学习                 │
│  ─ 游戏中心 / 教具工具箱                      │
│  ─ 每日挑战 / 出题工坊 / 数学小报             │
│  ─ 智能诊断 / 学习路径                        │
│  ─ 个人中心 / 成就墙 / 错题本 / 宠物养成      │
│  ─ 家长学院 / 家长辅导仪表盘                  │
└─────────────────────────────────────────────┘
```

### 1.2 技术特性

| 特性 | 说明 |
|-----|------|
| **响应式布局** | 自适应PC、平板、手机屏幕宽度 |
| **多主题支持** | 亮系 / 暗系 / 护眼 三套主题 |
| **PWA 支持** | 可"添加到主屏幕"，离线访问 |
| **本地存储** | 学习进度、错题、成就、设置均保存在本地浏览器，不需要登录 |
| **无后端依赖** | 纯前端应用，打开即可使用 |

### 1.3 启动方式

```bash
# 1. 安装依赖（首次运行需要）
npm install

# 2. 开发模式启动
npm run dev
# 访问 http://localhost:5173

# 3. 生产构建
npm run build

# 4. 预览构建结果
npm run preview
```

## 三、专题学习模块（Learning）

### 3.1 学习总览（/learning）

页面功能：展示全部知识点卡片，支持按年级和类型筛选。

**筛选方式：**

| 筛选项 | 选项值 | 说明 |
|-------|--------|------|
| **按年级** | 全部 / 一年级 ~ 六年级 | 对应课本年级的知识体系 |
| **按分类** | 全部 / 校内基础 / 奥数专题 | 蓝色标签 = 校内基础；橙色标签 = 奥数专题 |

**专题卡片信息：**
- 📖 图标：知识点视觉标识
- 年级标签 & 类型标签（基础/奥数）
- 知识名称 & 简短介绍（1-2行）
- ⭐ 难度星级（1~5颗星）
- **进入学习 →** 按钮：进入专题详情页

### 3.2 专题详情页（/learning/topic/:topicId）

每个知识点包含6类学习卡片，按照"引入 → 讲解 → 练习 → 变式 → 纠错 → 亲子"的顺序组织：

| 序号 | 卡片类型 | 功能说明 |
|-----|---------|---------|
| 1 | 📖 **故事卡片 (StoryCard)** | 用数学故事/生活化情境引入知识点，降低学习门槛 |
| 2 | 📘 **概念卡片 (ConceptCard)** | 讲解核心概念、公式、定理，配有示意图和关键提示 |
| 3 | ✏️ **例题卡片 (ExampleCard)** | 经典例题+完整解题步骤，展示标准思维流程 |
| 4 | 🎯 **变式练习卡片 (VariantCard)** | 3道难度递进的变式题，学生可作答，点击查看答案 |
| 5 | ⚠️ **常见错误卡片 (MistakeCard)** | 列出本知识点常见的错误类型及分析，避免踩坑 |
| 6 | 👨‍👩‍👧 **亲子共学卡片 (ParentChildCard)** | 家长辅导脚本+家庭拓展活动，支持亲子互动 |

**使用方法：** 页面以纵向滚动方式呈现6种卡片，阅读顺序从前到后。每张卡片内：
- 点击 **"▶ 展开/收起"** 查看详细解题步骤
- 变式练习中，先让孩子答题，再点击 **"查看答案"** 对照
- 每完成一个知识点，系统自动在个人中心记录学习进度

### 3.3 学习路径（/learning-path）

根据智能诊断结果，系统推荐最适合的学习路线：
- **基础夯实路径**：适合诊断结果在50%以下的学生
- **巩固提高路径**：适合中等水平学生
- **竞赛提升路径**：适合基础扎实、向奥数冲刺的学生

路径为线性推进，完成前一专题后解锁下一专题。

## 四、知识地图（/map）

### 4.1 页面功能

以卡片网格形式直观展示所有知识点的掌握状态：

| 状态 | 视觉标识 | 含义 |
|-----|---------|------|
| **已掌握** | 绿色边框 + 勾选标记 ✓ | 该知识点已通过练习和测试 |
| **学习中** | 蓝色边框 + 进度条百分比 | 正在进行中的知识点 |
| **未开始** | 灰色边框 | 尚未进入学习的知识点 |

### 4.2 统计信息

顶部四格：
- 知识点总数
- 已掌握数
- 学习中数量
- 完成率（%）

### 4.3 筛选功能

与"学习总览"一致，支持按年级和分类筛选，便于聚焦某一阶段的学习内容。

## 五、智能诊断测试（/diagnosis）

### 5.1 诊断流程

```
欢迎页 → 开始测试 → 逐题作答 → 结果报告
   (10道题，每题4选1，不限时间但计时)
```

### 5.2 测试内容

共10道题，涵盖5个知识模块：
| 模块 | 题数 | 考察目标 |
|-----|------|---------|
| 计算 | 2题 | 四则运算与小数计算能力 |
| 应用题 | 2题 | 实际问题建模与求解 |
| 几何 | 2题 | 图形认识与面积计算 |
| 数论 | 2题 | 因数/倍数/质数等基础数论概念 |
| 计数 | 2题 | 枚举/组合/分类计数思维 |

### 5.3 结果报告（/diagnosis/result）

诊断完成后展示：
- **总分**：答对题数 / 总题数
- **用时**：总作答时间
- **模块分项得分**：5个模块每个的正确率（雷达图风格展示）
- **推荐学习路径**：根据得分推荐"基础夯实 / 巩固提高 / 竞赛提升"三种路径之一
- **薄弱知识点提示**：分项得分低的模块，附带跳转链接

## 六、每日挑战（/challenge/daily）

### 6.1 功能定位

每天一次的轻量级数学挑战，培养每日练习的习惯。

### 6.2 挑战规则

| 项目 | 规则 |
|-----|------|
| **题目数量** | 5题/天 |
| **题目类型** | 加 / 减 / 乘 / 除 / 混合运算 各1题 |
| **时间限制** | 300秒（5分钟）倒计时 |
| **奖励机制** | 答对得10经验值/题，提前完成额外奖励 |
| **每日唯一性** | 每天题目固定（基于日期的伪随机生成），保证公平性 |

### 6.3 挑战流程

1. 点击 **"开始挑战"** 进入答题
2. 倒计时开始，5道题依次作答
3. 选择题：点击选项直接确认；填空题：输入数字后回车
4. 答完5题或时间到后自动进入结果页
5. 结果页展示：得分 / 用时 / 奖励经验值 / 历史最佳记录

## 九、个人中心（/profile）

### 9.1 页面结构

```
┌──────────────────────────┐
│  👤 头像 | 昵称 | Lv.x等级  │
│      经验值进度条          │   ← 顶部信息区
├──────────────────────────┤
│  ✅ 答对题数 | 📅 学习天数 | 💎 宝石
│      （三格核心数据）       │   ← 成就统计
├──────────────────────────┤
│  🏆 成就墙 / 🐣 宠物小π     │
│  📝 错题本 / 📊 学习报告    │
│  📜 学习历程 / 🎨 主题设置  │   ← 功能菜单列表
└──────────────────────────┘
```

### 9.2 成就墙（/profile/achievements）

展示所有可获得的成就徽章：
- **已解锁**：彩色大图 + 解锁日期
- **未解锁**：灰色图案 + 解锁条件
- 成就类型涵盖：学习进度类、连续打卡类、正确率类、难度挑战类、限时游戏类

### 9.3 宠物小π（/profile/pet）

养成类激励系统：
- **成长阶段**：幼年期🥚 → 成长期🐣 → 少年期🐥 → 成年期🐔
- **成长方式**：每日学习获得经验值喂养宠物；答对题目额外喂养
- **互动**：点击宠物触发动画；达到经验值自动升级到下一阶段
- **目标**：让孩子在学习中获得情感回报，形成正向激励循环

### 9.4 错题本（/profile/wrong-book）

- **分类展示**：按知识模块/知识点分组错题
- **错题详情**：原题 + 你的错误答案 + 正确答案 + 解析
- **重做功能**：点击"重做本题"重新作答，答对后自动从错题本移除
- **错因统计**：自动统计"概念不清""计算错误""审题偏差"等各类错误比例

### 9.5 学习报告（/profile/report）

- **总体概览**：累计学习时长、完成知识点数、累计答对题数、正确率趋势
- **本周表现**：近7天学习热力图，直观看到哪几天学习更投入
- **薄弱知识点**：错题频率最高的TOP5知识点，附"去练习"跳转
- **能力雷达图**：计算/应用/几何/数论/计数 五大模块能力对比

### 9.6 学习历程（/profile/timeline）

以时间线形式展示：
- 首次使用日期
- 完成的知识点里程碑（10个/20个/50个...）
- 获得的成就
- 历史最高连续学习天数

### 9.7 主题设置（/profile/theme）

- **主题风格**：明亮（默认） / 暗黑 / 护眼（低饱和度绿）
- **字号大小**：标准 / 大 / 超大（适合低龄儿童）
- **应用范围**：设置保存在本地，重启应用自动生效

## 十、出题工坊（/workshop）

### 10.1 功能介绍

自定义生成数学练习卷，支持在线作答或打印给孩子做纸质练习。

### 10.2 参数配置

| 配置项 | 选项 | 说明 |
|-------|------|------|
| **题型** | 加减法 / 乘除法 / 混合运算 / 比较大小 | 四类基础题型 |
| **难度** | 简单（1~20）/ 中等（1~100）/ 困难（1~1000） | 数字范围不同 |
| **题数** | 5 / 10 / 15 / 20 | 每次生成的题目数量 |

### 10.3 使用流程

1. 选择"题型 + 难度 + 题数"三项参数
2. 点击 **"生成练习卷"**，系统随机生成对应题目
3. **在线作答**：直接在页面输入答案，点击"提交"自动批改
4. **打印练习**：点击"🖨️ 打印"按钮调出打印对话框
5. 结果展示：得分 / 正确答案对照 / 错误题目高亮

## 十一、数学小报生成器（/newspaper）

### 11.1 功能定位

将数学学习延伸到写作与表达，培养"用数学"的思维习惯。

### 11.2 四种模板

| 模板 | 图标 | 建议用途 |
|-----|------|---------|
| 📓 **数学日记** | 记录学习心得 | 每节课/每天的反思 |
| 📋 **知识总结** | 梳理知识点脉络 | 单元结束复习总结 |
| 🧩 **趣味数学** | 分享有趣问题 | 奥数题目和解题思路 |
| 📖 **数学家故事** | 讲述数学家故事 | 了解数学历史与人物 |

### 11.3 使用流程

1. 选择模板 → 系统自动载入默认栏目结构
2. 编辑"标题"和各栏目内容（可直接替换示例文本）
3. 点击 **"📄 预览"** 查看小报最终排版效果
4. 点击 **"💾 保存到本地"** → 保存到浏览器本地存储
5. 点击 **"📋 查看历史"** → 打开已保存的小报列表
6. 历史记录支持 **重新加载 / 删除** 操作

## 十二、家长学院（/parent-academy）

### 12.1 功能定位

为家长提供科学的数学辅导方法与教育理念文章，让家长也成为孩子的数学老师。

### 12.2 文章分类与内容

| 分类 | 示例文章 | 核心要点 |
|-----|---------|---------|
| **辅导方法** | 如何辅导孩子做应用题 / 错题本的正确使用方法 | "读题-画图-列式-检验"四步法 |
| **心理建设** | 孩子数学焦虑怎么办 / 如何培养数学自信 | 接纳情绪 + 降低难度 + 成长型思维 |
| **习惯培养** | 每天15分钟数学习惯养成 | 固定时间 + 明确内容 + 计时器 + 打卡记录 |
| **奥数入门** | 从什么时候开始学奥数 / 各年级学习建议 | 二年级趣味引入 → 四年级黄金期 |

### 12.3 使用方法

- 点击顶部标签页切换文章分类

## 十四、学习卡片详解（6种卡片）

### 14.1 📖 故事卡片 (StoryCard)

**功能**：用生活故事引入知识点。
**示例**：通过"小鸭子有8只，又来了5只，一共多少只？"的场景引入20以内进位加法。
**使用方法**：和孩子一起读故事 → 提问引导 → 自然过渡到概念卡片。

### 14.2 📘 概念卡片 (ConceptCard)

**功能**：讲解核心知识点、公式、方法。
**示例**：凑十法口诀 → "9要1、8要2、7要3、6要4、5要5"；配以示意图演示分解过程。
**使用方法**：让孩子边读边用手指画；重点部分让孩子自己复述。

### 14.3 ✏️ 例题卡片 (ExampleCard)

**功能**：展示完整解题流程的经典题目。
**结构**：题目 → 解题步骤（编号1、2、3…） → 最终答案 → 关键提示
**使用方法**：先让孩子独立思考 → 对照步骤查缺补漏 → 再尝试独立写出完整步骤。

### 14.4 🎯 变式练习卡片 (VariantCard)

**功能**：3道难度递进的变式题目（基础→提高→拓展）。
**交互**：
- 题目区显示题目
- 孩子作答后点击"查看答案"
- 答案区显示详细解析
- 点击"我答对了/答错了"计入统计

### 14.5 ⚠️ 常见错误卡片 (MistakeCard)

**功能**：列举2-4个本知识点最常见的错误及分析。
**示例**："20以内进位加法"中常见的错误——忘记进位、凑十时分错数、口算错误…
**使用方法**：让孩子先判断"这个错误是不是我也犯过？" → 加深印象 → 避免下次再犯。

### 14.6 👨‍👩‍👧 亲子共学卡片 (ParentChildCard)

**功能**：提供家长辅导脚本+家庭拓展活动。
**内容结构**：
1. 场景对话（孩子问 → 家长如何引导回答）
2. 家庭小游戏（如：扑克牌凑十游戏）
3. 生活中的数学（如：超市购物算账）
**使用方法**：每周一次，固定"亲子数学时间"，持续20-30分钟。

### 17.1 存储位置

应用使用浏览器的 `localStorage` 保存数据，**不需要账号登录**：

| 数据项 | 存储位置 | 持久化 |
|-------|---------|--------|

### 17.2 清除数据的方式

**方式一**：在"个人中心 → 设置 → 清除学习数据"（仅清除进度，保留设置）
**方式二**：浏览器开发者工具 → Application → Local Storage → 清除对应域
**方式三**：浏览器设置 → 清除缓存和数据（⚠️ 会清除所有本地数据）

> ⚠️ **重要提醒**：数据保存在本地浏览器中，清除浏览器数据/更换设备会丢失学习记录。如果需要跨设备同步，请使用同一浏览器账号开启同步功能，或定期截图保存重要进度。

## 十九、常见问题 FAQ

### Q1. 为什么我的进度没有保存？
**A**：请检查是否关闭了浏览器的本地存储功能。另外，清除浏览器数据也会导致进度丢失。

### Q2. 孩子应该从哪个年级开始使用？
**A**：根据孩子实际水平选择。如果孩子是一年级，从一年级校内基础开始；如果水平超前，可以尝试高一阶段的内容，但不要贪多求快。

### Q3. 每天应该学多久？
**A**：低年级建议15-20分钟（1个知识点+几道练习），中高年级30-45分钟。配合"每日挑战"和一个小游戏，学习体验会比较完整。

### Q4. 可以打印练习题吗？
**A**：可以。进入"出题工坊" → 自定义参数 → 生成练习卷 → 点击打印按钮，直接打印给孩子做纸质练习。

### Q5. 孩子做错的题目会怎么样？
**A**：系统会自动记录错题到"个人中心 → 错题本"，家长可以查看错题统计和错因分析，孩子可以重新做错的题直到做对。

### Q6. 如何让孩子保持学习兴趣？
**A**：建议采用混合策略——知识点学习 + 游戏奖励 + 宠物养成 + 亲子共学时间，形成"学-玩-互动"循环。每天固定时间学习，形成习惯更重要。

### Q7. 应用可以离线使用吗？
**A**：可以。首次在浏览器打开后，应用会被缓存（PWA机制）。之后即使没有网络，仍可在浏览器中访问并正常使用。也可以"添加到主屏幕"，像App一样打开。

### Q8. 如何"添加到主屏幕"？
**A**：
- **安卓/Chrome**：浏览器菜单 → "添加到主屏幕"
- **iOS/Safari**：分享菜单 → "添加到主屏幕"
- **Windows Edge**：右上角菜单 → "应用 → 将此站点作为应用安装"

### Q9. 不同孩子使用同一台设备会混淆吗？
**A**：目前版本是"单用户"设计。如果多个孩子共用，建议各自使用不同的浏览器/浏览器用户配置，或者为每个孩子创建独立的浏览器快捷方式。

### Q10. 游戏时间能限制吗？
**A**：家长可在"家长辅导仪表盘"查看孩子的学习/游戏时间比例，并据此进行引导。技术上的硬限制依赖家长的监督与家庭规则的建立。
