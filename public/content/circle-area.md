# 圆的面积与周长 [难度：★]

## 📌 它是什么？
圆是平面上到圆心距离等于半径的所有点组成的图形。周长是绕圆一周的长度，面积是圆所覆盖的平面大小。

## 🎯 为什么重要？
圆是最基本的曲线图形，生活中随处可见——车轮、钟面、披萨、花坛。掌握圆的计算是学习圆柱、圆锥等立体图形的基础。

## 📋 需要什么基础？
- 认识圆的各部分名称（圆心、半径、直径）
- 理解圆周率 π ≈ 3.14
- 熟练进行小数乘法运算

### 两个核心公式

**周长公式**（求围一圈的长度）
$$C = 2\pi r$$
或 $$C = \pi d$$

其中 r = 半径，d = 直径，π ≈ 3.14

**面积公式**（求圆面的大小）
$$S = \pi r^2$$

> **关键区别**：
> - 周长公式中 r 是"一次方"（乘一次）
> - 面积公式中 r 是"平方"（乘两次）
> - 周长的单位是长度单位（米），面积的单位是平方单位（平方米）

**记忆口诀**：周长两πr，面积πr方。
:::

::: example-card
### 母题：已知半径求周长和面积
**题目**：一个圆形花坛的半径是 5 米，求它的周长和面积。（π 取 3.14）

**第一步**：求周长
C = 2πr = 2 × 3.14 × 5 = 31.4（米）

**第二步**：求面积
S = πr² = 3.14 × 5² = 3.14 × 25 = 78.5（平方米）

**第三步**：写答语
答：花坛的周长是 31.4 米，面积是 78.5 平方米。

**验算**：直径 d = 10 米，用 C = πd = 3.14 × 10 = 31.4 米 ✔
:::

::: variant-card
### 变式一：已知直径求面积
一个圆形餐桌的直径是 1.2 米，要在桌面上铺一块圆形桌布，桌布的面积至少是多少平方米？（π 取 3.14）

**思路**：先求半径 r = d ÷ 2 = 0.6 米，再用面积公式。
**解答**：S = πr² = 3.14 × 0.6² = 3.14 × 0.36 = 1.1304（平方米）。
:::

::: variant-card
### 变式二：已知周长求面积
一个圆形跑道的周长是 125.6 米，求跑道围成的面积是多少平方米？（π 取 3.14）

**思路**：先由周长反推半径 C = 2πr，所以 r = C ÷ 2π。
**解答**：r = 125.6 ÷ (2 × 3.14) = 125.6 ÷ 6.28 = 20（米）。
面积 S = πr² = 3.14 × 20² = 3.14 × 400 = 1256（平方米）。
:::

::: mistake-card
### 常见错误：混淆周长和面积公式
**错误一**：求面积时写成 S = 2πr（这是周长公式！）。
**错误二**：求周长时写成 C = πr²（这是面积公式！）。
**错误三**：面积公式中忘记平方，写成 S = πr。
**原因**：对两个公式记忆不牢，没有理解 r² 的含义。
**层级**：L2 概念错误
**纠正**：用口诀"周长两πr，面积πr方"区分，做题时先判断求的是"长度"还是"大小"，再选公式。
:::
