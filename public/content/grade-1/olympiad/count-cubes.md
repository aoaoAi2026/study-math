# 数小正方体

## 📌 它是什么？

数小正方体是奥数中的经典空间思维题。给出一个由若干个小正方体堆叠而成的立体图形，要求数出总共有多少个小正方体。这类题目培养孩子的空间想象能力和有序思维能力。

## 🎯 为什么重要？

数小正方体是培养空间观念的最佳途径之一。它帮助孩子从二维平面思维过渡到三维立体思维，理解"隐藏"的方块。这种能力对以后学习几何、物理等学科非常重要。

## 📋 需要什么基础？

- 认识正方体
- 会数数
- 有基本的空间感知能力

### 积木城堡的秘密

小明用积木搭了一座城堡。妈妈问："你一共用了多少块积木呀？"

小明开始数："1、2、3、4、5……"

"等等，"爸爸打断他，"你这样数容易漏掉或者被挡住看不见的积木。"

"那怎么数呢？"小明问。

"我们可以一层一层地数，"爸爸说，"看，你的城堡有3层。"

"最上面一层有2块积木。"

"中间一层，你能看到3块，但上面一层有2块积木，它们下面一定还有2块积木支撑着，所以中间一层有3+2=5块？不对……"

"让我重新看，"小明说，"中间一层能看到3块，被上面挡住的有2块，所以中间一层有3块？不对，被挡住的也是积木啊！"

"对！"妈妈说，"所以中间一层能看到3块，还有2块被上面挡住了，一共5块。"

"最下面一层，能看到4块，被中间一层挡住的有5块？不对，中间一层只有3块是直接放在最下面一层的……"

"这样数太乱了，"爸爸说，"我们用更好的方法：按列数。"

"看城堡的正面，每一列有多高，就说明那一列有多少块积木。"

"第一列高3块，第二列高2块，第三列高1块……"

"把它们加起来：3+2+1=6块？"

"不对，"小明说，"我的城堡后面还有积木呢！"

"哦，那我们需要从多个方向观察，或者用更好的方法：分层数。"

"从上往下：
- 第一层（最上面）：2块
- 第二层：能看到3块，加上被第一层挡住的2块，共5块？不对，第一层只有2块，所以第二层被挡住的是2块，第二层共3+2=5块？"

"还是不对，"小明笑了，"我的城堡其实只有8块积木，让我拆开给你们看！"

小明拆开城堡，大家终于数清楚了。

"原来数积木这么有学问！"小明感叹道，"以后我要一层一层地数，还要记住被挡住的积木！"

### 数小正方体的秘诀

**1. 分层数法（推荐）**

- 从上往下，一层一层地数
- 每一层的积木数 = 能看到的 + 被上面一层挡住的
- 被上面一层挡住的积木数 = 上面一层的积木数

**2. 按列数法**

- 把立体图形看成若干列
- 每一列的高度就是该列的积木数
- 适用于简单的堆叠图形

**3. 注意事项**

- **不要忘记被挡住的积木**：上面的积木必须有下面的积木支撑
- **注意前后左右的位置**：有些积木在背面或侧面，容易被忽略
- **标记已数过的积木**：可以用手指点着数，或者做记号

**4. 验证方法**

- 数完后，把各层（或各列）的数字相加
- 尝试从不同角度再数一遍
- 如果可能，把积木拆开重新数
:::

### 例题：数积木

数一数，下面的立体图形由多少个小正方体组成？

```
    +---+
    |   |
+---+---+---+
|   |   |   |
+---+---+---+
```

（描述：底层前排3个，底层后排中间1个，第二层中间1个）

**解题步骤**：

**方法一：分层数**

**第一层（最上面）**：
- 能看到1个小正方体（在中间）
- 这一层共1个

**第二层（中间）**：
- 能看到1个小正方体（在中间）
- 被第一层挡住1个（第一层有1个，下面必须有1个支撑）
- 这一层共1+1=2个？不对，能看到的就是全部

重新观察图形：

```
    ②
①  ②  ③
    ④
```

**正确分层**：

**最上层（第3层）**：
- 中间位置有1个
- 共1个

**中间层（第2层）**：
- 中间位置有1个（支撑着最上层的那个）
- 共1个

**最下层（第1层）**：
- 前排3个
- 后排中间1个
- 共4个

**总计**：1 + 1 + 4 = 6个

**方法二：按列数**

从前方看，有3列：
- 左列：高1（只有底层1个）
- 中列：高3（底层1个 + 中间层1个 + 最上层1个）
- 右列：高1（只有底层1个）

但这样只有5个，少了后排的1个。

**正确按列数**（考虑前后）：

从前到后、从左到右：
- 前排左：1个
- 前排中：1个
- 前排右：1个
- 后排中：1个（在底层）
- 中间层中：1个
- 最上层中：1个

**总计**：1+1+1+1+1+1 = 6个

**答**：共有6个小正方体。

**小技巧**：分层数最不容易出错，记住"上面的积木下面一定有支撑"！
:::

- 第二层（下）：3个

**总计**：1 + 3 = 4个

### 变式二：L形堆叠

数一数，下面的图形有多少个小正方体？

```
+---+---+
|   |   |
+---+---+---+
|   |   |   |
+---+---+---+
```

（描述：底层前排3个，底层后排左边2个，第二层在后排左边2个的上面各叠1个）

**解答**：
- 上层：2个
- 下层：前排3个 + 后排2个 = 5个

**总计**：2 + 5 = 7个

**验证**：
- 后排左列：高2（上层1个 + 下层1个）
- 后排中列：高2（上层1个 + 下层1个）
- 前排左列：高1
- 前排中列：高1
- 前排右列：高1

总计：2+2+1+1+1 = 7个 ✓

### 常见错误

**错误1：忘记数被挡住的积木**
- 错例：只数能看到的，忘了上面的积木下面有支撑
- 纠正：记住"上有所依"，每个上面的积木下面一定有积木

**错误2：重复计数**
- 错例：分层数时，把支撑积木和上层积木分开数了两次
- 纠正：分层数时，每层只数该层实际有的积木

**错误3：漏数后排或侧面的积木**
- 错例：只从正面看，忘了后面还有积木
- 纠正：要从多个角度观察，或者按列数

**错误4：数错层数**
- 错例：把2层看成3层
- 纠正：找最高的那列，确定总层数

### 亲子共学脚本

**场景：玩积木**

**爸爸**：宝贝，我们用积木搭一个城堡，然后数一数用了多少块。

**孩子**：好！（搭了一个3层的城堡）

**爸爸**：来，数一数。

**孩子**：1、2、3、4、5……（手指点着数）

**爸爸**：等等，你这样数容易漏。我们一层一层数。最上面一层有几块？

**孩子**：1块！

**爸爸**：中间一层呢？

**孩子**：能看到2块，还有上面那块下面有1块支撑，所以中间一层有3块？

**爸爸**：对！最下面一层呢？

**孩子**：能看到4块，还有中间层3块下面有3块支撑，所以最下面有7块？不对，有些位置是空的……

**爸爸**：我们换个方法。看每一列有多高。

**孩子**：左边这列高2，中间这列高3，右边这列高2。

**爸爸**：那总共多少？

**孩子**：2+3+2=7块！

**爸爸**：我们去验证一下！（拆开数）1、2、3、4、5、6、7，真的是7块！

**课后小任务**：
- 用积木搭各种形状，互相出题数积木
- 画下搭好的积木，从不同角度观察
- 玩"我是建筑师"游戏，规定用多少块积木搭指定形状
