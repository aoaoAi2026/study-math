# 图形剪拼

## 📌 它是什么？

图形剪拼是奥数中的经典题型，通过剪切、平移、旋转等操作，将一个图形变成另一个图形，或将几个图形拼成一个新图形。它培养孩子的空间想象能力、动手操作能力和创造性思维。

## 🎯 为什么重要？

图形剪拼是连接几何与逻辑的桥梁。它不仅能加深孩子对图形特征的理解，还能培养"化繁为简"的数学思想。生活中，拼图、折纸、包装设计等都离不开图形剪拼的思维。

## 📋 需要什么基础？

- 认识基本平面图形（长方形、正方形、三角形、圆）
- 理解"相等""一样大"的概念
- 有基本的动手操作能力

### 巧手小木匠

森林里住着一只聪明的小松鼠，它是远近闻名的"巧手木匠"。

一天，小兔子来找小松鼠："我想做一个正方形的窗花，但我只有一张长方形的纸，长是8厘米，宽是4厘米。"

小松鼠想了想："这简单！你把长方形对折，让长边重合，就得到一个4厘米×4厘米的正方形。剪掉多余的部分就行了。"

小兔子照做了，果然得到了一个漂亮的正方形窗花！

小熊也来了："我有两个完全一样的直角三角形，能拼成一个长方形吗？"

"当然可以！"小松鼠说，"你把两个三角形的斜边拼在一起，就是一个长方形了。如果这两个三角形是等腰直角三角形，还能拼成正方形呢！"

小猴子的难题更有趣："我有一个正方形，想剪一刀，变成两个完全一样的图形，有几种剪法？"

小松鼠拿出纸和剪刀："看好了！

第一种：沿着对角线剪，变成两个完全一样的三角形。
第二种：沿着对边中点连线剪，变成两个完全一样的长方形。
第三种：只要经过正方形的中心点，任意剪一刀，都能分成两个完全一样的图形！"

小动物们都惊呆了。

最后，小松鼠给大家出了一个挑战："把一个正方形剪成4个完全一样的图形，有几种方法？"

小动物们纷纷动手尝试，发现了很多有趣的方法：
- 横竖各剪一刀，分成4个小正方形
- 沿两条对角线剪，分成4个三角形
- 还有很多其他的剪法！

小松鼠笑着说："图形剪拼的奥秘就在于多动手、多观察、多思考！"

### 图形剪拼的秘诀

**1. 剪拼的基本原则**
- 剪拼前后，图形的总面积不变
- "完全一样"意味着形状相同、大小相等
- 可以通过折叠找到对称轴或中点

**2. 常见剪拼方法**

- **一剪变两**：
  - 正方形沿对角线剪 → 2个三角形
  - 正方形沿中线剪 → 2个长方形
  - 长方形沿对角线剪 → 2个三角形
  - 等腰三角形沿对称轴剪 → 2个直角三角形

- **两拼成一**：
  - 2个完全一样的三角形 → 平行四边形
  - 2个完全一样的直角三角形 → 长方形
  - 2个完全一样的等腰直角三角形 → 正方形
  - 2个完全一样的梯形 → 平行四边形

- **一剪变四**：
  - 正方形横竖各一刀 → 4个小正方形
  - 正方形沿两条对角线 → 4个三角形
  - 经过中心点剪两刀 → 4个完全一样的图形

**3. 解题技巧**
- **折叠法**：先折出痕迹，再沿痕迹剪
- **对称法**：利用图形的对称性找剪拼线
- **尝试法**：多动手尝试，从简单的情况入手
- **分割法**：把复杂图形分割成熟悉的简单图形

### 例题：巧拼正方形

小明有两个完全一样的长方形，长是6厘米，宽是3厘米。他想用这两个长方形拼成一个正方形，应该怎么拼？拼成的正方形边长是多少？

**解题步骤**：

**第一步：分析已知条件**
- 长方形：长6厘米，宽3厘米
- 两个完全一样的长方形
- 目标：拼成正方形

**第二步：思考正方形的特点**
- 正方形的四条边都相等
- 拼成的正方形边长应该是多少？

**第三步：尝试拼接**
- 方法：把两个长方形的长边拼在一起
  - 新图形的长：6厘米
  - 新图形的宽：3 + 3 = 6厘米
  - 长 = 宽 = 6厘米，是正方形！

- 验证：
  - 拼成的图形边长为6厘米
  - 面积 = 6 × 6 = 36（平方厘米）
  - 原来两个长方形面积 = 2 × (6 × 3) = 36（平方厘米）
  - 面积相等，拼法正确！

**第四步：画图表示**
```
+--------+--------+
|        |        |
|   ①    |   ②    |  6厘米
|        |        |
+--------+--------+
    6厘米
```

**答**：把两个长方形的长边拼在一起，拼成的正方形边长是6厘米。

**延伸思考**：
- 如果用4个这样的长方形，能拼成什么图形？
- 能拼成更大的正方形吗？（需要边长为12厘米，但4个长方形只能拼成12×6或6×6）

### 变式一：剪正方形

把一个边长为4厘米的正方形剪成4个完全一样的图形，至少给出3种不同的剪法。

**解答**：

**剪法一**：横竖各剪一刀
```
+----+----+
|    |    |
+----+----+
|    |    |
+----+----+
```
得到4个边长为2厘米的小正方形。

**剪法二**：沿两条对角线剪
```
+----+
|\\  /|
| \\/ |
| /\\ |
|/  \\|
+----+
```
得到4个完全一样的等腰直角三角形。

**剪法三**：经过中心点，剪成4个完全一样的梯形
```
+----+
| /\\ |
|/  \\|
|\\  /|
| \\/ |
+----+
```

**剪法四**：先分成4个小正方形，再沿对角线剪
（可以得到更多不同的剪法）
:::

:::variant-card
### 变式二：拼长方形

用3个完全一样的正方形（边长都是2厘米），能拼成一个大长方形吗？如果能，这个大长方形的长和宽各是多少？

**解答**：
- 能拼成长方形
- 拼法：把3个正方形排成一排

```
+----+----+----+
|    |    |    |
| ①  | ②  | ③  |  2厘米
|    |    |    |
+----+----+----+
      6厘米
```

- 长：2 + 2 + 2 = 6（厘米）
- 宽：2（厘米）

**答**：能拼成长方形，长6厘米，宽2厘米。

**延伸**：用4个这样的正方形，除了拼成长方形（8×2或4×4），还能拼成正方形（4×4）！
:::

### 变式三：巧分土地

王大爷有一块正方形的地（如下图），中间有一口圆形的水井。他想把剩下的土地平均分给4个儿子，每个儿子得到的土地要完全一样（包括形状和大小），应该怎么分？

```
+----------+
|   (○)    |
|          |
|          |
+----------+
```

**解答**：
- 关键：经过正方形中心和水井中心的直线，可以把土地分成相等的两部分
- 方法：
  1. 找到正方形的中心（两条对角线的交点）
  2. 找到水井的中心
  3. 过这两点画一条直线，把土地分成两半
  4. 再过这两点（或正方形中心）画另一条垂直的直线
  5. 这样就把土地分成了4个完全一样的部分

**注意**：只要两条分割线都经过正方形的中心，且互相垂直，就能分成4个完全一样的部分。

### 常见错误

**错误1：认为"剪成4份"就是"横竖各一刀"**
- 错例：只知道一种剪法
- 纠正：剪法有很多种，要经过中心点，且4份完全一样

**错误2：拼图形时不考虑边长是否匹配**
- 错例：把长6宽2的长方形和长5宽3的长方形硬拼在一起
- 纠正：拼合的边长度必须相等

**错误3：认为"完全一样"只要面积相等**
- 错例：把一个正方形剪成一个三角形和一个五边形，说它们"一样"
- 纠正："完全一样"（全等）要求形状和大小都相同

**错误4：剪拼后不注意验算**
- 错例：剪完后不检查4份是否真的完全一样
- 纠正：剪完后，把剪下的图形叠在一起比一比，看是否完全重合

### 亲子共学脚本

**场景：折纸游戏**

**爸爸**：宝贝，我们来玩折纸吧。这是一张正方形的纸，你能折一折，剪一刀，变成两个完全一样的长方形吗？

**孩子**：（对折，沿折痕剪开）这样可以吗？

**爸爸**：真棒！那你能剪一刀，变成两个完全一样的三角形吗？

**孩子**：（沿对角线折，剪开）这样！

**爸爸**：太厉害了！现在难度加大，你能把这张正方形纸剪成4个完全一样的正方形吗？

**孩子**：（先对折，再对折，剪开）看，4个小正方形！

**爸爸**：完全正确！那如果用两张完全一样的长方形纸，能拼成一个正方形吗？

**孩子**：（尝试拼接）把长边拼在一起……哇，变成了正方形！

**爸爸**：你发现了吗？长方形的长是宽的两倍时，两个这样的长方形就能拼成正方形！

**课后小任务**：
- 用七巧板拼出不同的形状
- 用剪刀和纸，尝试把一个正方形剪成各种完全一样的图形
- 观察生活中的剪拼：地砖的铺设、墙砖的排列、拼图游戏
