# 一笔画

## 📌 它是什么？

一笔画是指笔不离开纸，每条线只画一次（不重复），画完整个图形。这是图论中最基础的问题之一，由数学家欧拉开创。一年级主要学习判断简单图形能否一笔画成，并找出画法。

## 🎯 为什么重要？

一笔画问题培养孩子的观察力、空间想象力和逻辑思维能力。它是图论的入门，也是很多益智游戏（如七桥问题、迷宫）的基础。生活中，邮递员送信路线、洒水车路线规划等都与此相关。

## 📋 需要什么基础？

- 认识基本图形
- 会数点和线
- 有基本的观察能力

### 七桥城的传说

很久很久以前，有一个美丽的城市叫"七桥城"。城里有一条大河，河上有七座桥，连接着河两岸和两个小岛。

城里的居民喜欢散步，他们想：能不能一次走遍七座桥，每座桥只走一次呢？

很多人尝试了很多年，都没有成功。

有一天，一位叫欧拉的大数学家来到了七桥城。他没有去试走，而是拿出纸和笔画了起来。

"把两岸和两个小岛看成4个点，把七座桥看成7条线，"欧拉说，"问题就变成了：能不能一笔画出这个图形？"

欧拉发现了一个秘密：

"每个点连接的线的数量很重要！如果一个点连接了奇数条线，我们叫它'奇点'；如果连接了偶数条线，就叫'偶点'。"

"一笔画的规律是：
- 如果图形中没有奇点（全是偶点），可以从任意一点开始，一笔画完，最后回到起点。
- 如果有2个奇点，必须从一个奇点开始，到另一个奇点结束。
- 如果有4个、6个……奇点，就不能一笔画成！"

欧拉数了数七桥城的图形：4个点都是奇点！

"所以，七桥城不可能一次走遍七座桥，每座桥只走一次！"

居民们恍然大悟。虽然有点失望，但他们很佩服欧拉的聪明。

从那以后，人们把这种问题叫做"一笔画问题"，也叫"欧拉路径问题"。

### 一笔画的秘诀

**1. 基本概念**

- **奇点**：连接奇数条线的点（1、3、5……）
- **偶点**：连接偶数条线的点（2、4、6……）

**2. 一笔画的判断规则**

- **0个奇点**：可以一笔画成，从任意点出发，最后回到该点（封闭图形）
- **2个奇点**：可以一笔画成，必须从一个奇点出发，到另一个奇点结束
- **4个或更多奇点**：不能一笔画成

**3. 常见图形的一笔画判断**

- 三角形：3个点，每个点连2条线 → 0个奇点 → 可以一笔画
- 正方形：4个点，每个点连2条线 → 0个奇点 → 可以一笔画
- "田"字：4个角点各连2条线（偶点），中间点连4条线（偶点），但4个边中点各连3条线（奇点） → 4个奇点 → 不能一笔画
- "日"字：上下各2个点，各连2条线（偶点），中间2个点各连3条线（奇点） → 2个奇点 → 可以一笔画

**4. 找奇点的方法**
- 在每个交点处数一数有几条线连出来
- 单数就是奇点，双数就是偶点
:::

### 例题：判断能否一笔画

判断下面的图形能否一笔画成。

```
    A
   / \
  /   \
 B-----C
 |     |
 D-----E
```

**解题步骤**：

**第一步：找出所有的点**
- A、B、C、D、E，共5个点

**第二步：数每个点连接的线数**

- 点A：连AB、AC → 2条线 → 偶点
- 点B：连BA、BC、BD → 3条线 → 奇点
- 点C：连CA、CB、CE → 3条线 → 奇点
- 点D：连DB、DE → 2条线 → 偶点
- 点E：连ED、EC → 2条线 → 偶点

**第三步：统计奇点个数**
- 奇点：B、C → 2个奇点

**第四步：判断**
- 有2个奇点 → 可以一笔画成
- 必须从奇点B（或C）开始，到另一个奇点C（或B）结束

**第五步：演示画法**
- 从B开始：B → A → C → B → D → E → C
- 检查：每条线只走了一次 ✓

**答**：可以一笔画成，从B开始到C结束（或从C开始到B结束）。

**小技巧**：先标出奇点，从奇点开始画，就不会走冤枉路！
:::

### 变式一：简单图形判断

判断下面的图形能否一笔画成。

```
A-----B
|     |
|     |
C-----D
```

**解答**：
- 点A：连AB、AC → 2条线 → 偶点
- 点B：连BA、BD → 2条线 → 偶点
- 点C：连CA、CD → 2条线 → 偶点
- 点D：连DB、DC → 2条线 → 偶点

- 奇点个数：0个
- 可以一笔画成，从任意点开始，最后回到起点

**画法**：A → B → D → C → A
:::

:::variant-card
### 变式二："十"字图形

判断"十"字图形能否一笔画成。

```
    |
    |
----+----
    |
    |
```

**解答**：
- 中间的点：连上、下、左、右4条线 → 偶点
- 上端点：连1条线 → 奇点
- 下端点：连1条线 → 奇点
- 左端点：连1条线 → 奇点
- 右端点：连1条线 → 奇点

- 奇点个数：4个
- 4个奇点 → 不能一笔画成

**答**：不能一笔画成。
:::

:::variant-card
### 变式三：添加一条线

下面的图形有4个奇点，不能一笔画成。请添加一条线，使它能够一笔画成。

```
A-----B
|     |
C-----D
```

（假设A、B、C、D都是奇点）

**解答**：
- 4个奇点不能一笔画
- 添加一条线可以连接2个奇点，使它们变成偶点
- 这样剩下2个奇点，就可以一笔画了

**方法**：在任意两个奇点之间添加一条线
- 例如：连接A和B（如果它们之间还没有线）
- 或者连接A和C

**验证**：
- 添加AC后：
  - A连3条线 → 偶点
  - C连3条线 → 偶点
  - 剩下B和D是奇点
  - 2个奇点 → 可以一笔画

**答**：在任意两个奇点之间添加一条线即可。
:::

:::mistake-card
### 常见错误

**错误1：数错每个点连接的线数**
- 错例：认为一个角上的点只连了1条线
- 纠正：要数清楚所有从这个点出发的线

**错误2：把端点忘了**
- 错例：只数了交叉点，没数线段的端点
- 纠正：所有线段的端点都是点，都要数

**错误3：认为0个奇点和2个奇点画法一样**
- 错例：0个奇点时从奇点开始画（其实没有奇点）
- 纠正：0个奇点可以从任意点开始，最后回到起点；2个奇点必须从一个奇点开始

**错误4：画的时候重复走了某条线**
- 错例：虽然判断对了，但画的时候走了回头路
- 纠正：画的时候要仔细，每条线只走一次
:::

### 亲子共学脚本

**场景：公园散步**

**爸爸**：宝贝，我们在公园散步，看看这个路线图。

```
入口 --- 花坛 --- 湖边
 |               |
游乐场 -------- 出口
```

**爸爸**：如果我们要从入口进去，每个地方只去一次，能走遍所有地方吗？

**孩子**：（数点）入口连2条线，花坛连2条线，湖边连2条线，游乐场连2条线，出口连2条线。都是偶点，0个奇点！可以！

**爸爸**：太棒了！那我们从入口开始，怎么走？

**孩子**：入口 → 花坛 → 湖边 → 出口 → 游乐场 → 入口。不对，回到了入口，但游乐场还没去……

**爸爸**：试试另一条路。

**孩子**：入口 → 游乐场 → 出口 → 湖边 → 花坛 → 入口。还是回到入口……

**爸爸**：0个奇点时，一笔画要回到起点。但我们不想回到起点，想去出口离开，怎么办？

**孩子**：那就需要有2个奇点！

**课后小任务**：
- 在纸上画各种图形，判断能否一笔画
- 玩"一笔画"游戏APP或益智玩具
- 观察小区的路线图，看邮递员怎么送信最省路
