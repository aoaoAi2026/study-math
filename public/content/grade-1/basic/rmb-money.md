# 认识人民币

## 📌 它是什么？

认识人民币是学习货币和购物的基础。孩子将认识各种面值的人民币（元、角、分），学会简单的换算和计算，理解人民币在生活中的作用。

## 🎯 为什么重要？

人民币是我们日常生活中必不可少的工具。购物、存钱、捐款都需要用到人民币。认识人民币，不仅能培养孩子的生活自理能力，还能帮助他们建立正确的金钱观。

## 📋 需要什么基础？

- 会认100以内的数
- 会进行100以内的加减法
- 理解"多"和"少"的概念

### 小明的第一次购物

小明今年7岁了，妈妈决定让他独立完成一次购物。

"这是20元钱，"妈妈递给小明一张纸币，"你去超市买一瓶酱油，要8元5角。记住，要拿好找回的钱。"

小明紧张地接过钱，放进口袋里，紧紧地捂着。

来到超市，小明找到酱油区，拿起一瓶酱油，看到价签上写着"8.50元"。

"8.50元就是8元5角，"小明自言自语。

他走到收银台，把酱油和20元钱递给收银员阿姨。

"20元减8元5角，"阿姨一边找钱一边说，"找你11元5角。"

小明接过找回的钱，仔细地数了数：一张10元，一张1元，一张5角。一共11元5角。

他在心里验算：8元5角加11元5角，5角加5角等于10角也就是1元，8元加11元再加1元等于20元。对！

回到家，小明兴奋地把酱油和找回的钱交给妈妈。

"做得真好！"妈妈表扬道，"那你知道1元等于多少角吗？"

"1元等于10角！"小明响亮地回答。

"那1角等于多少分呢？"

"1角等于10分！"小明自信地说。

"真棒！那2元5角等于多少角呢？"

"2元就是20角，再加5角，等于25角！"小明算得很快。

妈妈笑着说："看来你已经认识人民币了，下次可以帮你买更多的东西了！"

小明高兴极了，他觉得学会用钱真是一件了不起的事情！

### 人民币的秘密

**1. 人民币的单位**
- 元、角、分
- 1元 = 10角
- 1角 = 10分
- 1元 = 100分

**2. 常见面值**
- **纸币**：1元、5元、10元、20元、50元、100元
- **硬币**：1分、2分、5分、1角、5角、1元

**3. 元角分的表示**
- 小数表示：8.50元 = 8元5角0分 = 8元5角
- 几元几角几分：如3元2角5分

**4. 换算方法**
- 大单位换小单位：乘以进率
  - 例：5元 = 5 × 10 = 50角
- 小单位换大单位：除以进率
  - 例：30角 = 30 ÷ 10 = 3元

**5. 简单计算**
- 相同单位才能直接相加减
- 不同单位要先统一单位，再计算
- 满10分进1角，满10角进1元

### 例题：小红的存钱罐

小红打开存钱罐，里面有：3张1元、2张5角、4枚1角、5枚5分。

**问题**：
（1）小红一共有多少钱？
（2）她想买一个12元的玩具，还差多少钱？

**解题步骤**：

**第一问：计算总钱数**

- 1元的：3 × 1 = 3（元）
- 5角的：2 × 5 = 10（角）= 1（元）
- 1角的：4 × 1 = 4（角）
- 5分的：5 × 5 = 25（分）= 2（角）5（分）

- 合计：3元 + 1元 + 4角 + 2角5分 = 4元6角5分

**第二问：计算还差多少**

- 玩具价格：12元 = 12元0角0分
- 小红有：4元6角5分
- 12元 - 4元6角5分 = ?

- 计算：
  - 12元 = 11元9角10分
  - 11元9角10分 - 4元6角5分 = 7元3角5分

**答**：小红一共有4元6角5分，还差7元3角5分。

### 变式一：付钱方法

买一支铅笔需要8角，可以怎样付钱？（写出两种方法）

**解答**：
- 方法一：1张5角 + 3张1角 = 8角
- 方法二：8张1角 = 8角
- 方法三：1张5角 + 1张2角 + 1张1角 = 8角（如果有2角纸币）
- 方法四：4张2角 = 8角（如果有2角纸币）
:::

:::variant-card
### 变式二：找零计算

妈妈给收银员50元，买衣服花了36元8角，应找回多少钱？

**解答**：
- 50元 = 49元10角0分
- 49元10角0分 - 36元8角 = 13元2角

**验算**：
- 36元8角 + 13元2角 = 49元10角 = 50元 ✓

- 答：应找回13元2角。
:::

### 变式三：比较价格

小明有15元钱，他想买下面的东西，能买哪几样？

- 笔记本：6元5角
- 铅笔：1元5角
- 橡皮：8角
- 尺子：2元
- 文具盒：9元

**解答**：
- 方案一：笔记本 + 铅笔 + 橡皮
  - 6元5角 + 1元5角 + 8角 = 8元10角8分 = 9元8角 < 15元 ✓

- 方案二：文具盒 + 尺子
  - 9元 + 2元 = 11元 < 15元 ✓

- 方案三：笔记本 + 文具盒
  - 6元5角 + 9元 = 15元5角 > 15元 ✗

- 答：可以买笔记本、铅笔和橡皮；或者买文具盒和尺子等。
:::

### 常见错误

**错误1：单位不统一就计算**
- 错例：5元 + 3角 = 8元（或8角）
- 纠正：单位不同不能直接加，5元 + 3角 = 5元3角

**错误2：把元角分的进率当成100**
- 错例：1元 = 100角
- 纠正：1元 = 10角，1角 = 10分，1元 = 100分

**错误3：找零时减法算错**
- 错例：10元 - 3元5角 = 7元5角
- 纠正：10元 = 9元10角，9元10角 - 3元5角 = 6元5角

**错误4：小数表示理解错误**
- 错例：认为3.5元是3元5分
- 纠正：小数点后第一位表示角，3.5元 = 3元5角；3.05元才是3元5分
:::

### 亲子共学脚本

**场景：模拟超市**

**爸爸**：宝贝，今天我们来玩超市游戏。你当收银员，我当顾客。

**孩子**：好的！

**爸爸**：我要买这个玩具（标价7元5角），给你10元。

**孩子**：（思考）10元减7元5角……

**爸爸**：可以把10元变成9元10角哦。

**孩子**：9元10角减7元5角……9减7等于2元，10减5等于5角。找你2元5角！

**爸爸**：太棒了！那如果我要买两个这样的玩具呢？

**孩子**：7元5角加7元5角……7加7等于14元，5角加5角等于10角就是1元，一共15元！

**爸爸**：给你20元，找我多少？

**孩子**：20减15等于5元！找你5元！

**爸爸**：完全正确！现在换你当顾客，我给你一些钱，你去"买"东西吧。

**课后小任务**：
- 认识真实的人民币，学会辨别真假币
- 和家长一起去超市，让孩子尝试付钱和算找零
- 建立"储蓄罐"，记录每周的零花钱收支
