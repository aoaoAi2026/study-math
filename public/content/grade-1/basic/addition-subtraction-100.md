# 100以内加减法

## 📌 它是什么？

100以内加减法是在掌握100以内数的认识基础上，学习两位数的加法和减法运算。包括不进位加法、进位加法、不退位减法和退位减法，以及连加、连减和加减混合运算。

## 🎯 为什么重要？

100以内加减法是小学数学计算的核心基础。购物算账、时间计算、数量统计等日常生活场景都离不开它。掌握这些运算，孩子才能解决更复杂的数学问题。

## 📋 需要什么基础？

- 熟练掌握100以内数的认识
- 熟练背诵20以内加减法口诀
- 理解"凑十法"和"破十法"

### 100以内加减法的秘诀

**1. 不进位加法**
- 相同数位对齐，从个位加起
- 个位相加不满10，直接写得数
- 例：34 + 25 = 59
  - 个位：4 + 5 = 9
  - 十位：3 + 2 = 5

**2. 进位加法**
- 相同数位对齐，从个位加起
- 个位相加满10，向十位进1
- 例：38 + 26 = 64
  - 个位：8 + 6 = 14，写4进1
  - 十位：3 + 2 + 1 = 6

**3. 不退位减法**
- 相同数位对齐，从个位减起
- 个位够减，直接写得数
- 例：56 - 23 = 33
  - 个位：6 - 3 = 3
  - 十位：5 - 2 = 3

**4. 退位减法**
- 相同数位对齐，从个位减起
- 个位不够减，从十位退1当10
- 例：52 - 18 = 34
  - 个位：2减8不够，从5退1，12 - 8 = 4
  - 十位：4 - 1 = 3

**5. 连加连减和混合**
- 按从左到右的顺序计算
- 有括号的先算括号里的

### 例题：运动会报名

学校开运动会，一年级（1）班有男生28人，女生25人报名。

**问题**：
（1）一共有多少人报名？
（2）如果学校给每个班发50瓶水，还差多少瓶？

**解题步骤**：

**第一问：求总人数**
- 列式：28 + 25 = ?
- 个位：8 + 5 = 13，写3进1
- 十位：2 + 2 + 1 = 5
- 答案：53人

**第二问：求差多少瓶水**
- 需要53瓶，只有50瓶
- 列式：53 - 50 = ?
- 个位：3 - 0 = 3
- 十位：5 - 5 = 0
- 答案：还差3瓶

**验算**：
- 第一问：53 - 25 = 28 ✓
- 第二问：50 + 3 = 53 ✓

**答**：一共有53人报名，还差3瓶水。

### 变式一：购物问题

小明有78元，买了一个书包用去39元，又买了一支笔用去18元，还剩多少元？

**解答**：
- 方法一：依次减
  - 78 - 39 = 39（元）
  - 39 - 18 = 21（元）

- 方法二：先算一共用去多少
  - 39 + 18 = 57（元）
  - 78 - 57 = 21（元）

- 答：还剩21元。

### 变式二：填数字谜

在□里填上合适的数字，使算式成立。

```
  4□
+ □6
----
  82
```

**解答**：
- 个位：□ + 6 = 2 或 □ + 6 = 12
  - 因为和不可能是2（□最小为0，0+6=6），所以□ + 6 = 12
  - □ = 6，个位写2，向十位进1

- 十位：4 + □ + 1 = 8
  - 5 + □ = 8
  - □ = 3

- 验证：46 + 36 = 82 ✓
:::

### 变式三：比较大小

不计算，在○里填上">"、"<"或"="。

（1）45 + 28 ○ 45 + 26
（2）72 - 35 ○ 72 - 38
（3）36 + 24 ○ 26 + 34

**解答**：
- （1）两边都有45，28 > 26，所以 45 + 28 > 45 + 26
- （2）两边都有72，35 < 38，减得越少剩越多，所以 72 - 35 > 72 - 38
- （3）36 + 24 = 60，26 + 34 = 60，所以 36 + 24 = 26 + 34
  - 规律：一个加数多10，另一个加数少10，和不变
:::

### 常见错误

**错误1：数位不对齐**
- 错例：35 + 4 = 75（把4和3对齐了）
- 纠正：4是个位数，要和个位的5对齐
  - 正确：35 + 4 = 39

**错误2：进位后忘记加进位的1**
- 错例：58 + 27 = 75（十位5+2=7，忘了加进位的1）
- 纠正：个位8+7=15，写5进1；十位5+2+1=8
  - 正确：58 + 27 = 85

**错误3：退位后十位没有减1**
- 错例：63 - 28 = 45（十位6-2=4，忘了已经退1）
- 纠正：个位3减8不够，从6退1，13-8=5；十位5-2=3
  - 正确：63 - 28 = 35

**错误4：看到"还剩"就用减法**
- 错例：题目问"一共"，孩子列减法算式
- 纠正：仔细读题，理解题意
  - "一共"、"总共"用加法
  - "还剩"、"差多少"用减法

### 亲子共学脚本

**场景：超市购物**

**妈妈**：宝贝，这盒牛奶25元，这袋面包18元，一共多少钱？

**孩子**：25加18……5加8等于13，写3进1；2加1再加1等于4。一共43元！

**妈妈**：太棒了！妈妈给你50元，应该找回多少？

**孩子**：50减43……0减3不够，借1当10，10减3等于7；4减4等于0。找回7元！

**妈妈**：完全正确！那如果买3盒牛奶呢？

**孩子**：25加25加25……先算25加25等于50，再加25等于75元！

**妈妈**：真厉害！你发现了吗？25 + 25 = 50，50 + 25 = 75，这样算更快！

**孩子**：嗯！我可以找好算的先算！
