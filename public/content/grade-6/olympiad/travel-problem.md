# 六年级数学 - 奥数专题

## 行程问题综合

::: story-card
### 追及问题
小明和小红从同一地点出发，小明骑自行车先走，速度是15千米/时。5分钟后小红骑电动车出发，速度是25千米/时。小红多久能追上小明？
:::

::: concept-card
### 行程问题的三种基本类型

**相遇问题**：两人同时相向而行
- 公式：$路程 = 速度和 × 时间$

**追及问题**：两人同时同向而行
- 公式：$路程差 = 速度差 × 时间$

**环形跑道问题**：反复相遇
- 公式：$首次相遇时间 = 跑道周长 ÷ 速度和$
:::

::: example-card
### 追及问题
**题目**：小明先走5分钟后，小红开始追。小明速度15km/h，小红速度25km/h。多久追上？

**第一步**：转换单位
- 5分钟 = 5/60 小时 = 1/12 小时
- 小明先走的距离 = 15 × 1/12 = 1.25 km

**第二步**：分析追及
小红需要追上这1.25km的距离差

**第三步**：计算时间
$时间 = 距离差 ÷ 速度差 = 1.25 ÷ (25-15) = 0.125 小时 = 7.5 分钟$

**核心公式**：$追及时间 = 初始距离差 ÷ 速度差$
:::

::: parent-child-card
### 亲子共学脚本
**家长**："小明先走了，他走了多远？"
**孩子**：（计算中）1.25千米
**家长**："小红要追上他，需要比小明多走多少？"
**孩子**："一样多，1.25千米"
**家长**："小红每小时比小明快多少？"
**孩子**："10千米"
**家长**："那需要多长时间才能追上？"
:::
