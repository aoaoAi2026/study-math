# 进位制

## 📌 它是什么？

进位制是计数和记数的方法。我们日常使用的是十进制，即"满十进一"。但生活中还有二进制（计算机）、十二进制（时间）、六十进制（角度）等。理解不同进位制之间的转换，能帮助孩子深入理解数的本质。

## 🎯 为什么重要？

进位制是数论的基础内容，也是理解计算机科学的入门知识。掌握进位制转换，能培养孩子的抽象思维和逻辑推理能力，为初中学习指数、对数打下基础。

## 📋 需要什么基础？

- 熟练掌握十进制数的运算
- 理解乘方的概念
- 会进行带余除法
- 理解"位值"的概念

### 进位制的秘密

**1. 进位制的概念**

- n 进制：满 n 进一，有 0、1、2、...、(n-1) 共 n 个数字
- 十进制：0-9，满十进一
- 二进制：0-1，满二进一
- 八进制：0-7，满八进一
- 十六进制：0-9、A-F，满十六进一

**2. n 进制转十进制**

- 按位展开，每位数字乘以 n 的相应次方
- 例：(1011)₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰ = 8 + 0 + 2 + 1 = 11
- 例：(237)₈ = 2×8² + 3×8¹ + 7×8⁰ = 128 + 24 + 7 = 159

**3. 十进制转 n 进制**

- 除 n 取余法：不断除以 n，记录余数
- 余数从下往上读，就是 n 进制数
- 例：十进制 25 转二进制
  - 25 ÷ 2 = 12 余 1
  - 12 ÷ 2 = 6 余 0
  - 6 ÷ 2 = 3 余 0
  - 3 ÷ 2 = 1 余 1
  - 1 ÷ 2 = 0 余 1
  - 结果：(11001)₂

**4. 二进制与八进制、十六进制的转换**

- 二进制转八进制：三位一组
- 二进制转十六进制：四位一组
- 例：(1101011)₂ = (001 101 011)₂ = (153)₈

**5. 进位制运算**

- 加法：满 n 进一
- 减法：借一当 n
- 乘法：按乘法表计算，满 n 进一

### 例题：进位制转换

**题目**：
（1）将十进制数 156 转换成二进制。
（2）将二进制数 (110101)₂ 转换成十进制。
（3）将八进制数 (725)₈ 转换成十进制。

**解题步骤**：

**第（1）问：156 转二进制**

- 156 ÷ 2 = 78 余 0
- 78 ÷ 2 = 39 余 0
- 39 ÷ 2 = 19 余 1
- 19 ÷ 2 = 9 余 1
- 9 ÷ 2 = 4 余 1
- 4 ÷ 2 = 2 余 0
- 2 ÷ 2 = 1 余 0
- 1 ÷ 2 = 0 余 1

从下往上读：(10011100)₂

**验证**：128 + 16 + 8 + 4 = 156 ✓

**第（2）问：(110101)₂ 转十进制**

= 1×2⁵ + 1×2⁴ + 0×2³ + 1×2² + 0×2¹ + 1×2⁰
= 32 + 16 + 0 + 4 + 0 + 1
= 53

**第（3）问：(725)₈ 转十进制**

= 7×8² + 2×8¹ + 5×8⁰
= 7×64 + 2×8 + 5
= 448 + 16 + 5
= 469

**答**：（1）(10011100)₂；（2）53；（3）469。
:::

### 变式一：二进制运算

计算：(1011)₂ + (1101)₂

**解答**：
```
  1011
+ 1101
------
```
- 个位：1 + 1 = 2 = (10)₂，写 0 进 1
- 二位：1 + 0 + 1(进位) = 2 = (10)₂，写 0 进 1
- 三位：0 + 1 + 1(进位) = 2 = (10)₂，写 0 进 1
- 四位：1 + 1 + 1(进位) = 3 = (11)₂，写 1 进 1
- 五位：0 + 0 + 1(进位) = 1

结果：(11000)₂

**验证**：
- (1011)₂ = 11
- (1101)₂ = 13
- 11 + 13 = 24 = (11000)₂ ✓
:::

### 变式二：进制判断

一个自然数在十进制下是 123，在三进制下也是一个三位数。这个三进制数是多少？

**解答**：
- 十进制 123 转三进制
- 123 ÷ 3 = 41 余 0
- 41 ÷ 3 = 13 余 2
- 13 ÷ 3 = 4 余 1
- 4 ÷ 3 = 1 余 1
- 1 ÷ 3 = 0 余 1

结果：(11120)₃

**答**：三进制数是 (11120)₃。
:::

:::variant-card
### 变式三：进制应用

计算机用二进制存储信息。一个字节有 8 位二进制数，最多能表示多少个不同的数？

**解答**：
- 8 位二进制，每位有 0 或 1 两种可能
- 总数 = 2⁸ = 256
- 范围：00000000 到 11111111
- 即十进制的 0 到 255

**答**：最多能表示 256 个不同的数（0-255）。

**延伸**：为什么计算机用二进制？
- 电子元件只有两种状态：通电/断电
- 用 0 和 1 表示，简单可靠
:::

### 常见错误

**错误1：除 n 取余时余数读反**
- 错例：从下往上读变成从上往下读
- 纠正：除 n 取余，余数从下往上读

**错误2：位值计算错误**
- 错例：(101)₂ = 1×2 + 0×2 + 1×2
- 纠正：(101)₂ = 1×2² + 0×2¹ + 1×2⁰

**错误3：进制中数字超出范围**
- 错例：八进制中出现数字 8
- 纠正：n 进制最大数字是 n-1

**错误4：进制运算时进位错误**
- 错例：二进制加法 1+1=2
- 纠正：二进制 1+1=10（满二进一）

**错误5：混淆不同进制**
- 错例：把八进制数当十进制算
- 纠正：明确数的进制，必要时标注下标

### 亲子共学脚本

**场景：手指计数**

**爸爸**：宝贝，我们有 10 根手指，所以用十进制。如果外星人只有 2 根手指呢？

**孩子**：他们就用二进制！只有 0 和 1！

**爸爸**：对！二进制 10 等于十进制多少？

**孩子**：1 个二加 0 个一，等于 2！

**爸爸**：那十进制 5 转成二进制？

**孩子**：5 ÷ 2 = 2 余 1，2 ÷ 2 = 1 余 0，1 ÷ 2 = 0 余 1。
从下往上：(101)₂！

**爸爸**：验证一下：1×4 + 0×2 + 1 = 5 ✓

**孩子**：二进制真有趣！计算机就是用二进制的吧？

**爸爸**：对！因为电脑里的电路只有开和关两种状态，正好对应 0 和 1。
