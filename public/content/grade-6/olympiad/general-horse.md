# 几何最值（将军饮马）

## 📌 它是什么？

"将军饮马"问题是经典的几何最值问题：将军从营地出发，到河边饮马，再到目的地，求最短路径。这类问题利用轴对称（翻折）将折线路径转化为直线段，从而找到最短路径。

## 🎯 为什么重要？

几何最值问题是小学奥数几何模块的精华，能培养孩子的几何直观和优化思维。这类问题体现了数学的简洁美，也是初中学习轴对称、函数最值的重要铺垫。

## 📋 需要什么基础？

- 理解两点之间线段最短
- 掌握轴对称（翻折）的概念
- 会画对称点
- 理解三角形两边之和大于第三边

### 将军的智慧

古代有一位将军，他的营地在一座山的这边，要前往山那边的城池。

山中间有一条笔直的河流。将军需要先到河边饮马，然后再去城池。

"怎样走路线最短？"将军问军师。

军师在地上画了一条线代表河流："我们用对称的方法来解决。"

"假设营地在 A 点，城池在 B 点，河流是一条直线 l。"

"作 A 关于直线 l 的对称点 A'。"

"连接 A'B，与直线 l 交于点 P。"

"那么 A → P → B 就是最短的路线！"

将军不解："为什么？"

军师解释："因为 A 和 A' 关于 l 对称，所以 AP = A'P。"

"路线 A → P → B 的长度 = AP + PB = A'P + PB = A'B。"

"根据两点之间线段最短，A'B 是最短的直线段。"

"如果选河边的另一点 Q，那么 AQ + QB = A'Q + QB > A'B（三角形两边之和大于第三边）。"

将军恍然大悟："原来用对称可以把折线变直线！"
:::

### 将军饮马的秘密

**1. 基本模型**

- 两点 A、B 在直线 l 同侧
- 在 l 上找一点 P，使 PA + PB 最小
- 解法：作 A 关于 l 的对称点 A'，连接 A'B 交 l 于 P

**2. 原理**

- 对称点到对称轴上任意点距离相等
- AP = A'P
- PA + PB = A'P + PB = A'B（最短）
- 依据：两点之间线段最短

**3. 常见变形**

| 类型 | 解法 |
|------|------|
| 两定点到一直线 | 一个点对称，连线 |
| 两定点到两直线 | 两个点都对称，连线 |
| 一定点到两直线 | 分别对称，连线 |
| 三角形周长最小 | 两个对称点，连线 |

**4. 拓展：光的反射原理**

- 入射角 = 反射角
- 光线从 A 出发，经镜面反射到 B，路径最短
- 这与将军饮马问题本质相同

**5. 解题步骤**

（1）确定对称轴（河流、镜面等）
（2）作对称点
（3）连接对称点与另一定点
（4）连线与对称轴的交点即为所求
:::

### 例题：基本将军饮马

**题目**：如图，A、B 两点在直线 l 的同侧，A 到 l 的距离为 3，B 到 l 的距离为 5，A、B 在 l 上的垂足之间的距离为 8。在 l 上找一点 P，使 PA + PB 最小，并求这个最小值。

**解题步骤**：

**第一步：作对称点**

作 A 关于直线 l 的对称点 A'。
A' 到 l 的距离 = A 到 l 的距离 = 3。

**第二步：连接 A'B**

A'B 与 l 的交点即为所求的 P 点。

**第三步：求最小值**

最小值 = A'B 的长度。

构造直角三角形：
- 水平距离 = 8
- 垂直距离 = 3 + 5 = 8
- A'B = √(8² + 8²) = √128 = 8√2

**答**：PA + PB 的最小值为 8√2。

**验证**：
- 用勾股定理：8² + 8² = 64 + 64 = 128
- √128 = 8√2 ≈ 11.31 ✓
:::

### 变式一：两直线问题

角 MON = 30°，点 A 在 OM 上，OA = 4。在 ON 和 OM 上分别找点 P、Q，使三角形 APQ 的周长最小。

**解答**：
- 作 A 关于 OM 的对称点 A₁
- 作 A 关于 ON 的对称点 A₂
- 连接 A₁A₂，交 OM 于 Q，交 ON 于 P
- 三角形 APQ 周长 = AQ + QP + PA = A₁Q + QP + PA₂ = A₁A₂（最短）

由于 ∠MON = 30°，由对称性：
- ∠A₁OA₂ = 2 × 30° = 60°
- OA₁ = OA₂ = OA = 4
- 三角形 OA₁A₂ 是等边三角形
- A₁A₂ = 4

**答**：三角形 APQ 周长的最小值为 4。
:::

:::variant-card
### 变式二：实际应用

一个牧童在 A 处放牛，牛棚在 B 处。A、B 在河流 l 的同侧。牧童要先带牛到河边饮水，再回牛棚。已知 A 到 l 的距离为 2 千米，B 到 l 的距离为 4 千米，A、B 在 l 上的垂足相距 6 千米。求最短路线。

**解答**：
- 作 A 关于 l 的对称点 A'
- 连接 A'B
- 最小距离 = A'B = √(6² + (2+4)²) = √(36 + 36) = √72 = 6√2（千米）

**答**：最短路线长 6√2 千米。
:::

:::variant-card
### 变式三：带约束条件

矩形 ABCD 中，AB = 6，BC = 4。E 是 AB 的中点。在 BC 和 CD 上分别找点 P、Q，使 EP + PQ + QE 最小。

**解答**：
- 这是一个四边形周长最小问题
- 作 E 关于 BC 的对称点 E₁
- 作 E 关于 CD 的对称点 E₂
- 连接 E₁E₂，交 BC 于 P，交 CD 于 Q
- 路径 E → P → Q → E 的最小值 = E₁E₂

计算：
- E 是 AB 中点，AE = EB = 3
- E₁ 到 B 的距离 = 3，在 BC 延长线上
- E₂ 到 D 的距离 = 3，在 CD 延长线上
- 水平距离 = 6 + 3 = 9？需要重新建立坐标系

简化：设 A(0,4)，B(6,4)，C(6,0)，D(0,0)，E(3,4)
- E 关于 BC (x=6) 的对称点 E₁(9,4)
- E 关于 CD (y=0) 的对称点 E₂(3,-4)
- E₁E₂ = √((9-3)² + (4-(-4))²) = √(36 + 64) = √100 = 10

**答**：EP + PQ + QE 的最小值为 10。
:::

### 常见错误

**错误1：对称点作错位置**
- 错例：对称点作到对称轴的另一侧距离不对
- 纠正：对称点到对称轴的距离等于原点到对称轴的距离

**错误2：混淆同侧和异侧**
- 错例：两点在异侧也作对称点
- 纠正：两点在异侧时，直接连线就是最短路径

**错误3：最小值算错**
- 错例：用原两点距离代替对称点距离
- 纠正：最小值 = 对称点与另一点的距离

**错误4：交点找错**
- 错例：连线与对称轴的交点不是所求点
- 纠正：确保连线穿过对称轴

**错误5：忘记验证**
- 错例：算出答案后不画图验证
- 纠正：画出图形，检查合理性

### 亲子共学脚本

**场景：公园找路**

**爸爸**：宝贝，我们在 A 点，要去 B 点，中间要经过湖边的小路。怎么走最近？

**孩子**：把 A 点"翻"到湖的另一边！

**爸爸**：对！这就是对称。假设 A 到湖的距离是 3 格，B 到湖的距离是 5 格，水平距离是 8 格。

**孩子**：A' 到湖也是 3 格。连接 A'B，交湖边于 P。

**爸爸**：最短距离是多少？

**孩子**：用勾股定理！水平 8，竖直 3+5=8。
距离 = √(8² + 8²) = √128 = 8√2！

**爸爸**：完全正确！这个方法叫"将军饮马"，是古代将军找最短路线的方法。

**课后小任务**：
- 在纸上画不同的直线和点，练习找最短路径
- 用镜子验证光的反射路径
- 思考：如果要在两条河边分别取水，怎么办？
