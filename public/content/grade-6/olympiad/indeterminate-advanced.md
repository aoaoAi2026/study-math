# 不定方程进阶

## 📌 它是什么？

不定方程是指未知数的个数多于方程个数的方程，通常有无穷多组解。不定方程进阶是在基础不定方程（如二元一次不定方程）基础上的拓展，涉及更复杂的约束条件和实际应用。

## 🎯 为什么重要？

不定方程是数论的重要内容，能培养孩子的逻辑推理和分类讨论能力。这类问题在竞赛中非常常见，也是初中学习线性方程组、高中学习数论的基础。

## 📋 需要什么基础？

- 熟练掌握方程的解法
- 理解整除、余数的概念
- 会进行枚举和分类讨论
- 理解最大公因数

### 买文具的困惑

小明去文具店买笔记本和钢笔。笔记本每本 5 元，钢笔每支 7 元。小明一共花了 46 元。

"我买了多少本笔记本和多少支钢笔？"小明想。

设买了 x 本笔记本，y 支钢笔。

5x + 7y = 46

"x = 0 时，7y = 46，y 不是整数。"
"x = 1 时，5 + 7y = 46，7y = 41，不行。"
"x = 2 时，10 + 7y = 46，7y = 36，不行。"
"x = 3 时，15 + 7y = 46，7y = 31，不行。"
"x = 4 时，20 + 7y = 46，7y = 26，不行。"
"x = 5 时，25 + 7y = 46，7y = 21，y = 3！"
"x = 6 时，30 + 7y = 46，7y = 16，不行。"
"x = 7 时，35 + 7y = 46，7y = 11，不行。"
"x = 8 时，40 + 7y = 46，7y = 6，不行。"
"x = 9 时，45 + 7y = 46，7y = 1，不行。"

"所以 x = 5，y = 3。买了 5 本笔记本和 3 支钢笔！"

小明又想到："如果我有更多钱，比如 100 元，有多少种买法？"

"5x + 7y = 100"
"x = (100 - 7y)/5 = 20 - 7y/5"
"y 必须是 5 的倍数：y = 0, 5, 10"
"y = 0, x = 20；y = 5, x = 13；y = 10, x = 6"
"y = 15, x = -1（不行）"

"有 3 种买法！"

### 不定方程进阶的秘密

**1. 二元一次不定方程**

- 形式：ax + by = c（a、b、c 为整数）
- 有整数解的条件：c 能被 gcd(a,b) 整除
- 通解：先找一组特解 (x₀, y₀)，则通解为
  - x = x₀ + (b/d) × t
  - y = y₀ - (a/d) × t
  - 其中 d = gcd(a,b)，t 为整数

**2. 解法**

- **枚举法**：适合系数较小的情况
- **同余法**：利用模运算缩小范围
- **辗转相除法**：找特解

**3. 约束条件**

- x、y 为正整数
- x、y 为非负整数
- x、y 有范围限制

**4. 三元不定方程**

- 形式：ax + by + cz = d
- 通常先消元，化为二元不定方程
- 或利用不等式缩小范围

**5. 实际应用**

- "鸡兔同笼"类问题
- 购物问题
- 分配问题

### 例题：求正整数解

**题目**：求方程 3x + 5y = 47 的所有正整数解。

**解题步骤**：

**第一步：用一个变量表示另一个**

x = (47 - 5y)/3

**第二步：确定 y 的范围**

- y > 0，且 47 - 5y > 0，所以 y < 9.4
- y 可以取 1, 2, 3, 4, 5, 6, 7, 8, 9

**第三步：筛选使 x 为正整数的 y**

- y = 1：x = (47-5)/3 = 42/3 = 14 ✓
- y = 2：x = (47-10)/3 = 37/3，不是整数
- y = 3：x = (47-15)/3 = 32/3，不是整数
- y = 4：x = (47-20)/3 = 27/3 = 9 ✓
- y = 5：x = (47-25)/3 = 22/3，不是整数
- y = 6：x = (47-30)/3 = 17/3，不是整数
- y = 7：x = (47-35)/3 = 12/3 = 4 ✓
- y = 8：x = (47-40)/3 = 7/3，不是整数
- y = 9：x = (47-45)/3 = 2/3，不是整数

**第四步：写出所有解**

(x, y) = (14, 1), (9, 4), (4, 7)

**验证**：
- 3×14 + 5×1 = 42 + 5 = 47 ✓
- 3×9 + 5×4 = 27 + 20 = 47 ✓
- 3×4 + 5×7 = 12 + 35 = 47 ✓

**答**：正整数解为 (14, 1)、(9, 4)、(4, 7)。

### 变式一：同余法

求方程 7x + 11y = 100 的所有非负整数解。

**解答**：
- 7x = 100 - 11y
- 两边模 7：0 ≡ 100 - 11y (mod 7)
- 100 ≡ 2 (mod 7)，11 ≡ 4 (mod 7)
- 所以 2 - 4y ≡ 0 (mod 7)
- 4y ≡ 2 (mod 7)
- y ≡ 4 (mod 7)（因为 4×4=16≡2）

- y = 4, 11, 18, ...
- y = 4：x = (100-44)/7 = 56/7 = 8 ✓
- y = 11：x = (100-121)/7 < 0，不行

**答**：唯一非负整数解为 (8, 4)。
:::

### 变式二：三元不定方程

有三种笔，铅笔每支 2 元，圆珠笔每支 3 元，钢笔每支 5 元。小明用 30 元正好买了 10 支笔。三种笔各买了多少支？

**解答**：
- 设铅笔 x 支，圆珠笔 y 支，钢笔 z 支
- x + y + z = 10
- 2x + 3y + 5z = 30

- 由第一式：x = 10 - y - z
- 代入第二式：2(10-y-z) + 3y + 5z = 30
- 20 - 2y - 2z + 3y + 5z = 30
- y + 3z = 10

- z = 0：y = 10，x = 0（没有铅笔，舍去？看题意）
- z = 1：y = 7，x = 2
- z = 2：y = 4，x = 4
- z = 3：y = 1，x = 6
- z = 4：y = -2，不行

**答**：有三种买法：
- 铅笔 2 支，圆珠笔 7 支，钢笔 1 支
- 铅笔 4 支，圆珠笔 4 支，钢笔 2 支
- 铅笔 6 支，圆珠笔 1 支，钢笔 3 支
:::

### 变式三：有范围限制

某班级组织活动，需要租用大船（坐 6 人）和小船（坐 4 人）。全班 40 人，每条船都坐满。大船最多租 5 条，小船最多租 8 条。有多少种租船方案？

**解答**：
- 设大船 x 条，小船 y 条
- 6x + 4y = 40
- 化简：3x + 2y = 20

- x 的范围：0 ≤ x ≤ 5
- y 的范围：0 ≤ y ≤ 8

- x = 0：y = 10（超过 8，不行）
- x = 1：y = 8.5（不是整数）
- x = 2：y = 7 ✓
- x = 3：y = 5.5（不是整数）
- x = 4：y = 4 ✓
- x = 5：y = 2.5（不是整数）

**答**：有两种方案：
- 大船 2 条，小船 7 条
- 大船 4 条，小船 4 条

### 常见错误

**错误1：遗漏解**
- 错例：枚举时跳过了某些值
- 纠正：按顺序枚举，不遗漏

**错误2：范围判断错误**
- 错例：x > 0 写成 x ≥ 0
- 纠正：注意"正整数"和"非负整数"的区别

**错误3：同余计算错误**
- 错例：模运算时算错余数
- 纠正：仔细计算，多验证

**错误4：消元时出错**
- 错例：三元方程消元时计算错误
- 纠正：代入后仔细展开

**错误5：没有验证**
- 错例：求出解后不代回原方程
- 纠正：代入验证，确保正确

### 亲子共学脚本

**场景：购物计划**

**妈妈**：宝贝，我们有 50 元，要买苹果（5 元/斤）和香蕉（8 元/斤），正好花完。有哪些买法？

**孩子**：设苹果 x 斤，香蕉 y 斤。
5x + 8y = 50

**妈妈**：怎么解？

**孩子**：y 从 0 开始试：
- y=0：x=10 ✓
- y=1：x=8.4，不行
- y=2：x=6.8，不行
- y=3：x=5.2，不行
- y=4：x=3.6，不行
- y=5：x=2 ✓
- y=6：x=0.4，不行

**妈妈**：有两种买法！如果要求两种都买呢？

**孩子**：那就只有 x=2，y=5 这一种！

**妈妈**：对！不定方程就是要耐心枚举，找出所有可能的解。

**课后小任务**：
- 设计一道生活中的不定方程问题
- 用同余法解一道不定方程
- 思考：为什么有些不定方程没有整数解？
