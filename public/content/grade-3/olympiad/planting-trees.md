# 植树问题 [难度：★★]

## 📌 它是什么？
植树问题是研究在一定长度的路线上，按照特定间隔种植树木时，"棵数""间隔数""总距离"三者之间关系的问题。植树问题不仅限于"植树"，还包括摆花盆、插红旗、排队、锯木头、敲钟、爬楼梯等生活中常见的问题，是最实用的奥数题型之一！

## 🎯 为什么重要？
植树问题的核心是理解"点数（棵数）"和"段数（间隔数）"的关系，这是训练孩子"分类讨论"和"画图分析"能力的绝佳题材。一个看似简单的植树问题，因为条件不同（两端都栽、只有一端栽、两端都不栽、封闭路线）就有四种完全不同的解法。学会植树问题，孩子就能学会"先看条件、再选公式"的分类讨论思维！

## 📋 需要什么基础？
- 熟练掌握表内除法和乘法
- 理解"间隔"的含义（两个物体之间的距离）
- 会画简单的示意图（用圆点代表树，短线代表间隔）
- 能区分"直线"和"封闭曲线"（圆形、方形等）

### 植树问题的四种经典情形

**核心概念**：植树问题的关键是搞清楚"棵数"和"间隔数"的关系。

**基本关系**：间隔数 = 总距离 ÷ 间隔距离

> 比如：40米长的路，每4米一个间隔，间隔数 = 40÷4 = 10个

**情形一：两端都栽**（小路两头都种树）
```
🌳——🌳——🌳——🌳——🌳
```
- 棵数 = 间隔数 + 1
- 公式：棵数 = 距离 ÷ 间隔 + 1

**情形二：一端栽一端不栽**（比如一头有建筑物或到了路口）
```
🌳——🌳——🌳——🌳
```
- 棵数 = 间隔数
- 公式：棵数 = 距离 ÷ 间隔

**情形三：两端都不栽**（小路两头都是建筑物或河流）
```
——🌳——🌳——🌳——
```
- 棵数 = 间隔数 - 1
- 公式：棵数 = 距离 ÷ 间隔 - 1

**情形四：封闭路线**（圆形、方形、三角形等）
```
   🌳——🌳
  /         \
 🌳          🌳
  \         /
   🌳——🌳
```
- 棵数 = 间隔数
- 公式：棵数 = 周长 ÷ 间隔

**口诀记忆**：
> 两端都栽加个一，
> 两端不栽减个一，
> 一端不栽不加不减，
> 封闭路线不加不减也不乘！

**应用场景举例**：
| 场景 | 类型 | 说明 |
|------|------|------|
| 小路两旁种树 | 两端都栽 | 路的两头都有树 |
| 两栋楼之间种树 | 两端都不栽 | 楼房处不种树 |
| 圆形池塘边种树 | 封闭路线 | 首尾相连，重合了 |
| 爬楼梯 | 两端都栽 | 1楼到5楼，爬4层 |
| 锯木头 | 两端都不栽 | 锯几次→几段 |
| 敲钟 | 两端都栽 | 几点敲几下→间隔数=次数-1 |
| 排队 | 两端都栽 | 人数 = 间隔数 + 1 |

### 母题：两端都栽的直线植树问题

**题目**：学校一条笔直的小路长40米，计划每4米种1棵树，两端都要种。一共需要准备多少棵树苗？

**第一步**：理解题意
- 总距离 = 40米
- 间隔距离 = 4米
- 类型：两端都栽（两头都种树）

**第二步**：先求间隔数
- 间隔数 = 总距离 ÷ 间隔距离
- 间隔数 = 40 ÷ 4 = 10（个）

**第三步**：画图验证棵数和间隔数的关系
```
间隔1  间隔2        间隔10
  ↓     ↓             ↓
🌳——🌳——🌳——🌳——🌳——🌳——🌳——🌳——🌳——🌳——🌳
 ↑                                                        ↑
 树1                                                      树11
```

**第四步**：求棵数
- 两端都栽：棵数 = 间隔数 + 1
- 棵数 = 10 + 1 = 11（棵）

**第五步**：验算
- 11棵树之间有10个间隔
- 10 × 4 = 40（米）正好是小路全长 ✓

**答**：一共需要准备11棵树苗。

**核心公式**：$棵数 = 距离 \div 间隔 + 1（两端都栽）$

> 小提示：画示意图是避免出错的最好办法！刚开始学时，一定要让孩子动手画一画。
:::

### 变式一：两端都不栽

**题目**：两栋教学楼之间有一条30米长的小路，每3米种1棵树，两端靠近楼房处不种树。一共需要多少棵树？

**解答**：
- 类型：两端都不栽（楼房处不种树）
- 间隔数 = 30 ÷ 3 = 10（个）
- 棵数 = 间隔数 - 1 = 10 - 1 = 9（棵）

**画图验证**：
```
 楼       树      树       树       树       树       楼
  |——🌳——🌳——🌳——🌳——🌳——🌳——🌳——🌳——🌳——|
     \___________________________________________/
                      9棵树 → 10个间隔（30米）
```

**验算**：9棵树之间有10个间隔，10 × 3 = 30米 ✓

**答**：一共需要9棵树。

> 小技巧：两端都不栽，关键想——如果两端"假装"种上树，就是"两端都栽"的情形，再减去两头的2棵树，就能得到答案！
:::

::: variant-card
### 变式二：封闭路线（圆形/正方形）

**题目**：一个圆形池塘的周长是60米，每5米种1棵柳树。一共需要多少棵柳树？

**解答**：
- 类型：封闭路线（圆形）
- 关键理解：在封闭路线上种树，第一棵和最后一棵会重合，所以棵数=间隔数
- 棵数 = 周长 ÷ 间隔 = 60 ÷ 5 = 12（棵）

**画图验证**：
```
         🌳  🌳
        /        \
       🌳          🌳
       |    池塘    |
       🌳          🌳
        \        /
         🌳  🌳
       （共12棵）
```

**答**：一共需要12棵柳树。

> 小技巧：正方形、长方形等封闭图形的植树问题，公式和圆形一样！
> 拓展：如果是正方形的池塘，边长15米，每5米种1棵树（4个角都种），那每边种几棵？四边一共多少棵？
> （答案：每边种15÷5+1=4棵，四边一共4×4-4=12棵——因为4个角重复计算了，要减去4）
:::

::: variant-card
### 变式三：植树问题的生活应用（锯木头/爬楼梯）

**题目**：一根木头锯成4段需要12分钟，锯成6段需要多少分钟？

**提示**：锯木头时，锯的次数和段数是什么关系？画一画：
```
锯1次：🌲——|——🌲 → 2段
锯2次：🌲——|——🌲——|——🌲 → 3段
锯3次：🌲——|——🌲——|——🌲——|——🌲 → 4段
```

**解答**：
- 关键发现：锯的次数 = 段数 - 1
- 锯成4段：需要锯 4 - 1 = 3次，共用12分钟
- 每次需要：12 ÷ 3 = 4（分钟）
- 锯成6段：需要锯 6 - 1 = 5次
- 5次需要：5 × 4 = 20（分钟）

**答**：锯成6段需要20分钟。

> 小技巧：锯木头、爬楼梯、敲钟都是植树问题的变形！
> - 锯木头：次数 = 段数 - 1（相当于两端都不栽）
> - 爬楼梯：层数 = 楼数差（从1楼爬到5楼，爬了4层）
> - 敲钟：间隔数 = 敲的次数 - 1（敲5下有4个间隔）
:::

### 常见错误分析

**错误一：忘记区分情形，所有题都用同一个公式**
**错误**：两端都不栽的题也用"棵数=间隔数+1"来算
**原因**：没有养成"先判断类型再选公式"的习惯
**纠正**：每次做题，先问自己三个问题：(1) 是直线还是封闭？(2) 两端种不种？(3) 属于哪种情形？
**错误层级**：L2（方法错误）

**错误二：两端都栽时忘记加1**
**错误**：40米每4米一棵，算成40÷4=10棵（正确应为11棵）
**原因**：没有理解"两端都栽"的含义，画图不认真
**纠正**：用简单的数字试——1个间隔有2棵树，2个间隔有3棵树，所以棵数总比间隔数多1
**错误层级**：L2（概念错误）

**错误三：封闭路线上多加了1**
**错误**：圆形池塘60米每5米一棵，算成12+1=13棵
**原因**：把直线情形的公式用到了封闭路线
**纠正**：在封闭路线上画图——第一棵和最后一棵重合了，所以棵数=间隔数，不加1也不减1
**错误层级**：L2（理解错误）

**错误四：锯木头/爬楼梯时混淆"次数"和"段数"**
**错误**：锯成4段需要4次（正确应为3次）
**原因**：没有理解"次数比段数少1"的关系
**纠正**：画图！画一条线，画1条分隔线，就分成2段；画2条分隔线，就分成3段。所以次数=段数-1
**错误层级**：L2（概念错误）

### 亲子共学脚本（傻瓜模式）

**场景**：客厅地上用积木摆成一条"小路"，孩子正准备做植树问题的练习。

**家长**："宝贝，我们来玩一个'种树'的游戏好不好？这里有40厘米长的一条'小路'（指着积木摆成的线），每4厘米种1棵'树'（用小积木），两头都要种。我们需要多少棵'树'？"
**孩子**：（可能开始数积木或直接算）40÷4=10棵！

**家长**："我们来动手摆一摆，看看10棵树对不对。先在最左边放1棵，然后每隔4厘米放1棵。"
**孩子**：（开始摆）1棵、2棵、3棵……摆到第10棵后——咦？右边还有4厘米空着！

**家长**："发现问题了吗？10棵树之间只有9个间隔，每个间隔4厘米，一共只有36厘米，还没到40厘米呢！那我们需要多少棵？"
**孩子**："11棵！"
**家长**："太棒了！你自己发现了——'两头都种'的时候，棵数比间隔数多1。那我们来总结一下："
**孩子**："40÷4=10个间隔，10+1=11棵树！"

**家长**："太厉害了！那如果这条路变成圆形（把积木摆成一个圈），第一棵和最后一棵会怎么样？"
**孩子**：（摆成圆圈后）重合了！所以不用加1了！
**家长**："完全正确！那就是'棵数=间隔数'。最后我们来玩一个锯木头的游戏——如果把一根木头锯成4段需要锯几次？"
**孩子**：（画了一条线，画3条竖线分隔）锯3次！因为4段之间有3个间隔！
**家长**："太厉害了！你已经会把植树问题的方法用到生活中的其他问题啦！告诉爸爸/妈妈，你是怎么想到的？"
**孩子**：（兴奋地讲解画图的过程）

> **家长小贴士**：
> - 植树问题最重要的是"画图"——用实物摆一摆、画一画，比背公式管用100倍
> - 刚开始学，不要强迫孩子背四种情形的公式，先让孩子画图理解"棵数"和"间隔数"的关系
> - 可以拿家里的东西做实验：用硬币摆成一排看几个间隔，围成一圈看几个间隔
> - 爬楼梯、锯木头是植树问题的变形，在生活中遇到时可以让孩子想一想
> - 孩子刚开始容易混淆四种情形，做几道题后让孩子自己总结"什么时候加1""什么时候减1""什么时候不加不减"
