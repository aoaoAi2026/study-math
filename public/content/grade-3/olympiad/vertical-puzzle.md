# 竖式谜

## 知识点卡片

:::story-card
### 生活小故事

数学课上，老师在黑板上写了一个奇怪的竖式：

```
  数学
+ 学数
------
  99
```

"这里的'数'和'学'代表不同的数字，谁能解开这个谜？"

小明仔细观察："数 + 学 = 9，学 + 数 = 9，所以数 + 学 = 9。"

"如果数 = 1，学 = 8；数 = 2，学 = 7...有好多答案啊！"

老师笑着说："对！竖式谜就是要根据已知数字和运算规则，推理出未知的数字。这需要观察、分析和尝试！"

小明爱上了这种数字推理游戏，他发现：每一个竖式谜都是一个小侦探故事！
:::

### 核心概念

**什么是竖式谜**

在竖式中，某些数字被字母、图形或空格代替，要求根据运算法则推理出这些未知的数字。

**解题基本方法**

1. **观察分析法**
   - 从个位开始分析
   - 注意进位和借位

2. **首位分析法**
   - 看最高位，确定范围
   - 如：两位数加两位数得三位数，首位一定是1

3. **末位分析法**
   - 看个位，利用乘法口诀
   - 如：□ × 3 末位是7，□ = 9

4. **进位分析法**
   - 分析可能的进位情况
   - 排除不可能的情况

**常用技巧**
- 黄金三角：加法中，两个数相加得三位数，首位一定是1
- 0和1的特殊性
- 相同字母代表相同数字
:::

### 典型例题

**例1：加法竖式谜**
在□里填上合适的数字。
```
  3□6
+ □54
------
  8□0
```

**解答：**
- 个位：6 + 4 = 10，写0进1
- 十位：□ + 5 + 1 = □0，即□ + 6 的末位是0
  - 所以十位是4（4 + 6 = 10），写0进1
- 百位：3 + □ + 1 = 8
  - □ = 4
- 答案：346 + 454 = 800

**例2：减法竖式谜**
在□里填上合适的数字。
```
  7□3
- 2□5
------
  □98
```

**解答：**
- 个位：3 - 5 不够，向十位借1，13 - 5 = 8 ✓
- 十位：被借走1后，□ - 1 - □ = 9
  - 说明十位被借后不够减，要向百位借1
  - (□ + 10) - 1 - □ = 9，即 □ + 9 - □ = 9
  - 被减数十位可以是任意数，但减数十位必须是0（因为被借后+9-0=9）
  - 被减数十位 = 0（因为被个位借走1后变成9，9-0=9）
- 百位：7 - 1 - 2 = 4
- 答案：703 - 205 = 498

**例3：乘法竖式谜**
在□里填上合适的数字。
```
   □7
×  □3
------
   □□1
  □□
------
  □□31
```

**解答：**
- 个位：7 × 3 = 21，写1进2
- 十位：□ × 3 + 2 的末位是□
  - 设第一个乘数的十位是a：3a + 2 的末位需要分析
  - 看最后结果的个位是1，已经确定
  - 结果的十位是3，由第二行的十位和第三行的个位相加得到
  - 设第二个乘数的十位是b
  - 第二行（□7 × 3）：个位1，十位是3a+2的个位
  - 第三行（□7 × b）：个位要和第二行的十位相加得3
  - 尝试a=9：97 × 3 = 291
  - 结果十位是3，所以第三行个位 + 9 = 3 或 13
  - 第三行个位 = 4 或 14（不可能）
  - 97 × b 的个位是4，b=2（7×2=14）
  - 验证：97 × 23 = 2231
  - 97 × 3 = 291，97 × 20 = 1940
  - 291 + 1940 = 2231 ✓
- 答案：97 × 23 = 2231

**例4：字母代表数字**
相同的字母代表相同的数字，不同的字母代表不同的数字。
```
  A B C
+ A B C
-------
  B B C
```

**解答：**
- 个位：C + C = C，所以 C = 0
- 十位：B + B = B，所以 B = 0？但C已经是0了
- 重新考虑进位：B + B + 进位 = B + 10k
  - 如果B + B = B + 10，则B = 10，不可能
  - 如果B + B + 1 = B + 10，则B = 9
- 百位：A + A + 1 = B = 9，所以 A = 4
- 验证：490 + 490 = 980 ≠ 990
- 重新分析：A + A + 1 = 9，A = 4，但结果是BB C = 990
- A + A + 1 = 9，需要结果是9，但十位向百位进1
- 所以 A + A + 1 = 9，A = 4
- 490 + 490 = 980，不等于990
- 重新考虑：如果十位没有向百位进位
  - A + A = B
  - B + B = B（个位），B = 0
  - A + A = 0，A = 0，重复了
- 考虑C + C = C + 10，C = 10，不可能
- 或者C + C = C（无进位），C = 0
- B + B = B + 10（有进位），B = 10，不可能
- B + B + 1 = B + 10，B = 9
- A + A + 1 = 9 + 10k
  - k=0：A = 4，结果是990，但490+490=980≠990
  - k=1：A + A + 1 = 19，A = 9，与B重复
- 此题无解？或者题目有误
- 如果结果是BBC = 990，则ABC = 495
- 495 + 495 = 990 ✓
- 所以 A=4, B=9, C=5
- 验证个位：5+5=10，写0进1？但C=5，不是0
- 重新理解：C + C 的个位是C
  - 5 + 5 = 10，个位是0 ≠ 5
  - 0 + 0 = 0 ✓
  - 所以C = 0
- 但495+495=990中C=5，矛盾
- 可能题目应为 A B C + A B C = B C C 或其他形式
- 按ABC+ABC=BBC且C=0：
  - B+B=B（十位），不可能（除非B=0，但C=0）
  - 或B+B+1=B+10，B=9
  - A+A+1=9，A=4
  - 490+490=980≠990
- 结论：此题按给定形式无解，或需调整题目

在□里填上合适的数字。
```
   2□
×  □4
------
   □□
  □□
------
  □08
```
<details>
<summary>点击查看解答</summary>

**解答：**
- 结果个位是8，4 × □ 末位是8，□ = 2或7
- 尝试22 × 14 = 308 ✓
- 验证：22 × 4 = 88，22 × 10 = 220
- 88 + 220 = 308 ✓
- 答案：22 × 14 = 308
</details>

**变式3：除法谜**
在□里填上合适的数字。
```
    □□
  -----
7)□□□
   □□
   ---
    □□
    □□
    ---
     0
```
<details>
<summary>点击查看解答</summary>

**解答：**
- 被除数是三位数，商是两位数
- 7 × □□ = □□□
- 尝试：7 × 12 = 84（两位数，不符合）
- 7 × 15 = 105
- 验证：105 ÷ 7 = 15 ✓
- 答案：105 ÷ 7 = 15
</details>

**变式4：文字谜**
"我爱数学"各代表一个不同的数字。
```
   我爱
 + 数学
 ------
  数学爱
```
<details>
<summary>点击查看解答</summary>

**解答：**
- 两位数加两位数得三位数，百位一定是1
- 所以"数" = 1
- 个位：爱 + 学 = 爱 或 爱 + 10
  - 所以学 = 0 或 学 = 10（不可能）
  - 学 = 0
- 十位：我 + 数 = 学 或 学 + 10（考虑进位）
  - 我 + 1 = 10（因为个位没有进位，学=0）
  - 我 = 9
- 验证：90 + 10 = 100
- 但"数学爱" = 100，"爱" = 0，与"学" = 0 重复
- 重新考虑：个位爱 + 学 = 爱 + 10（有进位）
  - 学 = 10，不可能
- 或者：爱 + 学 = 爱（无进位），学 = 0
- 十位：我 + 数 + 0 = 学 + 10 = 10
  - 我 + 1 = 10，我 = 9
- 结果：90 + 10 = 100，"爱" = 0 = "学"，矛盾
- 调整理解："数学爱"是三位数，数=1，学=0，爱=?
- 如果爱 = 2：92 + 10 = 102，"爱" = 2 ✓
- 验证：92 + 10 = 102
- 我=9，爱=2，数=1，学=0
- 答：我=9，爱=2，数=1，学=0
</details>

### 常见错误

**错误1：忽略进位**
做加法竖式谜时，忘记考虑进位。

**正确：** 每一步都要考虑是否有进位，进位要加到高位。

**错误2：忽略借位**
做减法竖式谜时，忘记考虑借位。

**正确：** 不够减时要向高位借1，被借的位要减1。

**错误3：相同字母代表不同数字**
在字母谜中，给相同的字母赋不同的值。

**正确：** 相同的字母必须代表相同的数字。

**错误4：不验证**
填完数字后不验算。

**正确：** 竖式谜填完后，一定要完整计算一遍验证。

### 亲子互动

**给家长的辅导建议：**

1. **从简单入手**
   - 先练一位数的竖式谜
   - 再逐步增加难度

2. **强调数位对齐**
   - 竖式谜中数位对齐很重要
   - 进位借位要标清楚

3. **培养推理能力**
   - 不要急于试数
   - 先分析，缩小范围，再尝试

4. **一题多解**
   - 有些竖式谜有多个答案
   - 鼓励孩子找出所有可能

**亲子小游戏：**

**"数字侦探"**
- 家长设计竖式谜，孩子来解
- 交换角色，孩子设计，家长解

**"24点游戏"**
- 用四个数字，通过加减乘除得到24
- 培养数字敏感度

**生活中的问题：**
- 找一找生活中需要推理的数字问题
- 如：密码锁、数独游戏
