## 知识点卡片

### 生活小故事

周末，小明和妈妈去超市买东西。

"这个笔记本5元8角。"小明看着价签说。

"在超市里，5元8角通常写成5.80元。"妈妈指着价签说，"这个小圆点叫小数点，5.80就是小数。"

小明好奇地看着周围：苹果3.50元/斤，牛奶12.90元/盒，巧克力18.50元/袋...

"小数点左边是元，右边是角和分。"妈妈解释道，"5.80元就是5元8角0分。"

回到家，小明量了自己的身高："1米35厘米，写成小数就是1.35米！"

他发现：小数就在生活中，让表示价格、长度、重量变得更方便！

### 核心概念

**小数的认识**

像0.5、3.25、12.80这样的数叫做小数。小数中间的点叫做小数点。

**小数的组成**
- 小数点左边：整数部分
- 小数点右边：小数部分

**小数的意义**
- 一位小数表示十分之几
  - 0.1 = 1/10
  - 0.5 = 5/10
- 两位小数表示百分之几
  - 0.01 = 1/100
  - 0.25 = 25/100

**元角分与小数**
- 1元 = 10角 = 100分
- 3.50元 = 3元5角
- 2.08元 = 2元8分

**长度与小数**
- 1米 = 10分米 = 100厘米
- 1.25米 = 1米25厘米
- 0.80米 = 8分米

**小数的大小比较**
- 先比较整数部分，整数部分大的小数大
- 整数部分相同，比较小数部分，从高位比起
- 如：3.25 > 2.98，1.50 > 1.35

**小数的加减法**
- 小数点对齐，相同数位相加减
- 如：3.5 + 2.8 = 6.3
:::

### 典型例题

**例1：认识小数**
读出下面的小数，并说出它们表示的含义。
- 0.5
- 3.25
- 10.08

**解答：**
- 0.5 读作：零点五，表示5/10
- 3.25 读作：三点二五，表示3又25/100
- 10.08 读作：十点零八，表示10又8/100

**例2：元角分转换**
把下列金额写成用"元"作单位的小数。
- 5元3角
- 8角
- 12元5分

**解答：**
- 5元3角 = 5.30元
- 8角 = 0.80元
- 12元5分 = 12.05元

**例3：小数比较**
比较大小：
- 2.5 和 1.8
- 3.20 和 3.15
- 0.90 和 0.9

**解答：**
- 2.5 > 1.8（整数部分2 > 1）
- 3.20 > 3.15（整数部分相同，十分位2 > 1）
- 0.90 = 0.9（小数末尾添0或去0，大小不变）

**例4：小数加法**
计算：4.5 + 2.8

**解答：**
```
  4.5
+ 2.8
-----
  7.3
```
- 小数点对齐
- 5 + 8 = 13，写3进1
- 4 + 2 + 1 = 7
- 结果：7.3

**例5：小数减法**
计算：6.2 - 3.5

**解答：**
```
  6.2
- 3.5
-----
  2.7
```
- 小数点对齐
- 2减5不够，向个位借1，12 - 5 = 7
- 5 - 3 = 2
- 结果：2.7
:::

### 变式训练

**变式1：单位换算**
把下列长度写成用"米"作单位的小数。
- 1米5分米
- 8分米
- 2米35厘米

**解答：**
- 1米5分米 = 1.50米
- 8分米 = 0.80米
- 2米35厘米 = 2.35米
</details>

**变式2：排序**
把下列小数按从小到大的顺序排列：0.8、0.08、0.88、0.80、8.0
<details>
<summary>点击查看解答</summary>

**解答：**
- 0.08 < 0.8 = 0.80 < 0.88 < 8.0
- 注意：0.8 = 0.80
</details>

**变式3：实际问题**
小明有10元钱，买了一支钢笔6.50元，还剩多少钱？如果再用剩下的钱买一本2.80元的笔记本，够吗？
<details>
<summary>点击查看解答</summary>

**解答：**
- 10 - 6.50 = 3.50（元）
- 3.50 > 2.80，够
- 答：还剩3.50元，够买笔记本。
</details>

**变式4：错中求解**
小明做小数加法时，把其中一个加数3.2看成了32，结果得到48.5。正确的和应该是多少？
<details>
<summary>点击查看解答</summary>

**解答：**
- 先求另一个加数：48.5 - 32 = 16.5
- 再算正确的和：16.5 + 3.2 = 19.7
- 答：正确的和是19.7。
</details>

### 常见错误

**错误1：小数点不对齐**
计算：3.5 + 2.8
```
  3.5
+  2.8
------
```
错因：小数点没有对齐，导致数位对错。

**正确：**
```
  3.5
+ 2.8
-----
  6.3
```

**错误2：读小数时读错**
3.05读作：三点五（错误）

**正确：** 3.05读作：三点零五。小数部分要依次读出每个数字。

**错误3：认为小数都比1小**
认为所有小数都小于1。

**正确：** 小数可以大于1，如3.5、10.8都大于1。只有整数部分是0的小数才小于1。

**错误4：单位换算错误**
5元3分 = 5.30元（错误）

**正确：** 5元3分 = 5.03元。3分是百分位，不是十分位。

### 亲子互动

**给家长的辅导建议：**

1. **联系生活实际**
   - 去超市看价签，读小数
   - 量身高、称体重，用小数表示

2. **强调小数点对齐**
   - 计算时先对齐小数点
   - 可以用尺子比着画竖式

3. **理解小数的含义**
   - 0.5元 = 5角，让孩子感受具体金额
   - 0.5米 = 5分米，用尺子量一量

4. **分数与小数联系**
   - 1/10 = 0.1，1/100 = 0.01
   - 帮助孩子建立联系

**亲子小游戏：**

**"超市购物"**
- 准备一些标有价格的物品图片
- 让孩子计算买两样东西要多少钱
- 如：牛奶3.50元 + 面包2.80元 = 6.30元

**"价格猜猜猜"**
- 家长说"三元五角"
- 孩子写成小数"3.50元"
- 互相出题，看谁写得又快又准

**生活中的问题：**
- 一支铅笔0.80元，买5支要多少钱？
- 你的身高是多少米？（用小数表示）
- 10元钱，买了一本5.50元的书，还能买一支2.80元的笔吗？
:::
