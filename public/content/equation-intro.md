# 简易方程 [难度：★]

## 📌 它是什么？
用字母表示未知数，根据等量关系列出含有未知数的等式（方程），再通过运算求出未知数的值。

## 🎯 为什么重要？
方程是数学从"算术思维"迈向"代数思维"的关键一步。学会方程后，很多复杂的应用题可以变得非常简单。

## 📋 需要什么基础？
- 熟练的四则运算能力
- 理解等号"两边相等"的含义
- 会用字母表示数（如 a、x、n）

### 三个核心概念

**等式**：含有等号（=）的式子，表示左右两边的值相等。
- 例：3 + 2 = 5，7 × 4 = 28

**未知数**：用字母表示还不知道的数，通常用 x。
- 例：x 颗玻璃球、n 个同学

**方程**：含有未知数的等式。
- 例：x = 5，x + 3 = 8，2x - 1 = 9

> **判断方程的两个条件**：① 是等式（有 = 号）；② 含有未知数。缺一不可！
>
> **解方程的原理**：等式两边同时加上/减去/乘以/除以同一个数（不为0），等式仍然成立。
:::

::: example-card
### 母题：解一元一次方程
**题目**：解方程 x + 3 = 8

**第一步**：写出方程
x + 3 = 8

**第二步**：等式两边同时减去 3
x + 3 - 3 = 8 - 3

**第三步**：化简得到
x = 5

**第四步**：验算
把 x = 5 代入原方程：5 + 3 = 8 ✔ 等式成立！

**核心方法**：利用等式的性质，把未知数"孤立"到等号的一边。
:::

::: variant-card
### 变式一：减法方程
解方程：x - 7 = 15

**思路**：等式两边同时加上 7，把 x 留在左边。
**解答**：x - 7 + 7 = 15 + 7，所以 x = 22。
**验算**：22 - 7 = 15 ✔
:::

::: variant-card
### 变式二：乘法方程
解方程：4x = 36

**思路**：等式两边同时除以 4，把 x 的系数化为 1。
**解答**：4x ÷ 4 = 36 ÷ 4，所以 x = 9。
**验算**：4 × 9 = 36 ✔
:::

::: mistake-card
### 常见错误：忘记等式两边同时操作
**错误**：解 2x = 10 时，直接写 x = 10 - 2 = 8。
**原因**：把方程当成算术式，没有用"等式性质"来思考。
**层级**：L2 概念错误
**纠正**：牢记——要对等式两边做同样的运算。2x = 10 应该两边同时除以 2，得到 x = 5。
:::
