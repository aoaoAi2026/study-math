# 数列规律 [难度：★★]

## 📌 它是什么？
学习观察数列中数字的变化规律，能根据规律填出缺失的数，并能创造简单的有规律的数列。

## 🎯 为什么重要？
找规律是数学发现的核心能力。数列规律培养孩子的观察力、归纳能力和数感，是函数思想的萌芽。

## 📋 需要什么基础？
- 熟练掌握加减乘除
- 理解"重复""递增""递减"的概念
- 会计算相邻数的差

### 常见数列规律

**规律一：等差数列**
相邻两个数的差相等。
$$2、5、8、11、14... \quad (每次 +3)$$
$$20、17、14、11、8... \quad (每次 -3)$$

**规律二：等比数列**
相邻两个数的商相等（倍数关系）。
$$2、4、8、16、32... \quad (每次 \times 2)$$
$$81、27、9、3、1... \quad (每次 \div 3)$$

**规律三：斐波那契数列**
从第三项开始，每个数等于前两个数之和。
$$1、1、2、3、5、8、13、21...$$

**规律四：双重数列**
两个数列交叉排列。
$$1、10、2、20、3、30、4、40...$$
（单数位：1、2、3、4...；双数位：10、20、30、40...）

**规律五：平方数列**
$$1、4、9、16、25、36... \quad (1^2、2^2、3^2、4^2...)$$

**规律六：图形数**
三角形数：$1、3、6、10、15...$

**找规律的方法**：
1. 算相邻两数的差
2. 算相邻两数的商
3. 隔一个数看（双重数列）
4. 看是否与前几个数有关

**记忆口诀**：
> 数列规律不难找，
> 先算差来后算商，
> 隔项看看也不错，
> 前项相加也有可能。
:::

### 母题：找规律填数
**题目**：找出规律，填出括号里的数。
$$3、6、9、12、(\quad)、(\quad)$$

**第一步**：算相邻数的差
$$6 - 3 = 3$$
$$9 - 6 = 3$$
$$12 - 9 = 3$$

**第二步**：发现规律
每次增加 3，这是一个等差数列。

**第三步**：填数
$$12 + 3 = 15$$
$$15 + 3 = 18$$

**答案**：$3、6、9、12、\mathbf{15}、\mathbf{18}$

**验证**：$15 - 12 = 3$，$18 - 15 = 3$ ✓
:::

::: variant-card
### 变式一：递减数列
**题目**：找出规律，填出括号里的数。
$$50、45、40、35、(\quad)、(\quad)、(\quad)$$

**第一步**：算相邻数的差
$$50 - 45 = 5$$
$$45 - 40 = 5$$
$$40 - 35 = 5$$

**第二步**：发现规律
每次减少 5。

**第三步**：填数
$$35 - 5 = 30$$
$$30 - 5 = 25$$
$$25 - 5 = 20$$

**答案**：$50、45、40、35、\mathbf{30}、\mathbf{25}、\mathbf{20}$
:::

::: variant-card
### 变式二：倍数数列
**题目**：找出规律，填出括号里的数。
$$2、6、18、54、(\quad)、(\quad)$$

**第一步**：算相邻数的差
$$6 - 2 = 4$$
$$18 - 6 = 12$$
差不一样，不是等差数列。

**第二步**：算相邻数的商
$$6 \div 2 = 3$$
$$18 \div 6 = 3$$
$$54 \div 18 = 3$$

**第三步**：发现规律
每次乘 3。

**第四步**：填数
$$54 \times 3 = 162$$
$$162 \times 3 = 486$$

**答案**：$2、6、18、54、\mathbf{162}、\mathbf{486}$

> 技巧：差不一样时，试试算商（倍数关系）。
:::

::: variant-card
### 变式三：斐波那契型数列
**题目**：找出规律，填出括号里的数。
$$2、3、5、8、13、(\quad)、(\quad)$$

**第一步**：算相邻数的差
$$3 - 2 = 1$$
$$5 - 3 = 2$$
$$8 - 5 = 3$$
$$13 - 8 = 5$$
差是 1、2、3、5... 本身也有规律！

**第二步**：发现规律
每个数等于前两个数之和：
$$2 + 3 = 5$$
$$3 + 5 = 8$$
$$5 + 8 = 13$$

**第三步**：填数
$$8 + 13 = 21$$
$$13 + 21 = 34$$

**答案**：$2、3、5、8、13、\mathbf{21}、\mathbf{34}$

> 这就是斐波那契数列的变种！兔子数列真神奇。
:::

::: variant-card
### 变式四：双重数列
**题目**：找出规律，填出括号里的数。
$$1、5、2、10、3、15、4、(\quad)、(\quad)$$

**第一步**：隔一个数看
- 单数位（第 1、3、5、7... 位）：1、2、3、4...
- 双数位（第 2、4、6、8... 位）：5、10、15...

**第二步**：发现规律
- 单数位：每次 +1
- 双数位：每次 +5

**第三步**：填数
- 第 8 位（双数位）：$15 + 5 = 20$
- 第 9 位（单数位）：$4 + 1 = 5$

**答案**：$1、5、2、10、3、15、4、\mathbf{20}、\mathbf{5}$

> 技巧：忽大忽小的数列，试试隔一个数看！
:::

::: mistake-card
### 常见错误

**错误 1**：只看前两个数就确定规律
$$\text{错：} 2、4、(\quad)，\text{填 6（以为每次 +2）}$$
**纠正**：如果后面还有 8、16，规律就是每次乘 2，应该填 8。要多看几个数再确定规律。

**错误 2**：忽略数列的多种可能
$$\text{错：} 1、2、4、(\quad)，\text{只想到填 8（每次乘 2）}$$
**纠正**：也可能是差递增：$1、2、4、7、11$（差为 1、2、3、4）。要看完整题目给出的数。

**错误 3**：计算错误
$$\text{错：} 54 \times 3 = 152$$
**纠正**：$50 \times 3 = 150$，$4 \times 3 = 12$，$150 + 12 = 162$。
:::

### 亲子共学

**家庭小游戏：规律接龙**

1. 家长说一个数列的前 3 个数，孩子接龙说出后面的数
2. 交换角色，孩子出题，家长接龙
3. 比一比谁发现的规律多

**生活中的数列**：
- 楼梯台阶：1、2、3、4、5...（每次 +1）
- 一周天数：7、14、21、28...（每次 +7）
- 月份天数：31、28、31、30...（大小月规律）

**创造数列**：
让孩子自己创造有规律的数列，考考家长：
- "我创造的数列：3、6、12、24... 规律是什么？"
- "再来一个：1、4、7、10... 下一个是什么？"

**记忆口诀**：
> 找规律，别着急，
> 先看差，后看商，
> 隔项观察也试试，
> 前项相加有可能，
> 多写几个来验证。
