# 列表推理 [难度：★★★]

## 📌 它是什么？
学习用表格整理信息，通过排除法进行逻辑推理，解决"谁是...""哪个是..."等判断问题。

## 🎯 为什么重要？
列表推理是逻辑思维的重要工具，能帮助孩子有条理地整理信息、排除错误选项。这是数学建模和逻辑推理的基础。

## 📋 需要什么基础？
- 理解"是""不是"的判断
- 会画简单的表格
- 有基本的逻辑判断能力

### 侦探小明的推理
小明、小红、小刚三个人参加了数学、语文、英语三个兴趣小组，每人只参加一个。

已知：
1. 小明说："我不参加英语小组。"
2. 小红说："我不参加数学小组。"
3. 小刚说："我看到小明和小红都在语文小组。"——等等，这不可能，每人只能参加一个！

老师笑着说："小刚在开玩笑。正确的线索是：小刚参加数学小组。"

小明拿出纸和笔，画了一个表格：

| | 数学 | 语文 | 英语 |
|---|---|---|---|
| 小明 | | | |
| 小红 | | | |
| 小刚 | √ | × | × |

小明推理：小刚参加数学，那小明和小红都不参加数学。小明不参加英语，那小明只能参加语文。小红不参加数学，也不参加语文（因为小明参加了），所以小红参加英语。

小明得意地说："我推理出来了！"
:::

### 列表推理法

**核心方法**：
> 把信息填入表格，用"√"表示"是"，用"×"表示"不是"，通过排除找到答案。

**推理步骤**：
1. **画表格**：根据题目确定行和列
2. **填已知**：把确定的对应关系填入表格
3. **做排除**：根据已知条件排除不可能的情况
4. **得结论**：每个行和列只能有一个"√"

**常用技巧**：

**技巧一：直接确定**
已知"小明参加语文"，直接打"√"，这一行和这一列的其他位置打"×"。

**技巧二：排除法**
已知"小明不参加英语"，在小明-英语的位置打"×"。

**技巧三：唯一选择**
如果某一行已经有两个"×"，剩下的那个一定是"√"。

**记忆口诀**：
> 列表推理有方法，
> 画好表格填信息，
> 是打勾来否打叉，
> 排除法儿最厉害，
> 每行每列只一个。
:::

### 母题：兴趣小组推理
**题目**：小明、小红、小刚三人分别参加了美术、音乐、体育三个兴趣小组。已知：
1. 小明不参加美术小组
2. 小红不参加体育小组
3. 小刚参加音乐小组

问：三人各参加什么小组？

**第一步**：画表格

| | 美术 | 音乐 | 体育 |
|---|---|---|---|
| 小明 | | | |
| 小红 | | | |
| 小刚 | | | |

**第二步**：填已知
- 小刚参加音乐 → 小刚-音乐打"√"
- 小刚参加音乐 → 小刚-美术、小刚-体育打"×"
- 小刚参加音乐 → 小明-音乐、小红-音乐打"×"

| | 美术 | 音乐 | 体育 |
|---|---|---|---|
| 小明 | | × | |
| 小红 | | × | |
| 小刚 | × | √ | × |

**第三步**：继续推理
- 小明不参加美术 → 小明-美术打"×"
- 小明-美术×、小明-音乐× → 小明只能参加体育 → 小明-体育打"√"
- 小明参加体育 → 小红-体育打"×"
- 小红-音乐×、小红-体育× → 小红只能参加美术 → 小红-美术打"√"

**第四步**：完成表格

| | 美术 | 音乐 | 体育 |
|---|---|---|---|
| 小明 | × | × | √ |
| 小红 | √ | × | × |
| 小刚 | × | √ | × |

**答**：小明参加体育小组，小红参加美术小组，小刚参加音乐小组。

### 变式一：比赛名次推理
**题目**：小明、小红、小刚参加跑步比赛，得了前三名（没有并列）。已知：
1. 小明不是第一名
2. 小红不是第二名
3. 小刚不是第三名
4. 小红比小刚名次高

问：三人的名次各是多少？

**第一步**：画表格

| | 第一名 | 第二名 | 第三名 |
|---|---|---|---|
| 小明 | | | |
| 小红 | | | |
| 小刚 | | | |

**第二步**：填已知
- 小明不是第一 → 小明-第一打"×"
- 小红不是第二 → 小红-第二打"×"
- 小刚不是第三 → 小刚-第三打"×"
- 小红比小刚高 → 小红名次数字 < 小刚名次数字

**第三步**：推理
- 小红比小刚高，小红不是第二
  - 如果小红是第三，小刚比她低，不可能
  - 所以小红是第一名
- 小红是第一 → 小红-第一"√"，小明-第一"×"（已×），小刚-第一"×"
- 小刚不是第三，也不是第一 → 小刚是第二
- 小刚是第二 → 小明是第三

**验证**：
- 小明第三，不是第一 ✓
- 小红第一，不是第二 ✓
- 小刚第二，不是第三 ✓
- 小红（1）比小刚（2）高 ✓

**答**：小红第一名，小刚第二名，小明第三名。
:::

### 变式二：职业推理
**题目**：甲、乙、丙三人分别是医生、教师、工程师。已知：
1. 甲不是医生
2. 乙不是教师
3. 丙不是工程师
4. 甲和乙都不是工程师

问：三人的职业各是什么？

**第一步**：画表格

| | 医生 | 教师 | 工程师 |
|---|---|---|---|
| 甲 | | | |
| 乙 | | | |
| 丙 | | | |

**第二步**：填已知
- 甲不是医生 → 甲-医生"×"
- 乙不是教师 → 乙-教师"×"
- 丙不是工程师 → 丙-工程师"×"
- 甲不是工程师 → 甲-工程师"×"
- 乙不是工程师 → 乙-工程师"×"

**第三步**：推理
- 甲-工程师×、乙-工程师× → 丙是工程师
- 但丙-工程师×（已知条件）！

等等，条件 3 和条件 4 矛盾了？让我重新检查...

哦，条件 4 说"甲和乙都不是工程师"，那丙一定是工程师。但条件 3 说"丙不是工程师"。

这说明题目条件有矛盾！如果去掉条件 3：
- 丙是工程师
- 甲不是医生、不是工程师 → 甲是教师
- 乙不是教师、不是工程师 → 乙是医生

**答**：甲是教师，乙是医生，丙是工程师。

> 注意：如果推理中出现矛盾，要检查题目条件是否有误。

### 变式三：综合推理
**题目**：三个小朋友分别姓张、王、李，他们分别喜欢红、黄、蓝三种颜色。已知：
1. 姓张的小朋友不喜欢红色
2. 姓王的小朋友不喜欢黄色
3. 喜欢蓝色的小朋友姓李
4. 姓张的小朋友喜欢黄色

问：三个小朋友各喜欢什么颜色？

**第一步**：画表格

| | 红 | 黄 | 蓝 |
|---|---|---|---|
| 张 | | | |
| 王 | | | |
| 李 | | | |

**第二步**：填已知
- 姓张不喜欢红 → 张-红"×"
- 姓王不喜欢黄 → 王-黄"×"
- 喜欢蓝的姓李 → 李-蓝"√"
- 姓张喜欢黄 → 张-黄"√"

**第三步**：推理
- 李-蓝"√" → 李-红"×"、李-黄"×"、张-蓝"×"、王-蓝"×"
- 张-黄"√" → 张-红"×"（已×）、张-蓝"×"（已×）、王-黄"×"（已×）
- 张-黄√、李-蓝√ → 王只能喜欢红色 → 王-红"√"

**完成表格**：

| | 红 | 黄 | 蓝 |
|---|---|---|---|
| 张 | × | √ | × |
| 王 | √ | × | × |
| 李 | × | × | √ |

**答**：姓张的喜欢黄色，姓王的喜欢红色，姓李的喜欢蓝色。
:::

::: mistake-card
### 常见错误

**错误 1**：表格画错行列
$$\text{错：} \text{把人名当列，把属性当行}$$
**纠正**：通常把人名或物品名当行，属性当列，但也可以反过来，关键是保持一致。

**错误 2**：忘记排除
$$\text{错：} \text{填了一个"√"后，忘记在同一行同一列打"×"}$$
**纠正**：每个"√"确定后，立即把同行同列的其他位置打"×"。

**错误 3**：条件理解错误
$$\text{错：} \text{"甲不是乙"理解成"甲是乙"}$$
**纠正**：仔细读题，"不是"打"×"，"是"打"√"。

### 亲子共学

**家庭小游戏：家庭推理**

1. 爸爸、妈妈、孩子三人，分别喜欢三种水果
2. 给出几个线索，让孩子用表格推理
3. 交换角色，孩子出题，家长来推理

**示例题目**：
"小明、小红、小刚分别养了猫、狗、兔。小明不养猫，小红养的是会汪汪叫的动物。谁养了什么？"

**推理故事**：
给孩子讲侦探故事，然后用列表法"破案"：
"三个嫌疑人，三种作案工具，根据线索找出真凶..."

**记忆口诀**：
> 逻辑推理画表格，
> 行列对应要清楚，
> 已知条件先填入，
> 排除法儿来帮助，
> 每行每列只一个，
> 答案自然现出来。
