# 有序数图形 [难度：★★★]

## 📌 它是什么？
学习按照一定的顺序和方法数图形（线段、角、三角形、长方形等），做到不重复、不遗漏。

## 🎯 为什么重要？
有序思考是数学思维的核心。数图形问题培养孩子的观察力、分类能力和有序思考习惯，是组合数学的启蒙。

## 📋 需要什么基础？
- 认识基本图形（线段、角、三角形、长方形）
- 理解"按顺序"的概念
- 熟练掌握加法

### 数线段比赛
数学课上，老师画了一条线段，上面有几个点：
```
A —— B —— C —— D
```

老师说："这条线段上一共有几条线段？"

小马虎马上说："3 条！AB、BC、CD。"

小明说："不对，还有 AC、BD、AD，一共 6 条！"

老师点点头："小明说得对。数图形要按顺序，从每个点出发，依次往后数。"

小马虎明白了：原来数图形不能只看最明显的，要按照规律来数！

### 有序数图形的方法

**核心方法**：
> 按顺序数，标序号，分类数，不重复，不遗漏。

**数线段**：
给点标上字母 A、B、C、D...
- 以 A 为起点的：AB、AC、AD...
- 以 B 为起点的：BC、BD...
- 以 C 为起点的：CD...
- 总数 = $3 + 2 + 1 = 6$

**数角**：
给射线标上序号 1、2、3、4...
- 由 1 个基本角组成的：3 个
- 由 2 个基本角组成的：2 个
- 由 3 个基本角组成的：1 个
- 总数 = $3 + 2 + 1 = 6$

**数三角形**：
- 由 1 个基本三角形组成的
- 由 2 个基本三角形组成的
- 由 3 个基本三角形组成的
- ...以此类推

**公式规律**：
如果有 n 个基本图形排成一排：
$$总数 = n + (n-1) + (n-2) + \cdots + 1 = \frac{n \times (n+1)}{2}$$

**记忆口诀**：
> 数图形，要有序，
> 标上号，分类数，
> 单个的，组合的，
> 加起来，不遗漏。
:::

### 母题：数线段
**题目**：下图中有几条线段？
```
A —— B —— C —— D —— E
```

**方法一：按起点分类**
- 以 A 为起点的：AB、AC、AD、AE（4 条）
- 以 B 为起点的：BC、BD、BE（3 条）
- 以 C 为起点的：CD、CE（2 条）
- 以 D 为起点的：DE（1 条）

**第二步**：相加
$$4 + 3 + 2 + 1 = 10 \text{（条）}$$

**方法二：按基本线段数**
- 基本线段（1 段）：AB、BC、CD、DE（4 条）
- 由 2 条组成的：AC、BD、CE（3 条）
- 由 3 条组成的：AD、BE（2 条）
- 由 4 条组成的：AE（1 条）

$$4 + 3 + 2 + 1 = 10 \text{（条）}$$

**答**：一共有 10 条线段。

> 规律：5 个点，线段数 = $4 + 3 + 2 + 1 = 10$
:::

::: variant-card
### 变式一：数角
**题目**：下图中有几个角？
```
      /
     /
    /
   /
```
（从一个顶点出发有 4 条射线）

**分析**：把射线标上序号 1、2、3、4。

**按基本角数**：
- 由 1 个基本角组成的：3 个
- 由 2 个基本角组成的：2 个
- 由 3 个基本角组成的：1 个

**计算**：
$$3 + 2 + 1 = 6 \text{（个）}$$

**答**：一共有 6 个角。

> 规律：4 条射线，角数 = $3 + 2 + 1 = 6$
:::

::: variant-card
### 变式二：数三角形
**题目**：下图中有几个三角形？
```
    /\
   /  \
  /____\
  /\    /\
 /  \  /  \
/____\/____\
```
（一个大三角形被分成 3 个小三角形）

**分析**：按组成个数分类。

**第一步**：数单个的
- 基本三角形：3 个

**第二步**：数组合的
- 由 2 个基本三角形组成的：0 个（因为 2 个拼起来不是三角形）
- 由 3 个基本三角形组成的：1 个（最大的那个）

**计算**：
$$3 + 1 = 4 \text{（个）}$$

**答**：一共有 4 个三角形。

> 注意：三角形不一定都能两两组合成新三角形，要仔细观察！
:::

::: variant-card
### 变式三：数长方形
**题目**：下图中有几个长方形？
```
+----+----+
|    |    |
+----+----+
|    |    |
+----+----+
```
（2 行 2 列的格子）

**分析**：长方形由长和宽决定，分别数长和宽的组合。

**数长边的线段数**（横向）：
$$2 + 1 = 3 \text{（条）}$$

**数宽边的线段数**（纵向）：
$$2 + 1 = 3 \text{（条）}$$

**长方形总数**：
$$3 \times 3 = 9 \text{（个）}$$

**验证**：
- 1×1 的小长方形：4 个
- 1×2 的长方形：2 个（横向）+ 2 个（纵向）= 4 个
- 2×2 的大长方形：1 个
- 一共：$4 + 4 + 1 = 9$ 个 ✓

**答**：一共有 9 个长方形。

> 规律：m 列 n 行的格子，长方形数 = (m + m-1 + ... + 1) × (n + n-1 + ... + 1)
:::

::: mistake-card
### 常见错误

**错误 1**：只数最基本的，遗漏组合的
$$\text{错：} AB、BC、CD \text{ 共 3 条线段}$$
**纠正**：还有 AC、AD、BD 等组合的线段，一共 6 条。

**错误 2**：重复计数
$$\text{错：} \text{数角时，同一个角数了两次}$$
**纠正**：按顺序数，从一条边出发，依次和后面的边组成角。

**错误 3**：把不是三角形的当成三角形
$$\text{错：} \text{两个三角形拼成的四边形也算成三角形}$$
**纠正**：三角形必须有 3 条边 3 个角，四边形不是三角形。
:::

### 亲子共学

**家庭小游戏：图形大搜索**

1. 在房间里找长方形（窗户、门、书本）
2. 数一数窗户被分成几个小长方形，一共有几个长方形
3. 用火柴棒摆三角形，数一数有几个

**画图练习**：
让孩子自己画：
- 在一条直线上画 4 个点，数线段
- 从一点画 4 条射线，数角
- 画一个被分割的三角形，数三角形

**规律探索**：
| 基本图形数 | 线段/角总数 |
|---|---|
| 2 | 3 |
| 3 | 6 |
| 4 | 10 |
| 5 | 15 |

让孩子发现规律：总数 = $n + (n-1) + \cdots + 1$

**记忆口诀**：
> 数图形，不着急，
> 标上序号按顺序，
> 单个组合都数到，
> 加在一起是总数。
