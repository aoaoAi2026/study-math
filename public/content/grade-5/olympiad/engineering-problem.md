# 工程问题

## 📌 它是什么？

工程问题是小学数学奥数中的经典题型，它研究"工作总量、工作效率、工作时间"三者之间的关系。工程问题的核心是：把工作总量看作单位"1"，用"1/天数"表示工作效率（每天完成几分之几）。典型的工程问题包括：两人合作、多人合作、轮流工作、中途休息、进水放水等多种变化形式。

## 🎯 为什么重要？

工程问题是分数应用题的重要应用，是小升初和各类奥数竞赛的必考题型。它不仅能锻炼孩子的逻辑思维能力，还能培养"建模"思想——把一个实际问题转化为数学模型来解决。工程问题的解题方法（假设法、方程法、整体法）也是解决其他数学问题的通用方法。掌握工程问题，孩子的数学思维水平将上一个台阶。

## 📋 需要什么基础？

- 熟练掌握分数的加减法和乘除法
- 理解分数应用题的量率对应
- 会用方程解应用题
- 具备一定的逻辑推理能力
- 理解"单位1"的思想

小明家买了一套新房子，准备装修啦！爸爸给小明出了一道有趣的数学题。

"小明，我们家要请装修队来刷墙。如果请甲装修队单独做，需要12天完成；如果请乙装修队单独做，需要18天完成。如果两队一起做，需要多少天完成？"

小明想了想："甲队12天做完，每天做 1/12；乙队18天做完，每天做 1/18。两队一起做，每天做 1/12 + 1/18 = 3/36 + 2/36 = 5/36。需要 1 ÷ 5/36 = 36/5 = 7.2（天）！"

爸爸笑着说："太棒了！这就是工程问题的基本解法。工作总量看作单位'1'，工作效率就是 1/天数！"

过了几天，爸爸又出题了："一项工程，甲队独做需要20天，乙队独做需要30天。现在甲队先做了5天后，乙队也加入一起做，还需要多少天完成？"

小明认真地分析：

剩下的由两队一起做，每天做 1/20 + 1/30 = 3/60 + 2/60 = 5/60 = 1/12。

小明陷入了深思……

### 核心概念与方法

**一、基本概念**

1. **工作总量**：把工程的全部工作量看作单位"1"
2. **工作效率**：单位时间内完成的工作量（通常是每天完成几分之几）
3. **工作时间**：完成工作所需的时间

**二、基本公式**

```
工作总量 = 工作效率 × 工作时间
工作效率 = 工作总量 ÷ 工作时间
工作时间 = 工作总量 ÷ 工作效率
```

**三、基本类型与解法**

**类型1：基本合作问题**

特征：已知各队单独完成的时间，求合作完成的时间
方法：先求出各队效率，相加得总效率
公式：
```
甲队效率 = 1/甲单独天数
乙队效率 = 1/乙单独天数
合作效率 = 甲队效率 + 乙队效率
合作时间 = 1 ÷ 合作效率
```

**类型2：先独做后合作**

特征：某队先单独做一段时间，然后加入另一队合作
方法：先算已完成的工作量，再求剩余工作量所需时间

**类型3：中途休息**

特征：合作过程中某队中途休息若干天
方法：假设都没休息，多做的工作量就是休息期间的工作量

**类型4：轮流工作**

特征：两队或多队轮流工作，不是同时做
方法：一个周期（轮）为单位计算

**类型5：三人合作**

特征：三个队一起做
方法：三队效率相加

**类型6：进水放水（水箱问题）**

特征：有进水管和出水管，一个注水一个放水
方法：净效率 = 进水效率 - 放水效率

**四、重要方法**

**方法1：方程法**

设未知数，根据工作总量等于各部分工作量之和列方程

**方法2：假设法**

假设所有队都没休息，计算多做的工作量就是休息期间的工作量

**方法3：整体法**

观察已知条件，通过加、减等运算求整体效率

**五、温馨提示**

工程问题的关键：
1. 始终把工作总量看作单位"1"
2. 工作效率 = 1/工作时间
3. 工作时间 = 1/工作效率
4. 合作效率 = 各队效率之和

### 典型例题精讲

**例1：基本合作**

一项工程，甲队独做需要12天，乙队独做需要18天。两队合作需要多少天完成？

**解答：**

甲队效率 = 1/12
乙队效率 = 1/18
合作效率 = 1/12 + 1/18 = 3/36 + 2/36 = 5/36
合作时间 = 1 ÷ 5/36 = 36/5 = 7.2（天）

答：两队合作需要7.2天完成。

---

**例2：先独做后合作**

一项工程，甲队独做需要20天，乙队独做需要30天。现在甲队先做了5天后，乙队也加入一起做，还需要多少天完成？

**解答：**

甲队效率 = 1/20
乙队效率 = 1/30

甲队先做5天完成：5 × 1/20 = 1/4
剩余工作量：1 - 1/4 = 3/4

合作效率：1/20 + 1/30 = 3/60 + 2/60 = 5/60 = 1/12

还需要的时间：3/4 ÷ 1/12 = 3/4 × 12 = 9（天）

答：还需要9天完成。

---

**例3：中途休息**

一项工程，甲队独做需要20天，乙队独做需要30天。两队合作，中途甲队休息了3天，乙队休息了若干天，从开始到完成共用了16天。乙队休息了多少天？

**解答：**

假设两队都没休息，16天可以完成：
16 × (1/20 + 1/30) = 16 × 5/60 = 16 × 1/12 = 4/3

多做了：4/3 - 1 = 1/3

其中甲队休息3天少做的：3 × 1/20 = 3/20
乙队休息x天少做的：x × 1/30 = x/30

少做的总和应该等于多做的：
3/20 + x/30 = 1/3
9/60 + 2x/60 = 20/60
9 + 2x = 20
2x = 11
x = 5.5

答：乙队休息了5.5天。

---

**例4：三人合作**

一项工程，甲、乙两队合作需要12天完成，乙、丙两队合作需要15天完成，甲、丙两队合作需要20天完成。三队一起做需要多少天完成？

**解答：**

甲+乙 = 1/12
乙+丙 = 1/15
甲+丙 = 1/20

三式相加：
2×(甲+乙+丙) = 1/12 + 1/15 + 1/20
= 5/60 + 4/60 + 3/60 = 12/60 = 1/5

甲+乙+丙 = 1/10

三队一起做需要：1 ÷ 1/10 = 10（天）

答：三队一起做需要10天完成。

---

**例5：轮流工作**

一项工程，甲队独做需要12天，乙队独做需要18天。现在甲、乙两队轮流工作，甲队先做1天，乙队接着做1天，甲队再做1天……如此交替，直到完成。完成这项工程需要多少天？

**解答：**

甲效率 = 1/12，乙效率 = 1/18

一个周期（2天）完成：1/12 + 1/18 = 3/36 + 2/36 = 5/36

7个周期（14天）完成：7 × 5/36 = 35/36

剩余工作量：1 - 35/36 = 1/36

轮到甲队做：1/36 ÷ 1/12 = 1/3（天）

总共需要：14 + 1/3 = 14 1/3（天）

答：完成这项工程需要14 1/3天。

---

**例6：进水放水**

一个水池有两个进水管和一个出水管。单开甲管6小时可以注满，单开乙管8小时可以注满，单开出水管12小时可以放完。三管同时开，几小时可以注满？

**解答：**

甲管进水效率 = 1/6（每小时注入1/6池）
乙管进水效率 = 1/8
出水管放水效率 = 1/12

净效率 = 1/6 + 1/8 - 1/12 = 4/24 + 3/24 - 2/24 = 5/24

注满时间 = 1 ÷ 5/24 = 24/5 = 4.8（小时）

答：4.8小时可以注满。

---

**例7：效率转换**

一项工程，由甲、乙两队合作，12天可以完成。现在由甲队先做8天，剩下的由乙队独做需要20天完成。如果由甲队独做，需要多少天完成？

**解答：**

设甲队效率为a，乙队效率为b

根据题意：
a + b = 1/12 （合作12天完成）
8a + 20b = 1 （甲做8天+乙做20天完成）

由第一个式子：b = 1/12 - a

代入第二个式子：
8a + 20(1/12 - a) = 1
8a + 5/3 - 20a = 1
-12a = 1 - 5/3 = -2/3
a = 1/18

甲队独做需要：1 ÷ 1/18 = 18（天）

答：甲队独做需要18天完成。

### 常见错误分析

**错误一：工作效率与工作时间混淆**

"甲队独做需要10天，乙队独做需要15天，两队合作需要多少天？"
错解：(10+15)/2 = 12.5（天）

**分析：** 合作时间不能用时间的平均数！应该先求效率，再用"1 ÷ 合作效率"求时间。

**改正：**
甲效率 = 1/10，乙效率 = 1/15
合作效率 = 1/10 + 1/15 = 3/30 + 2/30 = 5/30 = 1/6
合作时间 = 1 ÷ 1/6 = 6（天）

---

**错误二：把效率加反**

"甲队独做需要12天，乙队独做需要18天，两队合作多少天完成？"
错解：1 ÷ (12 + 18) = 1/30（天）

**分析：** 工作效率是"1/工作时间"，不是"工作时间"。应该用1/12+1/18，而不是12+18。

**改正：**
合作效率 = 1/12 + 1/18 = 5/36
合作时间 = 1 ÷ 5/36 = 36/5 = 7.2（天）

---

**错误三：不理解"单位1"**

在有具体数量的工程问题中，忘记把工作总量看作单位"1"。

**分析：** 工程问题中，不论工作量具体多大，都把它看作单位"1"。这是工程问题的核心假设。

**改正：** 始终牢记"工作总量=1"，各队效率 = 1/工作时间。

---

**错误四：轮流工作周期数错误**

"甲效率1/10，乙效率1/15，轮流工作"错解：1 ÷ (1/10+1/15) = 6（天）

**分析：** 轮流工作不是合作工作！一个周期（2天）完成1/10+1/15，需要按周期数计算，然后处理最后不完整周期。

**改正：**
一个周期完成：1/10 + 1/15 = 1/6
6个周期（12天）正好完成1
答：12天完成。

### 亲子共学脚本：工程问题的魔法

**场景：** 周末下午，爸爸和小明一起在家攻克工程问题！

**准备道具：** 草稿纸、笔、一些小木块（或硬币）代表工作量

**👨‍👩‍👧‍👦 家长引导：**

**第一步：认识"单位1"（15分钟）**

家长："小明，今天我们来玩一个'工程积木'游戏！"

拿出12个小木块：

家长："假设这12个木块代表一项工程，甲队一天可以做2块，乙队一天可以做1块。甲队单独做完需要多少天？"

孩子："12 ÷ 2 = 6（天）！"

家长："太棒了！乙队单独做完需要多少天？"

孩子："12 ÷ 1 = 12（天）！"

家长："如果把全部工作看作单位'1'，甲队每天做几分之几？"

孩子："2/12 = 1/6！"

家长："乙队呢？"

孩子："1/12！"

家长："完全正确！这就是工程问题的核心：工作效率 = 1/工作时间！"

---

**第二步：合作的力量（20分钟）**

家长："现在两队一起做，每天可以做多少？"

孩子："1/6 + 1/12 = 2/12 + 1/12 = 3/12 = 1/4！"

家长："那么两队一起做需要多少天完成？"

孩子："1 ÷ 1/4 = 4（天）！"

家长："完美！我们验证一下：
每天实际做2+1=3块，12块需要12÷3=4天，完全一致！"

孩子："原来数学真神奇！"

---

**第三步：中途休息的秘密（25分钟）**

家长："现在来个升级版本！一项工程甲独做10天，乙独做15天。两队合作，中途甲休息了2天，乙也休息了几天，从开始到完成共用了11天。乙休息了多少天？"

家长："我们用'假设法'！假设甲乙都没休息，11天可以做多少？"

孩子："11 × (1/10 + 1/15) = 11 × 1/6 = 11/6"

家长："多做了多少？"

孩子："11/6 - 1 = 5/6"

家长："多做的就是休息期间应该做的！甲休息2天少做多少？"

孩子："2 × 1/10 = 1/5"

家长："所以乙休息少做的 = 5/6 - 1/5 = 25/30 - 6/30 = 19/30？不对……"

重新来：

假设都没休息，11天做了：11 × (1/10+1/15) = 11 × 1/6 = 11/6
多做：11/6 - 1 = 5/6

实际上甲休息2天，少做：2/10 = 1/5
乙休息x天，少做：x/15

少做的总量 = 多做的量：
1/5 + x/15 = 5/6
3/15 + x/15 = 5/6
(3+x)/15 = 5/6
6(3+x) = 75
18 + 6x = 75
6x = 57
x = 9.5
