# 沙漏模型（相似三角形）

## 📌 它是什么？

沙漏模型，也叫"八字模型"，是小学奥数几何模块中的重要模型。当两条平行线被两条相交的直线所截时，形成的两个三角形是相似三角形。这个图形看起来就像一个沙漏，所以叫做"沙漏模型"。沙漏模型的核心是：相似三角形的对应边成比例，对应面积比等于相似比的平方。

## 🎯 为什么重要？

沙漏模型是认识相似三角形的入门模型。在小升初和各类奥数竞赛中，沙漏模型经常出现在梯形、组合图形的面积计算问题中。掌握沙漏模型后，可以快速求出：① 两个相似三角形的边长比；② 两个相似三角形的面积比；③ 梯形中对角线交点分割的各个三角形面积关系。沙漏模型还为后续学习更复杂的几何问题（如蝴蝶模型、燕尾模型）打下了坚实的基础。

## 📋 需要什么基础？

- 熟练掌握三角形面积公式
- 理解比和比例的概念
- 理解"平行线间距离相等"
- 具备一定的图形观察能力

小明在数学课上看到老师拿出一个古代的沙漏。

"看这两条平行线 AB 和 CD，再画两条交叉的直线交于点 O。"

"你们发现了什么？△AOB 和 △COD 长得是不是很像？"

小明仔细看了看："好像！△AOB 就像是 △COD 按比例缩小（或放大）了一样！"

老师点点头："对啦！这种'形状相同、大小不同'的三角形，我们叫它们相似三角形。"

"如果 AB = 6，CD = 4，那么 △AOB 和 △COD 的相似比就是 6:4 = 3:2。"

小明问道："那它们的面积比也是 3:2 吗？"

老师笑了笑："这个问题问得好！我们一起来算一算。假设它们的高分别是 3h 和 2h。"

"△COD 的面积 = ½ × 4 × 2h = 4h"

"所以面积比是 9h:4h = 9:4，也就是 3²:2²！"

小明恍然大悟："原来面积比是相似比的平方！这就是沙漏的秘密！"

老师："完全正确！掌握这个规律，很多复杂的几何问题就能迎刃而解啦！"

### 核心概念详解

**一、什么是沙漏模型**

当两条平行线被两条相交直线所截时，形成的两个三角形构成沙漏模型。

```
    A ────── B
     \      /
      \    /
       \  /
        O
       /  \
      /    \
     /      \
    C ────── D
```

AB ∥ CD，直线 AC 与 BD 交于 O，则 △AOB ∽ △COD（"∽"表示相似）。

**二、核心结论**

1. **相似关系**：△AOB ∽ △COD
2. **相似比（边长比）**：
   - AO:CO = BO:DO = AB:CD = k
3. **面积比**：
   - S△AOB : S△COD = k²（相似比的平方）

**三、梯形中的沙漏**

在梯形 ABCD 中，AD ∥ BC，对角线 AC 和 BD 交于 O：

1. △AOD ∽ △COB（注意对应关系！A对应C，O对应O，D对应B）
2. 相似比 = AD:BC
3. 面积比 = (AD:BC)²

**四、重要的面积关系**

在梯形中，除了两个相似三角形外，还有以下重要关系：

1. S△AOB = S△COD（两个"翅膀"面积相等，这是蝴蝶模型的结论）
2. S△AOB : S△AOD = BO : OD（同高三角形，面积比等于底之比）
3. S△AOB : S△BOC = AO : OC

**五、应用类型**

1. 已知边长比 → 求面积比
2. 已知面积比 → 求边长比（开平方）
3. 已知一个三角形面积 → 求另一个相似三角形面积
4. 在复杂图形中识别沙漏模型

**六、公式总结**

- 相似比 = 对应边长之比
- 面积比 = 相似比的平方
- 高之比 = 相似比
- 周长之比 = 相似比

### 典型例题详解

**例题1：基础沙漏——求面积比**

如图，AB ∥ CD，AB = 8，CD = 5，两条直线交于 O。求 △AOB 和 △COD 的面积比。

**解题步骤：**

第一步：判断相似

因为 AB ∥ CD，所以 △AOB ∽ △COD。

第二步：求相似比

相似比 k = AB:CD = 8:5

第三步：求面积比

面积比 = k² = 8²:5² = 64:25

答：△AOB 和 △COD 的面积比是 64:25。

---

**例题2：梯形沙漏——求三角形面积**

梯形 ABCD 中，AD ∥ BC，AD = 4，BC = 6，对角线 AC 和 BD 交于点 O。如果 S△AOD = 8，求 S△BOC。

**解题步骤：**

第一步：判断相似

在梯形中，AD ∥ BC，所以 △AOD ∽ △COB。

注意：A 对应 C，O 对应 O，D 对应 B。

第二步：求相似比

相似比 = AD:BC = 4:6 = 2:3

第三步：求面积比

S△AOD : S△BOC = 2²:3² = 4:9

第四步：求 S△BOC

8 : S△BOC = 4 : 9

S△BOC = 8 × 9 ÷ 4 = 18

答：S△BOC = 18。

---

**例题3：综合应用——求多个面积**

如图，AB ∥ CD，AO:OC = 2:3，S△AOB = 12。求 S△COD 和 S△AOD。

**解题步骤：**

第一步：求 S△COD

相似比 = AO:OC = 2:3

S△AOB : S△COD = 2²:3² = 4:9

12 : S△COD = 4 : 9

S△COD = 12 × 9 ÷ 4 = 27

第二步：求 S△AOD

△AOB 和 △AOD 是同高三角形（从 A 向 BD 作高）

面积比等于底之比：S△AOB : S△AOD = BO : OD

而 BO:OD = 相似比 = 2:3

所以 12 : S△AOD = 2 : 3

S△AOD = 12 × 3 ÷ 2 = 18

答：S△COD = 27，S△AOD = 18。

---

**例题4：沙漏中的距离问题**

梯形 ABCD 中，AD ∥ BC，AD:BC = 1:3，梯形的高为 8。求对角线交点 O 到 AD 和到 BC 的距离分别是多少。

**解题步骤：**

第一步：分析

O 到 AD 的距离 h₁ 加上 O 到 BC 的距离 h₂ 等于梯形的高。

即 h₁ + h₂ = 8

第二步：利用相似三角形

△AOD ∽ △COB，相似比 = 1:3

相似三角形对应高的比等于相似比，所以 h₁:h₂ = 1:3

第三步：求解

h₁ = 8 × 1/(1+3) = 2

h₂ = 8 × 3/(1+3) = 6

答：O 到 AD 的距离是 2，到 BC 的距离是 6。

---

**例题5：复杂图形中的沙漏**

在 △ABC 中，D 在 AB 上，E 在 AC 上，DE ∥ BC。AD:DB = 2:3，S△ADE = 8。求 S△ABC。

**解题步骤：**

第一步：判断相似

因为 DE ∥ BC，所以 △ADE ∽ △ABC。

第二步：求相似比

AD:AB = AD:(AD+DB) = 2:(2+3) = 2:5

第三步：求面积比

S△ADE : S△ABC = 2²:5² = 4:25

第四步：求 S△ABC

8 : S△ABC = 4 : 25

S△ABC = 8 × 25 ÷ 4 = 50

答：S△ABC = 50。

### 变式练习1：基础面积比

如图，AB ∥ CD，AB = 10，CD = 7。求 △AOB 和 △COD 的面积比。

**解答：**

相似比 = AB:CD = 10:7

面积比 = 10²:7² = 100:49

答：△AOB 和 △COD 的面积比是 100:49。

---

### 变式练习2：梯形中的面积

梯形 ABCD 中，AD ∥ BC，AD = 3，BC = 7，S△AOD = 9。求 S△BOC。

**解答：**

相似比 = AD:BC = 3:7

面积比 = 3²:7² = 9:49

9 : S△BOC = 9 : 49

S△BOC = 49

答：S△BOC = 49。

---

### 变式练习3：逆用比例求边长

如图，AB ∥ CD，S△AOB = 64，S△COD = 36。求 AB:CD。

**解答：**

面积比 = 64:36 = 16:9

相似比 = √16:√9 = 4:3

AB:CD = 4:3

答：AB:CD = 4:3。

---

### 变式练习4：综合应用

在 △ABC 中，D 在 AB 上，E 在 AC 上，DE ∥ BC。S△ADE : S四边形DBCE = 4:21。求 AD:DB。

**解答：**

S△ADE : S△ABC = 4:(4+21) = 4:25

相似比 = √4:√25 = 2:5

AD:AB = 2:5

AD:DB = 2:(5-2) = 2:3

答：AD:DB = 2:3。
:::

### 常见错误分析

**错误1：面积比直接等于边长比**

题目：相似三角形边长比是 2:3，求面积比。

错误做法：面积比 = 2:3

正确做法：面积比 = 相似比的平方 = 2²:3² = 4:9

原因：面积是二维量，与长度是平方关系。

---

**错误2：相似三角形的对应关系搞错**

题目：梯形 ABCD 中，AD ∥ BC，对角线交于 O。写出相似三角形。

错误做法：△AOD ∽ △BOC

正确做法：△AOD ∽ △COB（A对应C，O对应O，D对应B）

原因：AD ∥ BC，所以 A 与 C 对应，D 与 B 对应。虽然 △BOC 和 △COB 是同一个三角形，但在复杂问题中正确识别对应关系非常重要。

---

**错误3：没有平行线也套用沙漏模型**

题目：看到两条交叉线形成的两个三角形，就认为是沙漏模型。

错误做法：直接套用相似比结论。

正确做法：必须有"平行线"这个条件，才能形成沙漏模型（相似三角形）。如果没有平行线，两个三角形不一定相似。

---

**错误4：混淆沙漏模型和蝴蝶模型的结论**

题目：在梯形中，混淆"相似三角形"和"面积相等的三角形"。

错误做法：认为沙漏模型中 S△AOB = S△COD。

正确做法：S△AOB = S△COD 是蝴蝶模型的结论（因为 △ABC 和 △DBC 同底等高，减去公共部分 △BOC 后剩余面积相等）。沙漏模型的核心是相似三角形的比例关系。

---

**错误5：忘记考虑线段的方向**

题目：AD:DB = 2:3，求 AD:AB。

错误做法：AD:AB = 2:3

正确做法：AD:AB = AD:(AD+DB) = 2:(2+3) = 2:5

原因：AB 是整条线段，不是只有 DB 部分。
:::

### 亲子共学脚本

**场景一：认识相似三角形——家里的"缩放"游戏**

爸爸："小明，你看我手里的这张照片，如果我把它放大两倍，形状会变吗？"

小明："形状不变，只是变大了！"

爸爸："对啦！这就是'相似'的意思——形状相同，大小不同。你能举出生活中相似的例子吗？"

小明："嗯……同一个人的大小两张照片？地图上的路和实际的路？"

爸爸："真棒！三角形也可以相似。当两个三角形形状相同、大小不同时，它们就是相似三角形。"

**互动游戏：**

1. 拿两张不同大小的照片（同一个人），让孩子观察它们形状相同但大小不同。
2. 用放大镜观察文字，文字被放大了但形状不变。
3. 用积木拼一个大三角形和一个小三角形，让孩子比较它们是否相似。

---

**场景二：理解"相似比的平方"——折纸算面积**

爸爸："如果一个正方形边长是另一个的 2 倍，面积是几倍呢？"

小明："让我想想……边长是 2 倍，面积应该也是 2 倍？"

爸爸："我们来算一算。小正方形边长是 1，面积是 1×1=1。大正方形边长是 2，面积是 2×2=4。大正方形面积是小正方形的 4 倍！"

小明："哦！原来是 4 倍，也就是 2² 倍！"

爸爸："对啦！长度是 2 倍，面积就是 2²=4 倍。如果长度是 3 倍，面积就是多少倍呢？"

小明："3²=9 倍！"

爸爸："太棒了！这个规律对三角形也适用。两个相似三角形，如果边长比是 k:1，面积比就是 k²:1！"

**动手实践：**

1. 在方格纸上画一个小三角形（比如底为2、高为3），计算它的面积。
2. 再画一个形状相同但各边放大2倍的三角形（底为4、高为6），计算它的面积。
3. 比较两个三角形的面积比，验证是否等于相似比的平方。

---

**场景三：在梯形中找沙漏——画图找相似**

爸爸："小明，我们来画一个梯形。上底 AD = 2，下底 BC = 4，高是 3。"

爸爸："画出两条对角线，它们相交于点 O。"

爸爸："你能找到相似三角形吗？"

小明："△AOD 和 △COB 看起来相似！"

爸爸："对！它们的相似比是多少？"

小明："AD:BC = 2:4 = 1:2！"

爸爸："很好！那么它们的面积比呢？"

小明："1²:2² = 1:4！"

爸爸："如果知道 △AOD 的面积是 5，你能算出 △COB 的面积吗？"

小明："5 × 4 = 20！"

爸爸："完全正确！你已经掌握沙漏模型啦！"

---

**给家长的建议：**

1. **从生活实例入手**：照片、地图、模型都是相似的好例子。孩子理解"形状相同、大小不同"是第一步。
2. **动手验证"平方"规律**：孩子很容易忘记"面积比是相似比的平方"。通过在方格纸上计算具体例子，孩子能深刻理解为什么是平方关系。
3. **先简单后复杂**：先练习只有一个沙漏的简单问题，再逐步过渡到梯形、组合图形中的沙漏。
4. **强调"平行线"条件**：沙漏模型的前提是有平行线。看到交叉线就套用沙漏是常见错误。
5. **画图辅助理解**：遇到复杂问题时，鼓励孩子把图形画出来，标出已知条件和比例关系。

---

**生活拓展思考：**

- 问题1：一张照片长12厘米、宽8厘米。如果把它按比例放大，长变成30厘米，宽变成多少厘米？面积变成原来的几倍？
- 问题2：一棵大树的影子长6米，旁边一根1米高的竹竿影子长2米。这棵大树有多高？（提示：太阳光线是平行的，形成了相似三角形！）
- 问题3：地图的比例尺是1:10000，这是什么意思？地图上1平方厘米的面积，实际代表多少平方米？
