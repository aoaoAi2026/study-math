# 不定方程入门

## 知识点卡片

:::story-card
### 生活小故事

小明去商店买文具，铅笔每支2元，笔记本每本5元。他花了23元。

"我买了几支铅笔和几个笔记本呢？"小明想。

设买了x支铅笔，y个笔记本。

"2x + 5y = 23"

"这是一个不定方程，有两个未知数，但只有一个方程。"

"x = (23 - 5y) / 2"

"y = 1时，x = 9"
"y = 3时，x = 4"

"y = 2时，x = 6.5，不是整数，不行。"

"y = 0时，x = 11.5，也不行。"

"所以有两种可能：9支铅笔1个笔记本，或者4支铅笔3个笔记本！"

老师说："不定方程的解可能不止一个，要根据实际情况选择合理的解。"
:::

### 核心概念

**什么是不定方程**

未知数的个数多于方程个数的方程，叫做不定方程。

**二元一次不定方程**

形如 ax + by = c 的方程（a, b, c 为整数）。

**有整数解的条件**

方程 ax + by = c 有整数解的充要条件是：gcd(a, b) | c
（a和b的最大公因数能整除c）

**求解方法**

1. **枚举法**：对一个未知数枚举，求另一个
2. **同余法**：利用同余缩小范围
3. **特解+通解**：找到一个特解，写出通解公式

**通解公式**

若 (x₀, y₀) 是 ax + by = c 的一个特解，则通解为：
- x = x₀ + (b/d)×t
- y = y₀ - (a/d)×t
其中 d = gcd(a, b)，t 为任意整数

**实际应用中的限制**

- 未知数通常为正整数或非负整数
- 需要根据实际意义筛选解

### 典型例题

**例1：基础不定方程**
求 3x + 4y = 23 的所有正整数解。

**解答：**
- y = 1：3x = 19，x不是整数
- y = 2：3x = 15，x = 5 ✓
- y = 3：3x = 11，x不是整数
- y = 4：3x = 7，x不是整数
- y = 5：3x = 3，x = 1 ✓
- y ≥ 6：4y ≥ 24 > 23
- 答：正整数解为 (x, y) = (5, 2) 或 (1, 5)。

**例2：同余法**
求 5x + 7y = 100 的所有正整数解。

**解答：**
- 5x = 100 - 7y
- 100 - 7y ≡ 0 (mod 5)
- 7y ≡ 100 ≡ 0 (mod 5)
- 2y ≡ 0 (mod 5)
- y ≡ 0 (mod 5)（因为2和5互质）
- y = 5, 10, ...
- y = 5：5x = 65，x = 13 ✓
- y = 10：5x = 30，x = 6 ✓
- y = 15：5x = -5，x为负数，不行
- 答：正整数解为 (13, 5) 和 (6, 10)。

**例3：百鸡问题**
鸡翁一，值钱五；鸡母一，值钱三；鸡雏三，值钱一。百钱买百鸡，问鸡翁、母、雏各几何？

**解答：**
- 设鸡翁x只，鸡母y只，鸡雏z只
- x + y + z = 100
- 5x + 3y + z/3 = 100
- 由第二式：15x + 9y + z = 300
- 减去第一式：14x + 8y = 200
- 化简：7x + 4y = 100
- y = (100 - 7x) / 4
- x = 4：y = 18，z = 78 ✓
- x = 8：y = 11，z = 81 ✓
- x = 12：y = 4，z = 84 ✓
- 答：有三组解：(4, 18, 78)、(8, 11, 81)、(12, 4, 84)。

**例4：通解公式**
求 6x + 10y = 32 的所有整数解。

**解答：**
- gcd(6, 10) = 2，2|32，有解
- 化简：3x + 5y = 16
- 观察得特解：x = 2, y = 2（因为6+10=16）
- 通解：x = 2 + 5t，y = 2 - 3t
- 若要求正整数：
- 2 + 5t > 0 → t ≥ 0
- 2 - 3t > 0 → t ≤ 0
- 所以 t = 0
- 唯一正整数解：(2, 2)
- 答：整数解为 x = 2 + 5t, y = 2 - 3t（t为整数）；正整数解为 (2, 2)。

**例5：实际应用**
有30人租船，大船每艘坐6人，小船每艘坐4人。租了若干艘船，刚好坐满。有几种租法？

**解答：**
- 设大船x艘，小船y艘
- 6x + 4y = 30
- 化简：3x + 2y = 15
- y = (15 - 3x) / 2
- x = 1：y = 6 ✓
- x = 3：y = 3 ✓
- x = 5：y = 0（如果允许不租小船）
- x = 0：y = 7.5，不行
- 答：有3种租法：(1,6)、(3,3)、(5,0)。

**变式2：同余法**
求 4x + 9y = 100 的所有正整数解。
<details>
<summary>点击查看解答</summary>

**解答：**
- 4x = 100 - 9y
- 100 - 9y ≡ 0 (mod 4)
- 9y ≡ 100 ≡ 0 (mod 4)
- y ≡ 0 (mod 4)（因为9≡1 mod 4）
- y = 4：4x = 64，x = 16 ✓
- y = 8：4x = 28，x = 7 ✓
- y = 12：4x = -8，不行
- 答：正整数解为 (16,4) 和 (7,8)。
</details>

**变式3：实际应用**
有100元买两种水果，苹果每个5元，梨每个3元。两种都买，有几种买法？
<details>
<summary>点击查看解答</summary>

**解答：**
- 5x + 3y = 100
- 5x = 100 - 3y
- 100 - 3y ≡ 0 (mod 5)
- 3y ≡ 0 (mod 5)
- y ≡ 0 (mod 5)（因为3和5互质）
- y = 5, 10, 15, ...
- y = 5：x = 17 ✓
- y = 10：x = 14 ✓
- y = 15：x = 11 ✓
- y = 20：x = 8 ✓
- y = 25：x = 5 ✓
- y = 30：x = 2 ✓
- y = 35：x = -1，不行
- 答：有6种买法。
</details>

**变式4：综合应用**
一个两位数，十位数字比个位数字大3，且这个两位数等于它的数字之和的4倍。求这个两位数。
<details>
<summary>点击查看解答</summary>

**解答：**
- 设个位为x，十位为y
- y = x + 3
- 10y + x = 4(x + y)
- 10(x+3) + x = 4(x + x + 3)
- 11x + 30 = 8x + 12
- 3x = -18
- x = -6，不行
- 重新检查：10y + x = 4(x+y)
- 10y + x = 4x + 4y
- 6y = 3x
- x = 2y
- 又 y = x + 3
- x = 2(x+3) = 2x + 6
- x = -6，还是不行
- 可能是题目条件有误，或理解为"十位比个位小3"
- 若 y = x - 3：
- x = 2y = 2(x-3) = 2x - 6
- x = 6，y = 3
- 两位数 = 36
- 验证：3+6=9，4×9=36 ✓
- 答：这个两位数是36（条件应为十位比个位小3）。
</details>

### 常见错误

**错误1：忽略正整数限制**
求出所有整数解，没有筛选正整数解。

**正确：** 实际问题通常要求正整数或非负整数解。

**错误2：同余计算错误**
同余化简时出错。

**正确：** 注意模运算的性质，特别是除法需要逆元。

**错误3：漏解**
枚举时步长不对，漏掉一些解。

**正确：** 枚举时步长为1，或者利用同余确定步长。

**错误4：判断有解条件错误**
没有检查gcd(a,b)是否整除c。

**正确：** 先判断是否有解，再求解。

### 亲子互动

**给家长的辅导建议：**

1. **理解不定方程**
   - 未知数多于方程
   - 解可能不唯一

2. **掌握枚举法**
   - 对一个变量枚举
   - 验证另一个是否为整数

3. **学习同余法**
   - 缩小枚举范围
   - 提高效率

4. **结合实际**
   - 根据题意筛选合理解
   - 验证答案
