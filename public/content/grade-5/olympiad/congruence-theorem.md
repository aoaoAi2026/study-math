# 同余定理

## 知识点卡片

:::story-card
### 生活小故事

小明在日历上发现了一件有趣的事：

"今天是星期三，7天后还是星期三，14天后也是星期三！"

"因为7天一个星期，所以每过7天，星期几就重复一次。"

老师说："这就是'同余'的概念。如果两个数除以同一个数余数相同，就说它们同余。"

"比如，3、10、17除以7都余3，所以它们模7同余。"

"写成 3 ≡ 10 ≡ 17 (mod 7)"

小明又问："那今天是3号星期三，下个月3号也是星期三吗？"

"不一定，因为一个月不一定有28天。但如果这个月有28天，那下个月3号也是星期三。"

小明发现：同余在生活中处处可见！
:::

### 核心概念

**什么是同余**

如果两个整数a和b除以正整数m的余数相同，就说a和b模m同余。

记作：a ≡ b (mod m)

**同余的性质**

1. **反身性**：a ≡ a (mod m)
2. **对称性**：若 a ≡ b (mod m)，则 b ≡ a (mod m)
3. **传递性**：若 a ≡ b (mod m)，b ≡ c (mod m)，则 a ≡ c (mod m)

**同余的运算性质**

若 a ≡ b (mod m)，c ≡ d (mod m)，则：

1. a + c ≡ b + d (mod m)
2. a - c ≡ b - d (mod m)
3. a × c ≡ b × d (mod m)
4. aⁿ ≡ bⁿ (mod m)

**重要定理**

1. **余数的可加性**：和的余数 = 余数的和（再取余）
2. **余数的可减性**：差的余数 = 余数的差（再取余）
3. **余数的可乘性**：积的余数 = 余数的积（再取余）

**常见应用**

- 求大数除以某数的余数
- 判断整除
- 周期问题
- 星期问题

### 典型例题

**例1：基础同余**
求 123 + 456 除以 7 的余数。

**解答：**
- 123 ÷ 7 = 17余4，所以 123 ≡ 4 (mod 7)
- 456 ÷ 7 = 65余1，所以 456 ≡ 1 (mod 7)
- 123 + 456 ≡ 4 + 1 ≡ 5 (mod 7)
- 验证：123 + 456 = 579，579 ÷ 7 = 82余5 ✓
- 答：余数是5。

**例2：大数求余**
求 2¹⁰⁰ 除以 7 的余数。

**解答：**
- 找规律：
- 2¹ ≡ 2 (mod 7)
- 2² ≡ 4 (mod 7)
- 2³ ≡ 8 ≡ 1 (mod 7)
- 周期为3
- 100 ÷ 3 = 33余1
- 2¹⁰⁰ ≡ 2¹ ≡ 2 (mod 7)
- 答：余数是2。

**例3：星期问题**
今天是星期三，100天后是星期几？

**解答：**
- 100 ÷ 7 = 14余2
- 100 ≡ 2 (mod 7)
- 星期三 + 2天 = 星期五
- 答：100天后是星期五。

**例4：复杂求余**
求 3²⁰²⁴ 除以 10 的余数。

**解答：**
- 找规律（看个位数）：
- 3¹ = 3
- 3² = 9
- 3³ = 27，个位7
- 3⁴ = 81，个位1
- 3⁵ = 243，个位3
- 周期为4
- 2024 ÷ 4 = 506余0
- 3²⁰²⁴ ≡ 3⁴ ≡ 1 (mod 10)
- 答：余数是1。

**例5：综合应用**
求 1 + 2 + 3 + ... + 100 除以 7 的余数。

**解答：**
- 和 = 100 × 101 / 2 = 5050
- 5050 ÷ 7 = 721余3
- 或者用同余：
- 每7个连续数的和：7k+1 + 7k+2 + ... + 7k+7 = 7的倍数 + 28
- 28 ≡ 0 (mod 7)
- 100 = 14 × 7 + 2
- 前98个数的和 ≡ 0 (mod 7)
- 还剩 99 + 100 = 199
- 199 ÷ 7 = 28余3
- 答：余数是3。
:::

**变式1：基础同余**
求 234 × 567 除以 5 的余数。

**变式2：周期求余**
求 5¹⁰⁰ 除以 6 的余数。
<details>
<summary>点击查看解答</summary>

**解答：**
- 5 ≡ -1 (mod 6)
- 5¹⁰⁰ ≡ (-1)¹⁰⁰ ≡ 1 (mod 6)
- 答：余数是1。
</details>

**变式3：星期问题**
某年1月1日是星期一，这一年的6月1日是星期几？（该年不是闰年）
<details>
<summary>点击查看解答</summary>

**解答：**
- 1月：31天，2月：28天，3月：31天，4月：30天，5月：31天
- 从1月1日到6月1日共：31+28+31+30+31 = 151天
- 151 ÷ 7 = 21余4
- 星期一 + 4天 = 星期五
- 答：6月1日是星期五。
</details>

**变式4：综合应用**
求 2¹ + 2² + 2³ + ... + 2¹⁰ 除以 7 的余数。
<details>
<summary>点击查看解答</summary>

**解答：**
- 和 = 2(2¹⁰ - 1)/(2-1) = 2(1024-1) = 2046
- 找2ⁿ mod 7的周期：2, 4, 1, 2, 4, 1, ... 周期为3
- 10 = 3×3 + 1
- 和 mod 7 = 3×(2+4+1) + 2 = 3×7 + 2 ≡ 2 (mod 7)
- 答：余数是2。
</details>

### 常见错误

**错误1：同余符号记错**
把 a ≡ b (mod m) 记成 a = b mod m。（不规范）

**正确：** 标准记法是 a ≡ b (mod m)。

**错误2：除法也直接用同余**
认为 a ≡ b (mod m) 则 a/c ≡ b/c (mod m)。（错误）

**正确：** 同余没有除法性质，除非c与m互质。

**错误3：周期找错**
找2ⁿ mod 10的周期，认为周期是2。（错误）

**正确：** 2ⁿ mod 10的周期是4（2,4,8,6）。

**错误4：余数为0时搞错**
算出来余0，写成余数等于除数。（错误）

**正确：** 余数为0，就是整除。

### 亲子互动

**给家长的辅导建议：**

1. **理解同余概念**
   - 除以同一个数余数相同
   - 用日历、钟表引入

2. **掌握运算性质**
   - 和、差、积的同余
   - 幂的同余

3. **找周期**
   - 大数求余先找周期
   - 用较小指数试算

4. **验证答案**
   - 直接计算验证
   - 确保周期正确

**亲子小游戏：**

**"星期推算"**
- 今天是星期几
- 推算100天后、1000天后
- 用同余计算

**"尾数游戏"**
- 大数乘积的尾数
- 用模10同余
- 快速计算

**生活中的问题：**
- 你的生日今年星期几？明年呢？
- 2²⁰²⁴的个位数是多少？
