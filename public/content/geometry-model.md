# 五大几何模型：蝴蝶模型 —— 六年级奥数

## 第一步：故事引入

::: story-card

### 蝴蝶翅膀对称的故事

春天到了，小芳在花园里观察蝴蝶。她发现蝴蝶的翅膀有一个奇妙的特点：

> **左翅膀和右翅膀虽然花纹不同，但面积几乎一样大！**

蝴蝶的身体把翅膀分成了左右两半，而翅膀上的花纹又形成了上下两部分。这四个部分的面积之间，竟然藏着巧妙的数学关系。

回到课堂上，数学老师画了一个三角形，在底边上取了一点，连接顶点，把三角形分成了两个小三角形。然后又在左边三角形内画了一条线，形成了四个小区域。

老师说："你们看，这四个区域的面积关系，就像蝴蝶的翅膀一样——**对角的两个区域面积之积相等**！"

同学们纷纷惊叹：几何图形里竟然也有"蝴蝶"！

这就是今天要学习的 **蝴蝶模型**。

## 第二步：核心概念

::: concept-card

### 蝴蝶模型原理

**基本图形：**

在三角形 ABC 中，D 是 BC 上一点，E 是 AD 上一点，连接 BE 和 CE，形成四个区域。

```
          A
         /|\
        / | \
       /  |  \
      / 4 | 3 \
     /____|____\
    B-----E-----C
    \  1  D  2 /
     \        /
      \      /
       \    /
        \  /
         \/
```

**蝴蝶模型结论：**

$$S_{\triangle ABE} \times S_{\triangle CDE} = S_{\triangle ADE} \times S_{\triangle BDE}$$

即：**对角两个三角形面积之积相等。**

**推导过程：**

设 S₁ = △BDE，S₂ = △CDE

因为 △BDE 和 △CDE **等高**（都从 E 向 BC 作高），所以：

$$\frac{S_1}{S_2} = \frac{BD}{DC}$$

同理，△ABE 和 △ACE 也等高（都从 A 向 BC 作高），所以：

$$\frac{S_{\triangle ABE}}{S_{\triangle ACE}} = \frac{BD}{DC}$$

又因为 △ACE = △ADE + △CDE = △ADE + S₂，且 △ABE = △ADE + S₁

所以：**S₁ × (△ADE + S₂) = S₂ × (△ADE + S₁)**

展开化简即得：**S₁ × △ADE = S₂ × △ADE 的等价形式**

最终：**S_{△ABE} × S₂ = S_{△ADE} × S₁**

**记忆口诀：** 对角乘积相等，蝴蝶展翅飞翔。

:::

## 第三步：例题精讲

::: example-card

### 例题：梯形 ABCD 中，AB ∥ CD，对角线 AC 和 BD 交于点 O。已知 △AOB 面积为 3，△COD 面积为 12，求梯形 ABCD 的总面积。

**分析：** 梯形中的蝴蝶模型。

```
    A -------- B
     \   3   /
      \  O  /
       \   /
    12  \ /
       D -------- C
```

**第一步：利用蝴蝶模型**

在梯形中，△AOB 和 △COD 是对角三角形，满足蝴蝶模型：

$$S_{\triangle AOB} \times S_{\triangle COD} = S_{\triangle AOD} \times S_{\triangle BOC}$$

$$3 \times 12 = S_{\triangle AOD} \times S_{\triangle BOC}$$

$$S_{\triangle AOD} \times S_{\triangle BOC} = 36$$

**第二步：利用等高关系**

△AOD 和 △COD 等高（都从 D 向 AC 作高）：

$$\frac{S_{\triangle AOD}}{S_{\triangle COD}} = \frac{AO}{OC}$$

△AOB 和 △COB 等高（都从 B 向 AC 作高）：

$$\frac{S_{\triangle AOB}}{S_{\triangle COB}} = \frac{AO}{OC}$$

所以：**S_{△AOD} / 12 = 3 / S_{△COB}**，即 S_{△AOD} × S_{△COB} = 36（与蝴蝶模型一致）

由 S_{△AOD} = S_{△COB}（在梯形蝴蝶模型中，两侧面积相等），得：

$$S_{\triangle AOD} = S_{\triangle BOC} = \sqrt{36} = 6$$

**第三步：求总面积**

$$S_{\text{梯形}} = 3 + 12 + 6 + 6 = 27$$

**答案：梯形 ABCD 的总面积为 27。**

:::

## 第四步：变式训练

::: variant-card

### 变式一：三角形中的蝴蝶模型

在 △ABC 中，D 是 BC 的中点，E 是 AD 上一点，连接 BE 和 CE。已知 △BDE 的面积为 4，△CDE 的面积为 6，求 △ABE 和 △ACE 的面积之比。

**分析：**

因为 D 是 BC 的中点，所以 BD = DC，即 BD/DC = 1。

由蝴蝶模型：S_{△ABE} × S_{△CDE} = S_{△ADE} × S_{△BDE}

又因为 BD = DC，所以 △ABD 和 △ACD 面积相等（等底等高）。

设 S_{△ADE} = x，则 S_{△ABE} = S_{△ABD} - x，S_{△ACE} = S_{△ACD} - x

由于 S_{△ABD} = S_{△ACD}，所以 S_{△ABE} = S_{△ACE}

**答案：△ABE 和 △ACE 的面积之比为 1:1。**

---

### 变式二：梯形中的面积计算

梯形 ABCD 中，AB ∥ CD，对角线交于 O。已知 S_{△AOB} = 4，S_{△AOD} = 6，求 S_{△BOC} 和梯形总面积。

**分析：**

由蝴蝶模型：S_{△AOB} × S_{△COD} = S_{△AOD} × S_{△BOC}

即 4 × S_{△COD} = 6 × S_{△BOC}

又由等高关系：S_{△AOD}/S_{△COD} = S_{△AOB}/S_{△BOC}（梯形蝴蝶模型性质）

即 6/S_{△COD} = 4/S_{△BOC}

解得：S_{△BOC} = 6，S_{△COD} = 9

**梯形总面积 = 4 + 6 + 6 + 9 = 25**

:::

## 第五步：避坑指南

::: mistake-card

### 常见错误

**错误一：混淆蝴蝶模型的适用图形**

蝴蝶模型适用于 **三角形** 和 **梯形** 中对角线相交的情况，不能随意套用到其他四边形。

```
错误：在任意四边形中直接套用蝴蝶模型。  ✗
正确：蝴蝶模型要求图形具有平行线（梯形）或共顶点（三角形）。  ✓
```

**错误二：忘记梯形蝴蝶模型的特殊性质**

在梯形蝴蝶模型中，两侧三角形面积 **相等**（S_{△AOD} = S_{△BOC}），这是梯形的特殊性质，普通三角形蝴蝶模型不一定满足。

```
错误：认为所有蝴蝶模型中两侧面积都相等。  ✗
正确：只有梯形蝴蝶模型中 S_{△AOD} = S_{△BOC}。  ✓
```

**错误三：面积比与线段比搞混**

等高三角形的面积比等于底边比，等底三角形的面积比等于高之比，不要混淆。

```
错误：面积比 = 高之比（底不同时）。  ✗
正确：等高时面积比 = 底之比；等底时面积比 = 高之比。  ✓
```

**错误四：计算时遗漏某个区域**

蝴蝶模型涉及四个区域，求总面积时不要漏掉任何一个。

```
错误：总面积 = S_{△AOB} + S_{△COD}  ✗
正确：总面积 = S_{△AOB} + S_{△COD} + S_{△AOD} + S_{△BOC}  ✓
```

:::
