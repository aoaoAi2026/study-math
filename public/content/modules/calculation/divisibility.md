# 跨年级模块 - 计算模块

## 数论基础

::: concept-card
### 整除的特征

**能被2整除**：个位是0、2、4、6、8

**能被3整除**：各位数字之和能被3整除

**能被4整除**：末两位数能被4整除

**能被5整除**：个位是0或5

**能被9整除**：各位数字之和能被9整除
:::

::: example-card
### 判断整除
**题目**：判断 372 能否被 2、3、4、5、9 整除

**2**：看个位，2是偶数 ✓
**3**：3+7+2=12，12÷3=4 ✓
**4**：看末两位72，72÷4=18 ✓
**5**：看个位，个位是2 ✗
**9**：3+7+2=12，12÷9=1余3 ✗

**答案**：能被2、3、4整除，不能被5、9整除
:::
