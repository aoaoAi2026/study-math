# 一半模型

## 知识点卡片

:::story-card
### 生活小故事

小明在数学兴趣课上遇到了一个有趣的图形题：一个长方形里有一个点，连接这个点和四个顶点，把长方形分成了四个三角形。

"这四个三角形的面积有什么关系呢？"老师问。

小明发现：相对的两个三角形面积之和竟然相等，而且都等于长方形面积的一半！

"太神奇了！"小明惊叹道。

老师笑着说："这就是'一半模型'。在很多图形中，阴影部分的面积恰好是整体面积的一半。掌握这个规律，很多难题就能迎刃而解！"

小明开始寻找各种"一半模型"，发现它们在三角形、长方形、梯形、平行四边形中都有出现。
:::

:::concept-card
### 核心概念

**什么是一半模型**

在特定条件下，某部分的面积恰好等于整体面积的一半，这样的模型叫做一半模型。

**常见的一半模型**

1. **三角形中的一半模型**
   - 等底等高的三角形面积是平行四边形面积的一半
   - 中线把三角形分成面积相等的两部分

2. **长方形（平行四边形）中的一半模型**
   - 长方形内任意一点与四个顶点连线，相对两个三角形面积之和 = 长方形面积的一半
   - 长方形一边上的中点与对边两个顶点连线，三角形面积 = 长方形面积的一半

3. **梯形中的一半模型**
   - 梯形两腰中点连线形成的区域
   - 特定条件下的面积关系

**核心结论**
- 等底等高的三角形面积相等
- 中线分三角形为面积相等的两部分
- 长方形内一点：S1 + S3 = S2 + S4 = 长方形面积的一半
:::

### 典型例题

**例1：长方形中的一半模型**
长方形ABCD内有一点P，连接PA、PB、PC、PD。已知长方形面积是60平方厘米，求△PAB + △PCD的面积。

**解答：**
- 根据一半模型：△PAB + △PCD = 长方形面积的一半
- = 60 ÷ 2 = 30（平方厘米）
- 同理：△PAD + △PBC = 30（平方厘米）
- 答：△PAB + △PCD的面积是30平方厘米。

**例2：三角形中线**
三角形ABC中，AD是中线，三角形面积是48平方厘米。求△ABD的面积。

**解答：**
- 中线把三角形分成面积相等的两部分
- △ABD的面积 = 48 ÷ 2 = 24（平方厘米）
- 答：△ABD的面积是24平方厘米。

**例3：平行四边形中的一半**
平行四边形ABCD中，E是AB边上任意一点。求△CDE的面积与平行四边形面积的关系。

**解答：**
- △CDE以CD为底，高与平行四边形相同
- △CDE的面积 = CD × 高 ÷ 2 = 平行四边形面积 ÷ 2
- 答：△CDE的面积是平行四边形面积的一半。

**例4：综合应用**
长方形ABCD中，E、F分别是AB、CD的中点。连接DE、BF。求阴影部分面积占长方形面积的几分之几。

**解答：**
- E、F是中点，所以AEFD和EBCF都是长方形
- △ADE的面积 = 长方形AEFD面积的一半
- △BCF的面积 = 长方形EBCF面积的一半
- 两个三角形面积之和 = 大长方形面积的一半
- 答：阴影部分占长方形面积的一半。
:::

### 变式训练

**变式1：长方形内一点**
长方形ABCD面积为80，P为长方形内任意一点。求△PAB与△PCD的面积之和。

**解答：**

- △PAB + △PCD = 长方形面积的一半 = 40
- 答：面积之和是40。
</details>

**变式2：梯形中的一半**
梯形ABCD中，AD∥BC，E、F分别是AB、CD的中点。求证：四边形AEFD的面积是梯形面积的一半。
<details>
<summary>点击查看解答</summary>

**解答：**
- EF是梯形的中位线
- 梯形面积 = (上底+下底) × 高 ÷ 2
- 通过割补或等积变形可证
- 四边形AEFD的面积 = 梯形面积的一半
</details>

**变式3：多个一半模型**
长方形ABCD中，E、F、G、H分别是各边中点。求四边形EFGH的面积。
<details>
<summary>点击查看解答</summary>

**解答：**
- 连接对角线
- 四个角上的三角形面积各占小长方形的一半
- 四边形EFGH的面积 = 长方形面积的一半
</details>

**变式4：实际应用**
一个长方形花坛长20米，宽15米。里面有一个三角形区域种玫瑰，三角形的底是长方形的长，高是长方形的宽。种玫瑰的面积是多少？
<details>
<summary>点击查看解答</summary>

**解答：**
- 长方形面积 = 20 × 15 = 300（平方米）
- 三角形面积 = 长方形面积的一半 = 150（平方米）
- 答：种玫瑰的面积是150平方米。
</details>

### 常见错误

**错误1：条件不满足就用一半模型**
任意三角形内一点与顶点连线，认为相对三角形面积之和是一半。

**正确：** 只有在长方形（平行四边形）中，内部任意一点与四个顶点连线才有这个性质。

**错误2：高找错**
计算三角形面积时，底和高不对应。

**正确：** 高必须是对应底边上的垂线段。

**错误3：中线性质记错**
认为角平分线也把三角形分成面积相等的两部分。

**正确：** 只有中线（连接顶点和对边中点）才把三角形分成面积相等的两部分。

**错误4：忽略前提条件**
用一半模型时，没有验证是否满足条件。

**正确：** 使用一半模型前，先确认图形和条件是否符合。
