# 四年级数学 - 奥数专题

## 盈亏问题

::: story-card
### 分配糖果
老师把一些糖果分给小朋友们。如果每人分5颗，还剩12颗；如果每人分7颗，还差4颗。
老师一共有多少糖果？有多少小朋友？
:::

::: concept-card
### 什么是盈亏问题？
盈亏问题研究的是"分配"问题：当按照某种方式分配时，可能会出现"多"（盈）或"少"（亏）的情况。

**核心规律**：多给每人1份，盈亏情况会变化

::: parent-notes
### 给家长的话
盈亏问题的关键是理解"盈"和"亏"的含义。
可以用糖果或硬币实物演示，让孩子直观感受分配过程。
:::
:::

::: example-card
### 母题：盈盈问题
**题目**：老师给小朋友发糖。如果每人分6颗，则多12颗；如果每人分8颗，则多6颗。有多少小朋友？

**第一步**：比较两种分法
- 方案A：每人6颗，多12颗
- 方案B：每人8颗，多6颗

**第二步**：分析变化
方案B比方案A每人多 8-6=2 颗

**第三步**：计算人数
两次分配相差的糖果：12-6=6 颗
人数 = 6 ÷ 2 = 3 人

**第四步**：求糖果总数
3 × 6 + 12 = 30 颗

**核心公式**：$人数 = \frac{盈_{1} - 盈_{2}}{每人分配差}$
:::

::: variant-card
### 变式一：盈亏问题
老师发糖，每人5颗多12颗，每人7颗少4颗。有多少人？
:::

::: variant-card
### 变式二：双亏问题
每人3本少2本，每人4本少6本。有多少人？有多少本？
:::

::: mistake-card
### 常见错误：混淆盈亏
**错误**：把"多"和"少"搞混，导致计算符号错误
**纠正**：明确记录每次分配的盈亏数量
**错误层级**：L2
:::

::: parent-child-card
### 亲子共学脚本
**家长**："这里有12颗糖要分。如果每人分5颗，会怎样？"
**孩子**："会多出来一些"
**家长**："对，多12颗。那如果每人分7颗呢？"
**孩子**："好像不够"
**家长**："少了4颗。那第二种情况比第一种，每人多了几颗？"
**孩子**："多了2颗"
**家长**："糖果总数差了多少呢？"（等待思考）
:::
