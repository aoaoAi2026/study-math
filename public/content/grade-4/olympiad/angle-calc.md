### 生活小故事

小明家新买了一把折叠椅。他好奇地观察椅子折叠时的角度变化。

"妈妈，这个椅子展开时是180度，折叠起来后，两个部分形成了一个角。"小明说。

"对，而且这两个角加起来还是180度。"妈妈回答。

第二天上数学课，老师出了一道题：两条直线相交，已知一个角是35度，求其他三个角。

小明立刻想到了折叠椅："对顶角相等，所以相对的角也是35度。邻补角加起来是180度，所以另外两个角都是145度！"

老师赞许地说："很好！角的计算要利用对顶角相等、邻补角互补这些基本关系。"

小明发现：生活中的很多物品都蕴含着角的奥秘！

### 核心概念

**角的分类**
- 锐角：大于0°小于90°
- 直角：等于90°
- 钝角：大于90°小于180°
- 平角：等于180°
- 周角：等于360°

**重要的角的关系**

1. **对顶角相等**
   - 两条直线相交，对顶角相等

2. **邻补角互补**
   - 两个角有一条公共边，另一边互为反向延长线
   - 邻补角之和 = 180°

3. **互余**
   - 两个角之和 = 90°

4. **互补**
   - 两个角之和 = 180°

**三角形内角和**
- 三角形三个内角之和 = 180°

**多边形内角和**
- n边形内角和 = (n-2) × 180°

**外角性质**
- 三角形的一个外角等于与它不相邻的两个内角之和

### 典型例题

**例1：对顶角和邻补角**
两条直线相交，已知∠1 = 40°，求其他三个角。

**解答：**
- ∠1和∠3是对顶角，所以∠3 = 40°
- ∠1和∠2是邻补角，所以∠2 = 180° - 40° = 140°
- ∠2和∠4是对顶角，所以∠4 = 140°
- 答：∠2 = 140°，∠3 = 40°，∠4 = 140°。

**例2：三角形内角和**
一个三角形中，∠A = 50°，∠B = 60°，求∠C。

**解答：**
- ∠C = 180° - 50° - 60° = 70°
- 答：∠C = 70°。

**例3：等腰三角形**
一个等腰三角形，顶角是80°，求底角。

**解答：**
- 两个底角之和 = 180° - 80° = 100°
- 每个底角 = 100° ÷ 2 = 50°
- 答：底角是50°。

**例4：外角性质**
三角形ABC中，∠A = 40°，∠B = 70°。求∠C的外角。

**解答：**
- ∠C = 180° - 40° - 70° = 70°
- ∠C的外角 = 180° - 70° = 110°
- 或者：∠C的外角 = ∠A + ∠B = 40° + 70° = 110°
- 答：∠C的外角是110°。

**例5：多边形内角和**
求五边形的内角和。

**解答：**
- 五边形内角和 = (5-2) × 180° = 3 × 180° = 540°
- 答：五边形的内角和是540°。

### 变式训练

**变式1：求角度**
一个直角三角形，一个锐角是35°，另一个锐角是多少度？
<details>
<summary>点击查看解答</summary>

**解答：**
- 直角三角形两锐角互余
- 另一个锐角 = 90° - 35° = 55°
- 答：另一个锐角是55°。
</details>

**变式2：折叠问题**
把一张长方形纸对折，折痕与一边成30°角。求折痕与邻边的夹角。
<details>
<summary>点击查看解答</summary>

**解答：**
- 长方形的角是90°
- 折痕与一边成30°，与邻边成90° - 30° = 60°
- 答：折痕与邻边的夹角是60°。
</details>

**变式3：三角形外角**
三角形的一个外角是120°，与它不相邻的一个内角是50°。求另一个不相邻的内角。
<details>
<summary>点击查看解答</summary>

**解答：**
- 外角 = 不相邻两内角之和
- 另一个内角 = 120° - 50° = 70°
- 答：另一个不相邻的内角是70°。
</details>

**变式4：综合应用**
一个多边形的内角和是1080°，这是几边形？
<details>
<summary>点击查看解答</summary>

**解答：**
- 内角和 = (n-2) × 180° = 1080°
- n - 2 = 1080° ÷ 180° = 6
- n = 8
- 答：这是八边形。
</details>

### 常见错误

**错误1：对顶角和邻补角混淆**
认为邻补角相等。

**正确：** 对顶角相等，邻补角互补（和为180°）。

**错误2：三角形内角和记错**
认为三角形内角和是360°。

**正确：** 三角形内角和是180°。

**错误3：外角性质记错**
认为外角等于相邻内角。

**正确：** 外角等于不相邻的两个内角之和。

**错误4：多边形内角和公式记错**
认为n边形内角和 = n × 180°。

**正确：** n边形内角和 = (n-2) × 180°。
