# 容斥原理

## 知识点卡片

:::story-card
### 生活小故事

四年级一班有40名同学。老师统计兴趣爱好：喜欢数学的有25人，喜欢语文的有20人。

"那喜欢数学和语文的一共有多少人呢？"小明问。

"25 + 20 = 45人..."小明算到一半愣住了，"不对啊，全班才40人！"

老师笑着说："因为有些同学既喜欢数学又喜欢语文，被重复计算了。"

"设既喜欢数学又喜欢语文的有x人，那么："

"25 + 20 - x = 40"

"45 - x = 40"

"x = 5"

"所以有5人两门都喜欢！"

老师赞许地说："这就是容斥原理——有重复部分时，要把重复的部分减去。"
:::

:::concept-card
### 核心概念

**容斥原理（两个集合）**

对于两个集合A和B：
- |A ∪ B| = |A| + |B| - |A ∩ B|

即：总数 = 属于A的 + 属于B的 - 既属于A又属于B的

**容斥原理（三个集合）**

对于三个集合A、B和C：
- |A ∪ B ∪ C| = |A| + |B| + |C| - |A∩B| - |B∩C| - |A∩C| + |A∩B∩C|

**常见题型**
1. 两量重叠问题
2. 三量重叠问题
3. 求只参加某一项的人数
4. 求都不参加的人数

**解题关键**
- 画韦恩图（文氏图）帮助理解
- 明确各部分含义
- 注意"至少"、"只"等关键词
:::

### 典型例题

**例1：两量容斥**
某班50人，参加足球队的有28人，参加篮球队的有25人，两项都参加的有10人。两项都没参加的有多少人？

**解答：**
- 至少参加一项的人数 = 28 + 25 - 10 = 43（人）
- 两项都没参加的人数 = 50 - 43 = 7（人）
- 答：两项都没参加的有7人。

**例2：求只参加一项的人数**
某班40人，喜欢唱歌的有22人，喜欢跳舞的有18人，两项都喜欢的有8人。只喜欢唱歌的有多少人？

**解答：**
- 只喜欢唱歌的 = 22 - 8 = 14（人）
- 只喜欢跳舞的 = 18 - 8 = 10（人）
- 答：只喜欢唱歌的有14人。

**例3：三量容斥**
某班60人，参加数学竞赛的有35人，参加语文竞赛的有30人，参加英语竞赛的有25人。参加数学和语文的有15人，参加语文和英语的有10人，参加数学和英语的有12人，三项都参加的有5人。至少参加一项竞赛的有多少人？

**解答：**
- 至少参加一项 = 35 + 30 + 25 - 15 - 10 - 12 + 5
- = 90 - 37 + 5
- = 58（人）
- 答：至少参加一项竞赛的有58人。

**例4：都不参加**
某班45人，订阅《数学报》的有28人，订阅《语文报》的有20人，两种都订阅的有12人。两种都没订阅的有多少人？

**解答：**
- 至少订阅一种的 = 28 + 20 - 12 = 36（人）
- 两种都没订阅的 = 45 - 36 = 9（人）
- 答：两种都没订阅的有9人。

### 变式训练

**变式1：基础两量**
一个班级有48人，会弹钢琴的有22人，会拉小提琴的有18人，两种都会的有8人。两种都不会的有多少人？
<details>
<summary>点击查看解答</summary>

**解答：**
- 至少会一种的 = 22 + 18 - 8 = 32（人）
- 两种都不会的 = 48 - 32 = 16（人）
- 答：两种都不会的有16人。
</details>

**变式2：求重叠部分**
某班50人，参加美术小组的有30人，参加音乐小组的有25人，每个人都至少参加一个小组。两个小组都参加的有多少人？
<details>
<summary>点击查看解答</summary>

**解答：**
- 都参加的 = 30 + 25 - 50 = 5（人）
- 答：两个小组都参加的有5人。
</details>

**变式3：三量容斥**
某校100名学生中，喜欢足球的有58人，喜欢篮球的有48人，喜欢排球的有52人。喜欢足球和篮球的有25人，喜欢篮球和排球的有20人，喜欢足球和排球的有22人，三种都喜欢的有10人。至少喜欢一种球类的有多少人？三种都不喜欢的有多少人？
<details>
<summary>点击查看解答</summary>

**解答：**
- 至少喜欢一种 = 58 + 48 + 52 - 25 - 20 - 22 + 10
- = 158 - 67 + 10
- = 101（人）
- 等等，101 > 100，说明数据有问题
- 按题目数据计算：101人
- 三种都不喜欢的 = 100 - 101 = -1（不合理）
- 可能是题目数据有误，或理解为101人中的100人
- 简化答：至少喜欢一种的有101人（数据可能有矛盾）
</details>

**变式4：只参加一项**
某班有学生参加三个兴趣小组：数学、语文、英语。参加数学的有20人，语文18人，英语16人。参加数学和语文的有8人，语文和英语的有6人，数学和英语的有7人，三项都参加的有3人。只参加一个小组的有多少人？
<details>
<summary>点击查看解答</summary>

**解答：**
- 只参加数学的 = 20 - 8 - 7 + 3 = 8（人）
- 只参加语文的 = 18 - 8 - 6 + 3 = 7（人）
- 只参加英语的 = 16 - 7 - 6 + 3 = 6（人）
- 只参加一个小组的 = 8 + 7 + 6 = 21（人）
- 答：只参加一个小组的有21人。
</details>
:::

:::mistake-card
### 常见错误

**错误1：重复部分没有减去**
28人喜欢足球，25人喜欢篮球，一共28+25=53人（错误）。

**正确：** 如果有重复，要减去重复部分。

**错误2：三量容斥符号搞错**
三量容斥时，最后三项都参加的该加还是该减搞错。

**正确：** 三量容斥公式：A+B+C-AB-BC-AC+ABC

**错误3："只参加"和"参加"混淆**
"参加数学的20人"包括只参加数学和同时参加其他的。

**正确：** 求"只参加数学的"要减去与其他重叠的部分。

**错误4：都不参加的忘记求**
只求了至少参加一项的，忘记用总数减。

**正确：** 都不参加的 = 总数 - 至少参加一项的

### 亲子互动

**给家长的辅导建议：**

1. **画韦恩图**
   - 用圆圈表示集合
   - 重叠部分表示交集
   - 直观理解容斥原理

2. **理解"至少"、"只"**
   - "至少参加一项" = 参加一项或多项
   - "只参加一项" = 仅参加一项

3. **从两量到三量**
   - 先掌握两量容斥
   - 再学习三量容斥

4. **生活中的例子**
   - 班级兴趣爱好统计
   - 家庭成员爱好统计

**亲子小游戏：**

**"兴趣调查"**
- 调查家庭成员的兴趣爱好
- 用容斥原理计算
- 如：喜欢音乐和喜欢画画的人

**"韦恩图绘画"**
- 用三个圆表示三个集合
- 填入具体数字
- 验证容斥原理

**生活中的问题：**
- 你们班有多少人既喜欢数学又喜欢语文？
- 家里有几种水果？哪些水果是大家都喜欢的？
