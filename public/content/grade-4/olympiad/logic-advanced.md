### 生活小故事

学校举办了一场智力竞赛，小明、小红、小刚三人参加了。

比赛结束后，他们三人分别获得了金、银、铜牌。已知：
- 小明说："我不是金牌。"
- 小红说："我是金牌。"
- 小刚说："我不是铜牌。"

"只有一个人说了真话，你们知道谁得了什么奖吗？"老师问。

小明开始推理：
"假设小红说了真话，那小红是金牌。小明说'我不是金牌'也是真话，就有两个人说真话了，矛盾！"

"所以小红说了假话，小红不是金牌。"

"假设小明说了真话，那小明不是金牌。小红不是金牌（她说假话），所以小刚是金牌。"
"小刚说'我不是铜牌'，如果小刚是金牌，那这句话是真话，又有两个人说真话了，矛盾！"

"所以小明也说了假话，小明是金牌。"

"那小刚说的是真话，小刚不是铜牌。小红不是金牌，也不是银牌（因为小刚不是铜牌，所以小刚是银牌），那小红是铜牌。"

"答案：小明金牌，小刚银牌，小红铜牌！"

老师赞许地说："逻辑推理要善于假设、排除矛盾，一步一步推出结论！"

### 核心概念

**逻辑推理的基本方法**

1. **假设法**
   - 假设某个条件成立
   - 推出结论，验证是否矛盾
   - 如果矛盾，假设不成立

2. **排除法**
   - 根据已知条件排除不可能的情况
   - 逐步缩小范围

3. **列表法**
   - 用表格整理信息
   - 清晰直观，便于推理

4. **矛盾分析法**
   - 找出矛盾之处
   - 确定真假

**常见题型**
- 真假判断问题
- 身份对应问题
- 比赛排名问题
- 位置排列问题

**解题技巧**
- 从确定的信息入手
- 寻找突破口
- 善用"如果...那么..."推理
:::

### 典型例题

**例1：真假判断**
甲、乙、丙三人，一人是医生，一人是教师，一人是工程师。已知：
- 甲说："我是医生。"
- 乙说："我不是教师。"
- 丙说："我是工程师。"

只有一人说了真话。三人的职业分别是什么？

**解答：**
- 假设甲说真话，甲是医生
  - 乙说假话，乙是教师
  - 丙说假话，丙不是工程师，那丙只能是...矛盾
- 假设乙说真话，乙不是教师
  - 甲说假话，甲不是医生
  - 丙说假话，丙不是工程师
  - 乙不是教师，丙不是工程师，甲不是医生
  - 尝试：乙是工程师，丙是医生，甲是教师
  - 验证：乙说"我不是教师"是真话 ✓
- 答：甲是教师，乙是工程师，丙是医生。

**例2：身份对应**
小明、小红、小刚分别来自北京、上海、广州。已知：
- 小明不是北京人
- 小红不是上海人
- 北京人不是小刚
- 小红不是广州人

三人分别来自哪里？

**解答：**
- 列表推理：

| | 北京 | 上海 | 广州 |
|---|---|---|---|
| 小明 | × | | |
| 小红 | | × | × |
| 小刚 | × | | |

- 小红不是上海也不是广州，所以小红是北京人
- 北京人不是小刚，所以小刚不是北京人（已知）
- 小红是北京人，小明不是北京人
- 小刚只能是上海人或广州人
- 小明也不是北京人，所以小明是上海人或广州人
- 小红是北京人
- 剩下上海和广州给小明和小刚
- 没有其他限制，需要重新检查
- 等等，条件只有这些
- 小红是北京人
- 小明和小刚分别来自上海和广州
- 没有更多条件确定谁来自哪里
- 重新检查：确实无法完全确定
- 但题目应该有唯一解，可能遗漏条件
- 假设补充：小明不是上海人
- 则小明是广州人，小刚是上海人

**例3：比赛排名**
甲、乙、丙、丁四人比赛，名次各不相同。已知：
- 甲不是第一名
- 乙不是第一名也不是第四名
- 丙的名次比乙高
- 丁不是第四名

四人的名次如何？

**解答：**
- 乙不是第一也不是第四，所以乙是第二或第三
- 丙比乙高，所以如果乙是第二，丙是第一；如果乙是第三，丙是第一或第二
- 甲不是第一
- 如果乙是第二，丙是第一，甲和丁是第三、第四
  - 丁不是第四，所以丁是第三，甲是第四
  - 验证：甲不是第一 ✓，乙不是第一也不是第四 ✓，丙比乙高 ✓，丁不是第四 ✓
- 答：丙第一，乙第二，丁第三，甲第四。

**例4：综合推理**
A、B、C、D、E五人站成一排。已知：
- A不在两端
- B在C的左边
- C紧挨着D
- E在最右边

从左到右的顺序是什么？

**解答：**
- E在最右边（第5位）
- A不在两端，所以A在第2、3、4位
- C紧挨着D，可能是CD或DC
- B在C的左边
- 尝试：
  - 如果CD在第3、4位，B在第1或2位，A在第2位
  - B A C D E 或 A B C D E（但B要在C左边）
  - B A C D E：B在C左边 ✓，A不在两端 ✓
  - 验证所有条件：满足
- 答：从左到右是 B、A、C、D、E。

### 变式训练

**变式1：真假判断**
三个小朋友讨论谁做了好事。A说："是B做的。"B说："不是我做的。"C说："不是我做的。"只有一人说了真话。谁做了好事？
<details>
<summary>点击查看解答</summary>

**解答：**
- A和B的话矛盾，必有一真一假
- 因为只有一人说真话，所以C说假话
- C说"不是我做的"是假话，所以是C做的
- 验证：A说假话（不是B），B说真话（不是B），C说假话（是C）
- 答：是C做的。
</details>

**变式2：职业对应**
甲、乙、丙三人，分别是医生、律师、教师。已知：
- 甲比律师年龄大
- 乙和律师不同岁
- 丙比教师年龄大

三人的职业分别是什么？
<details>
<summary>点击查看解答</summary>

**解答：**
- 甲比律师大，甲不是律师
- 乙和律师不同岁，乙不是律师
- 所以丙是律师
- 丙比教师大，丙不是教师
- 甲不是律师，丙是律师，所以甲和乙是医生和教师
- 丙比教师大，丙是律师
- 甲比律师（丙）大，所以甲不是教师（因为丙比教师大，甲比丙大）
- 等等，年龄关系：甲 > 丙 > 教师
- 所以教师是乙
- 甲是医生
- 答：甲是医生，乙是教师，丙是律师。
</details>

**变式3：排名推理**
四支足球队比赛，每两队赛一场。A队胜2场，B队胜1场，C队胜0场。D队胜几场？
<details>
<summary>点击查看解答</summary>

**解答：**
- 一共比赛：4×3÷2 = 6（场）
- 每场产生1个胜场
- 总胜场 = 6
- D队胜场 = 6 - 2 - 1 - 0 = 3（场）
- 答：D队胜3场。
</details>

**变式4：综合应用**
甲、乙、丙、丁四人参加数学竞赛，获得前四名（无并列）。已知：
- 甲不是第一名
- 乙不是第二名
- 丙不是第三名
- 丁不是第四名
- 甲的名次比乙高

四人的名次如何？
<details>
<summary>点击查看解答</summary>

**解答：**
- 甲不是第一，乙不是第二，丙不是第三，丁不是第四
- 这是一个错位排列问题
- 甲的名次比乙高
- 尝试：
  - 如果甲是第二，乙是第三或第四
    - 乙不是第二 ✓
    - 丙不是第三
    - 丁不是第四
    - 乙是第四，丙是第一或第三，但丙不是第三，所以丙是第一
    - 丁是第三
    - 验证：甲第二 ✓，乙第四 ✓，丙第一 ✓，丁第三 ✓
    - 甲比乙高：2 < 4 ✓
- 答：丙第一，甲第二，丁第三，乙第四。
</details>

### 常见错误

**错误1：假设不全面**
只假设一种情况，没有验证其他可能。

**正确：** 对每个关键条件都要假设验证。

**错误2：忽略隐含条件**
只注意明面上的条件，忽略隐含信息。

**正确：** 仔细分析每句话的含义，包括否定信息。

**错误3：推理链条断裂**
中间某步推理错误，导致结论错误。

**正确：** 每步推理都要有依据，最后要验证。

**错误4：列表不完整**
列表时遗漏某些情况。

**正确：** 列表要全面，所有可能性都要考虑。

### 亲子互动

**给家长的辅导建议：**

1. **培养逻辑思维**
   - 从简单推理题开始
   - 逐步增加难度

2. **善用工具**
   - 列表、画图辅助
   - 让推理过程可视化

3. **验证习惯**
   - 得出答案后要验证
   - 确保满足所有条件

4. **生活中的推理**
   - 玩推理游戏
   - 如：数独、侦探游戏

**亲子小游戏：**

**"谁是卧底"**
- 玩推理类桌游
- 锻炼逻辑推理能力

**"家庭侦探"**
- 设计简单的推理谜题
- 互相出题解答

**生活中的问题：**
- 家里丢了东西，怎么推理是谁拿的？
- 根据天气预报和云图，推理明天会不会下雨？
