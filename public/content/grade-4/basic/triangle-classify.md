# 三角形分类

## 知识点卡片

### 生活小故事

美术课上，老师让同学们用吸管拼三角形。

小明用三根一样长的吸管拼了一个三角形："我的三角形三条边都相等！"

"这叫等边三角形。"老师说。

小红用两根一样长、一根不同长的吸管拼了一个："我的只有两条边相等。"

"这叫等腰三角形。"老师回答。

小刚用三根都不一样的吸管拼了一个："我的三条边都不相等。"

"这叫不等边三角形，也叫 scalene triangle。"

接着老师又问："那按角来分呢？"

同学们用三角尺量了量，发现有的三角形三个角都是锐角，有的有一个直角，有的还有一个钝角。

"三角形按边分和按角分，是两种不同的分类方法哦！"老师总结道。

小明发现：三角形的世界真丰富！

### 核心概念

**三角形按角分类**

1. **锐角三角形**：三个角都是锐角（小于90°）
2. **直角三角形**：有一个角是直角（等于90°）
3. **钝角三角形**：有一个角是钝角（大于90°小于180°）

**三角形按边分类**

1. **不等边三角形**：三条边都不相等
2. **等腰三角形**：有两条边相等
   - 相等的两边叫做腰
   - 另一边叫做底
   - 两腰的夹角叫做顶角
   - 腰和底的夹角叫做底角
3. **等边三角形**：三条边都相等
   - 等边三角形是特殊的等腰三角形
   - 三个角都是60°

**三角形的内角和**
- 三角形三个内角的和等于180°

**三角形边的关系**
- 三角形任意两边之和大于第三边
- 三角形任意两边之差小于第三边

### 典型例题

**例1：按角分类**
一个三角形的三个角分别是50°、60°、70°，这是什么三角形？

**解答：**
- 三个角都是锐角
- 答：这是锐角三角形。

**例2：按边分类**
一个三角形的三条边分别是5cm、5cm、8cm，这是什么三角形？

**解答：**
- 有两条边相等（5cm = 5cm）
- 答：这是等腰三角形。

**例3：求角的度数**
一个等腰三角形，顶角是80°，底角是多少度？

**解答：**
- 三角形内角和 = 180°
- 两个底角之和 = 180° - 80° = 100°
- 每个底角 = 100° ÷ 2 = 50°
- 答：底角是50°。

**例4：判断能否组成三角形**
下面哪组线段能组成三角形？
- 3cm、4cm、5cm
- 2cm、2cm、5cm

**解答：**
- 第一组：3+4>5，3+5>4，4+5>3，能组成三角形
- 第二组：2+2=4<5，不能组成三角形
- 答：第一组能组成三角形。

**例5：综合判断**
一个三角形有两个角分别是40°和70°，按角分是什么三角形？按边分呢？

**解答：**
- 第三个角 = 180° - 40° - 70° = 70°
- 三个角都是锐角，是锐角三角形
- 有两个角相等（70° = 70°），对应的两条边也相等
- 所以也是等腰三角形
- 答：按角分是锐角三角形，按边分是等腰三角形。

### 变式训练

**变式1：角度计算**
一个直角三角形，一个锐角是35°，另一个锐角是多少度？
<details>
<summary>点击查看解答</summary>

**解答：**
- 直角三角形两锐角之和 = 90°
- 另一个锐角 = 90° - 35° = 55°
- 答：另一个锐角是55°。
</details>

**变式2：等边三角形**
一个等边三角形的周长是36厘米，边长是多少？
<details>
<summary>点击查看解答</summary>

**解答：**
- 等边三角形三条边相等
- 边长 = 36 ÷ 3 = 12（厘米）
- 答：边长是12厘米。
</details>

**变式3：三角形边的关系**
有两根小棒，分别长5厘米和8厘米。再找一根多长的小棒，就能和这两根围成三角形？
<details>
<summary>点击查看解答</summary>

**解答：**
- 第三边要满足：8-5 < 第三边 < 8+5
- 即：3 < 第三边 < 13
- 答：第三根小棒的长度要大于3厘米且小于13厘米。
</details>

**变式4：综合应用**
一个等腰三角形，一条边长6厘米，另一条边长13厘米。这个三角形的周长是多少？
<details>
<summary>点击查看解答</summary>

**解答：**
- 情况1：腰长6厘米，底13厘米
  - 6+6=12<13，不能组成三角形
- 情况2：腰长13厘米，底6厘米
  - 13+6>13，13+13>6，能组成三角形
  - 周长 = 13+13+6 = 32（厘米）
- 答：周长是32厘米。
</details>

### 常见错误

**错误1：分类标准混淆**
认为等腰三角形一定是锐角三角形。

**正确：** 等腰三角形按角分可以是锐角、直角或钝角三角形。

**错误2：内角和记错**
认为三角形内角和是360°。

**正确：** 三角形内角和是180°。

**错误3：判断三角形时只看一组边**
判断3cm、4cm、8cm能否组成三角形，只看3+4>8。

**正确：** 必须验证任意两边之和大于第三边。3+4=7<8，不能组成。

**错误4：等边三角形与等腰三角形关系不清**
认为等边三角形不是等腰三角形。

**正确：** 等边三角形是特殊的等腰三角形。

### 亲子互动

**给家长的辅导建议：**

1. **动手操作**
   - 用吸管、小棒拼各种三角形
   - 直观感受不同类型的三角形

2. **测量验证**
   - 用量角器量三角形各角
   - 验证内角和是180°

3. **分类对比**
   - 按角分和按边分对比
   - 画表格整理

4. **生活中的三角形**
   - 找生活中的三角形
   - 讨论为什么设计成三角形

**亲子小游戏：**

**"三角形分类"**
- 准备一些三角形纸片
- 让孩子按角分和按边分分类

**"拼三角形"**
- 用不同长度的小棒
- 尝试哪些能拼成三角形
- 验证三边关系

**生活中的问题：**
- 自行车架为什么是三角形的？
- 晾衣架利用了三角形的什么特性？
