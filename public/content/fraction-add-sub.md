# 分数加减法 [难度：★★]

## 📌 它是什么？
学习同分母和异分母分数的加法与减法运算，是五年级数学的核心内容。

## 🎯 为什么重要？
分数运算在日常生活中无处不在——做饭、测量、分享物品都需要它。

## 📋 需要什么基础？
- 理解分数的意义（分子、分母、分数单位）
- 会求几个数的最小公倍数
- 掌握约分的方法

### 分披萨与喝果汁
生日派对上，妈妈买了一个大披萨。小明吃了 $\frac{2}{8}$，小红吃了 $\frac{3}{8}$。两人一共吃了多少？还剩多少？

后来，小明喝了一杯果汁的 $\frac{1}{3}$，小红喝了同一杯果汁的 $\frac{1}{4}$。两人一共喝了多少杯果汁？分母不一样怎么办呢？

### 分数加减法规则

**同分母分数加减**：分母不变，分子相加减
$$\frac{a}{c} \pm \frac{b}{c} = \frac{a \pm b}{c}$$

**异分母分数加减**：先通分，再按同分母方法计算
$$\frac{a}{b} \pm \frac{c}{d} = \frac{ad}{bd} \pm \frac{bc}{bd} = \frac{ad \pm bc}{bd}$$

**通分步骤**：
1. 找出两个分母的最小公倍数（LCM）
2. 把每个分数化成以 LCM 为分母的等值分数
3. 分子相加减
4. 结果能约分的要约分

**注意**：计算结果必须化为最简分数！
:::

::: example-card
### 母题：同分母加法
**题目**：小明吃了披萨的 $\frac{2}{8}$，小红吃了 $\frac{3}{8}$，一共吃了多少？

**第一步**：分母相同（都是 8），直接分子相加
$$\frac{2}{8} + \frac{3}{8} = \frac{2 + 3}{8} = \frac{5}{8}$$

**第二步**：约分检查
$$\frac{5}{8}$$ 已经是最简分数，无需约分。

**答**：两人一共吃了披萨的 $\frac{5}{8}$。

**核心公式**：$\frac{a}{c} + \frac{b}{c} = \frac{a+b}{c}$
:::

::: variant-card
### 变式一：异分母加法
**题目**：小明喝了果汁的 $\frac{1}{3}$，小红喝了 $\frac{1}{4}$，一共喝了多少？

**第一步**：通分——找 3 和 4 的最小公倍数 = 12
$$\frac{1}{3} = \frac{4}{12}, \quad \frac{1}{4} = \frac{3}{12}$$

**第二步**：分子相加
$$\frac{4}{12} + \frac{3}{12} = \frac{7}{12}$$

**答**：两人一共喝了果汁的 $\frac{7}{12}$。
:::

::: variant-card
### 变式二：异分母减法
**题目**：一块布料，做衣服用去了 $\frac{3}{4}$，做裤子用去了 $\frac{1}{6}$，做衣服比做裤子多用去多少？

**第一步**：通分——找 4 和 6 的最小公倍数 = 12
$$\frac{3}{4} = \frac{9}{12}, \quad \frac{1}{6} = \frac{2}{12}$$

**第二步**：分子相减
$$\frac{9}{12} - \frac{2}{12} = \frac{7}{12}$$

**答**：做衣服比做裤子多用去布料的 $\frac{7}{12}$。
:::

::: mistake-card
### 常见错误

**错误 1**：异分母时分子直接加，分母直接加
$$\text{错：} \frac{1}{3} + \frac{1}{4} = \frac{2}{7}$$
**纠正**：必须先通分，再分子相加，分母不变。

**错误 2**：通分时分子没有同步扩大
$$\text{错：} \frac{1}{3} = \frac{1}{12} \quad (\text{只扩大了分母})$$
**纠正**：$\frac{1}{3} = \frac{1 \times 4}{3 \times 4} = \frac{4}{12}$，分子分母要同时乘相同的数。

**错误 3**：忘记约分
$$\text{错：最终结果写 } \frac{4}{8} \text{ 而不约分}$$
**纠正**：养成习惯，最后一步检查能否约分，$\frac{4}{8} = \frac{1}{2}$。
:::
