# 常见易错点

## 数学易错点汇总

### 计算易错点

1. **去括号时符号错误**
   - $-(a+b) = -a - b$（不是 $-a + b$）

2. **分数运算顺序**
   - 先乘除，后加减
   - 同一级从左到右

3. **单位换算**
   - 1千米 = 1000米
   - 1平方米 = 100平方分米
   - 进率是100不是10

### 概念易错点

1. **倍数问题找错1份量**
   - "A是B的几倍"中，B是1份量

2. **余数总比除数小**

3. **0不能作除数**

### 解题易错点

1. **审题不清，漏看条件**
2. **计算正确但写错答案**
3. **单位不统一就计算**
