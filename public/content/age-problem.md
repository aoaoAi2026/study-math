# 年龄问题 [难度：★★]

## 📌 它是什么？
已知两个人（或多人）现在的年龄和年龄关系，求过去或未来的年龄。

## 🎯 为什么重要？
年龄问题是生活中最常见的数学应用之一，掌握"年龄差不变"这一核心原理，就能轻松应对各种年龄推理题。

## 📋 需要什么基础？
- 加减乘除运算熟练
- 理解"差不变"的含义

::: story-card
### 父子年龄的故事
小明今年 8 岁，爸爸今年 34 岁。
小明好奇地问："爸爸，我几岁的时候，你的年龄刚好是我的 3 倍呢？"
爸爸笑着说："你想想看，我们俩的年龄差会变吗？"
:::

::: concept-card
### 核心原理：年龄差永远不变
```
今年：  小明 8 岁    爸爸 34 岁    差 = 34 - 8 = 26 岁
5年后： 小明 13 岁   爸爸 39 岁    差 = 39 - 13 = 26 岁
10年后：小明 18 岁   爸爸 44 岁    差 = 44 - 18 = 26 岁
```

**关键发现**：不管过多少年，两个人的年龄差永远不变！

**解题策略**：
1. 先算出年龄差
2. 用年龄差和倍数关系，求出当时的年龄
3. 再推算是"几年前"还是"几年后"

**注意**：年龄差不变，但年龄的"倍数"会随时间变化——越往后倍数越接近 1。
:::

::: example-card
### 母题：年龄与倍数
**题目**：小明今年 8 岁，爸爸今年 34 岁。几年前爸爸的年龄是小明的 5 倍？

**第一步**：算年龄差 → 34 - 8 = 26 岁（永远不变）

**第二步**：设那时小明 x 岁，则爸爸 5x 岁
年龄差：5x - x = 4x = 26，所以 x = 26 ÷ 4 = 6.5

**第三步**：小明那时 6.5 岁，说明是 8 - 6.5 = 1.5 年前

**验算**：1.5 年前，小明 6.5 岁，爸爸 32.5 岁，32.5 ÷ 6.5 = 5 ✓

**核心方法**：年龄差 ÷ (倍数 - 1) = 当时较小人的年龄
:::

::: variant-card
### 变式一：求未来年龄
妈妈今年 32 岁，女儿今年 6 岁。几年后妈妈的年龄是女儿的 3 倍？

**提示**：年龄差 = 32 - 6 = 26 岁。设 x 年后，女儿 (6+x) 岁，妈妈 3(6+x) 岁。
3(6+x) - (6+x) = 26 → 2(6+x) = 26 → 6+x = 13 → x = 7。

**答案**：7 年后。验算：女儿 13 岁，妈妈 39 岁，39 ÷ 13 = 3 ✓
:::

::: variant-card
### 变式二：兄弟年龄差
哥哥今年 12 岁，弟弟今年 5 岁。当兄弟俩年龄之和是 35 岁时，哥哥多少岁？

**提示**：年龄差 = 12 - 5 = 7 岁（不变）。设那时弟弟 x 岁，则哥哥 (x+7) 岁。
x + (x+7) = 35 → 2x = 28 → x = 14，哥哥 = 14 + 7 = 21 岁。

**答案**：哥哥 21 岁。验算：21 + 14 = 35 ✓，21 - 14 = 7 ✓
:::

::: mistake-card
### 常见错误：认为年龄差也会变
**错误**：小明 8 岁，爸爸 34 岁。5 年后爸爸比小明大 34 + 5 = 39 岁。
**原因**：把"年龄增长"和"年龄差"混淆了——两个人每年都长 1 岁，差不会变。
**层级**：L2 概念错误
**纠正**：画时间轴！每年两个人各加 1 岁，差永远不变。记住口诀——"同长同长，差不变"。
:::

### 亲子共学脚本
**家长**："你今年几岁？妈妈今年几岁？妈妈比你大多少岁？"
**孩子**：（回答年龄差）
**家长**："那你 10 岁的时候，妈妈比你大多少岁？"
**孩子**：（可能犹豫）
**家长**："我们一起来画个表，从今年算到 10 岁看看，差变不变？"
**家长**："对啦！差永远不变。那如果妈妈比你大 26 岁，什么时候妈妈的年龄是你的 3 倍呢？"
