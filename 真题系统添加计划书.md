---

## ⚠️ 重要声明

**本系统的数据来源策略：**
- ❌ 不生成"模拟题"、"改编题"冒充真题
- ❌ 不捏造考试名称、年份、来源信息
- ✅ 所有入库试卷必须有**真实的来源依据**
- ✅ 用户提供真实试卷 → 系统格式化录入
- ✅ 暂无真实来源的 → 提供**收集指南**供用户自行搜集

## 一、图形题存储方案（SVG 内嵌）

### 1.1 为什么用 SVG 内嵌

| 对比项 | 图片引用 | SVG 内嵌 | CSS 绘制 |
|--------|---------|---------|---------|
| 真实性 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐ |
| 容错率 | ⭐⭐（缺文件就404） | ⭐⭐⭐⭐⭐（永不丢失） | ⭐⭐⭐ |
| 依赖 | 需额外图片文件 | 零外部依赖 | 无 |
| 缩放质量 | 有损放大 | 无损放大 | 无 |
| 适用场景 | 复杂实景图 | **几何图形题** | 简单示意 |

**结论：** 数学题大量涉及几何图形，SVG 是最优方案。

### 1.2 支持的字段

```json
{
  "id": "q1",
  "type": "choice",
  "stem": "观察下图，这是一个平行四边形。底是 8cm，高是 5cm，面积为多少？",

  "image": "/images/questions/g3-para-01.png",  // 图片题（有现成图片时用）
  "svg": "<svg viewBox='0 0 200 120'>...</svg>",  // SVG 题（推荐）
  "html": "<div class='matchsticks'>...</div>",   // HTML/CSS 图形

  "options": ["12平方厘米", "40平方厘米", "80平方厘米", "20平方厘米"],
  "answer": "1",
  "solution": "平行四边形面积 = 底 × 高 = 8 × 5 = 40（平方厘米）"
}
```

### 1.3 渲染优先级

```
有 image → 渲染图片
无 image，有 svg → 渲染 SVG
无 image/svg，有 html → 渲染 HTML
以上都没有 → 仅渲染文字题干
```

### 2.4 单套试卷入库检查清单

每套入库的试卷必须满足以下全部条件：

- [ ] **来源可查**：标注年份 + 地区 + 考试名称（如"2024年 北京市 人教版 六年级上册 期末"）
- [ ] **题目完整**：含完整题干、选项/填空位置、答案
- [ ] **答案准确**：有标准答案或解析
- [ ] **图形标注**：几何题配有 SVG 图形或图片引用
- [ ] **题型覆盖**：包含填空/选择/解答至少两种题型
- [ ] **分值标注**：每题标注分值


**验证真题真实性的方法：**
1. 查 ISBN（实体书）→ 国家图书馆检索
2. 对比多份资料中的同一道题（看答案是否一致）
3. 检查题目年份/届次/来源是否合理（太新的假题多）
4. 咨询学校老师或培训机构验证
